변수명 변수 설명 측정 단위 데이터 유형 SPSS 측도 실제 데이터 출처 비고
obs_id 관찰 ID 문자 식별자 명목 시범용 현장조사 관찰 단위 고유 ID
date 관찰 날짜 yyyy-mm-dd 날짜 명목 현장조사 날짜별 패턴 분석
weekday 요일 월~금 범주 명목 현장조사 요일별 차이 분석
location_id 장소 ID 문자 식별자 명목 현장조사 학교 공간 구분
location_name 관찰 장소명 문자 범주 명목 현장조사 급식실, 매점 앞 등
location_type 공간 유형 문자 범주 명목 현장조사 cafeteria/store/classroom 등
latitude 위도 도 연속 척도 현장 GPS/학교 지도 지도 시각화용
longitude 경도 도 연속 척도 현장 GPS/학교 지도 지도 시각화용
students_observed 관찰 학생 수 명 연속 척도 현장 관찰 노출량 통제 변수
lunch_menu 대표 급식 메뉴 문자 범주 명목 NEIS 급식식단정보 급식·포장 쓰레기 배경
event_day 행사일 여부 0/1 이분형 명목 학교 일정 1=행사·동아리 활동 등
plastic_bottle_count 플라스틱 병 수 개 연속 척도 쓰레기 분류 조사 품목별 배출량
plastic_cup_count 플라스틱 컵 수 개 연속 척도 쓰레기 분류 조사 품목별 배출량
straw_spoon_count 빨대·일회용 숟가락 수 개 연속 척도 쓰레기 분류 조사 품목별 배출량
snack_wrapper_count 과자·간식 포장재 수 개 연속 척도 쓰레기 분류 조사 품목별 배출량
plastic_bag_count 비닐봉투 수 개 연속 척도 쓰레기 분류 조사 품목별 배출량
meal_container_count 일회용 용기 수 개 연속 척도 쓰레기 분류 조사 품목별 배출량
total_plastic_items 총 플라스틱 품목 수 개 연속 척도 계산 변수 품목 합계
plastic_weight_g 플라스틱 추정 무게 g 연속 척도 현장 계량/추정 배출량 규모
food_waste_kg 음식물 잔반량 kg 연속 척도 급식실 조사 급식 관련 쓰레기 보조
recycling_bin_count 분리수거함 수 개 연속 척도 현장 관찰 분리배출 환경
recycling_bin_distance_m 가장 가까운 분리수거함 거리 m 연속 척도 현장 관찰 접근성 변수
separation_accuracy_pct 분리배출 정확도 % 연속 척도 현장 관찰 오염·혼합 배출 수준
reusable_bottle_rate_pct 텀블러·다회용기 사용률 % 연속 척도 학생 설문/관찰 대체 행동 지표
single_use_purchase_count 일회용품 구매 추정 건수 건 연속 척도 매점·설문 구매 행동 지표
student_awareness_score 환경 인식 점수 1~5 리커트 서열 학생 설문 5점 척도
waste_score 쓰레기 배출 위험점수 0~100 연속 척도 계산 변수 높을수록 관리 필요
reduction_potential_score 감축 가능성 점수 0~100 연속 척도 계산 변수 개선 우선순위
priority_level 개선 우선순위 등급 낮음~매우 높음 범주 서열 계산 변수 캠페인 우선장소
expected_reduction_items 예상 감축 가능 품목 수 개 연속 척도 감축 시나리오 캠페인 효과 추정

Q3 나는 텀블러나 다회용 물병을 자주 사용한다. 1~5 서열

Q5 분리수거함 위치가 쓰레기를 버리기 편한 곳에 있다고 생각한다. 1~5 서열
Q6 나는 플라스틱 병, 컵, 비닐, 일반쓰레기를 정확히 구분할 수 있다. 1~5 서열
Q7 친구들이 분리배출을 정확히 하지 않는 장면을 자주 본다. 1~5 서열
Q8 학교가 다회용기 사용 캠페인을 하면 참여할 의향이 있다. 1~5 서열
Q9 매점에서 일회용 포장을 줄일 수 있다고 생각한다. 1~5 서열
Q10 급식실 또는 교실에서 발생하는 쓰레기 정보를 공개하면 행동이 바뀔 것 같다. 1~5 서열

1. 데이터 불러오기 Excel/CSV 데이터를 SPSS로 불러온다. 전체 변수 파일 → 열기 → 데이터 첫 행을 변수명으로 읽어오고, 한글 변수명 대신 영문 변수명 사용
2. 빈도분석 요일, 장소유형, 우선순위 등 범주형 변수 분포를 확인한다. weekday, location_type, priority_level 분석 → 기술통계량 → 빈도분석 어느 공간에서 관찰이 많이 이루어졌는지 확인
3. 기술통계 플라스틱 품목별 평균·최대값을 확인한다. total_plastic_items, plastic_weight_g, separation_accuracy_pct 분석 → 기술통계량 → 기술통계 평균보다 높은 공간을 우선 파악
4. 장소유형별 차이 공간 유형별 플라스틱 배출량 차이를 비교한다. location_type, total_plastic_items, waste_score 분석 → 평균비교 → 일원배치 분산분석 매점·급식실·교실복도 간 차이 해석
5. 상관분석 분리수거 환경과 배출량·감축 가능성의 관계를 본다. recycling_bin_distance_m, separation_accuracy_pct, reusable_bottle_rate_pct, reduction_potential_score 분석 → 상관분석 → 이변량 상관계수 거리↑ 정확도↓, 다회용기↑ 배출량↓ 여부 확인
6. 회귀분석 플라스틱 배출량을 설명하는 요인을 검증한다. total_plastic_items, students_observed, single_use_purchase_count, reusable_bottle_rate_pct 분석 → 회귀분석 → 선형 학생 수를 통제했을 때 구매·다회용기 영향 확인
7. 군집분석 쓰레기 배출 패턴이 비슷한 공간을 유형화한다. total_plastic_items, separation_accuracy_pct, reusable_bottle_rate_pct, waste_score 분석 → 분류분석 → K-평균 군집분석 고배출형, 분리배출 취약형, 저위험형으로 명명
8. 시각화 장소별 감축 우선순위를 그래프로 표현한다. location_name, reduction_potential_score 그래프 → 차트 작성기 학생·교사·학교가 직관적으로 볼 수 있게 표현
