  1. 플라스틱 사용과 학교 쓰레기 배출 분석

학교에서 가장 많이 발생하는 플라스틱 쓰레기는 무엇인가

1. 프로젝트 개요

본 프로젝트는 학교생활 속에서 학생들이 실제로 마주하는 플라스틱 쓰레기 문제를 데이터로 측정하고, 품목별 배출량과 감축 가능성을 분석하여 학교 플라스틱 감축 캠페인을 설계하는 데이터 기반 문제해결 보고서이다.

구분 내용
문제의식 학교에서 가장 많이 발생하는 플라스틱 쓰레기는 무엇이며, 어디에서 집중적으로 발생하는가
데이터 쓰레기 분류 조사, 매점·급식 관련 데이터, 학생 설문, 현장 관찰 기록
분석 방법 품목별 배출량 분석, 장소별 비교, 상관분석, 회귀분석, 군집분석, 감축 시나리오
산출물 학교 플라스틱 감축 캠페인, 감축 우선순위표, 분리배출 개선 제안
연결 전공 환경정책, 경영학, 지속가능성, 데이터사이언스

2. 문제 정의 카드

질문 답변
내가 본 문제는 무엇인가? 점심시간과 쉬는 시간 이후 매점 앞, 급식실 퇴식구, 교실 복도에 플라스틱 병·컵·포장재가 집중적으로 쌓인다.
왜 이 문제가 나에게 중요한가? 쓰레기는 단순히 청소 담당자의 일이 아니라 학교 공동체의 생활습관과 지속가능성의 문제이기 때문이다.
어떤 데이터를 사용할 것인가? 품목별 플라스틱 개수, 추정 무게, 분리배출 정확도, 다회용기 사용률, 매점 이용, 학생 인식 설문, 급식 메뉴 자료를 사용한다.
어떤 분석을 할 것인가? 장소별 배출량 비교, 품목별 비중 분석, 다회용기 사용률과 배출량의 상관분석, 감축 가능성 점수화를 수행한다.
누구에게 제안할 것인가? 학교, 학생회, 환경동아리, 학부모회, 매점 운영자, 급식실, 성북구 자원순환 담당 부서에 제안한다.
전공 관심과의 연결 환경정책과 경영학 관점에서 학교라는 작은 조직의 자원순환 시스템을 설계하는 경험으로 확장된다.

3. 데이터 설계

분석 단위는 ‘날짜 × 학교 공간 × 관찰 사례’이다. 시범 데이터는 10개 학교 공간을 5일 동안 관찰한 50행으로 구성했다. 실제 조사에서는 학생들이 같은 기준으로 품목을 세고, 설문과 급식·매점 자료를 결합하면 된다.

변수명 설명 SPSS 측도
total_plastic_items 관찰 시간 동안 발생한 플라스틱 품목 총수 척도
plastic_weight_g 플라스틱 쓰레기 추정 무게 척도
separation_accuracy_pct 분리배출 정확도 척도
reusable_bottle_rate_pct 텀블러·다회용기 사용률 척도
student_awareness_score 환경 인식 점수(1~5) 서열
waste_score 쓰레기 배출 위험점수(0~100) 척도
reduction_potential_score 감축 가능성 점수(0~100) 척도
priority_level 개선 우선순위 등급 서열

실제 공공데이터는 지역 단위의 비교 기준을 만들고, 학교 내부 데이터는 학생의 생활공간에서 발생하는 쓰레기를 직접 측정하는 핵심 자료로 사용한다. 공공데이터만으로 학교 내부 플라스틱 품목별 배출량을 알 수 없으므로 현장조사와 학생 설문이 반드시 필요하다.

6. SPSS 분석 절차

단계 분석 목적 SPSS 메뉴
1. 데이터 불러오기 Excel/CSV 데이터를 SPSS로 불러온다. 파일 → 열기 → 데이터
2. 빈도분석 요일, 장소유형, 우선순위 등 범주형 변수 분포를 확인한다. 분석 → 기술통계량 → 빈도분석
3. 기술통계 플라스틱 품목별 평균·최대값을 확인한다. 분석 → 기술통계량 → 기술통계
4. 장소유형별 차이 공간 유형별 플라스틱 배출량 차이를 비교한다. 분석 → 평균비교 → 일원배치 분산분석
5. 상관분석 분리수거 환경과 배출량·감축 가능성의 관계를 본다. 분석 → 상관분석 → 이변량 상관계수
6. 회귀분석 플라스틱 배출량을 설명하는 요인을 검증한다. 분석 → 회귀분석 → 선형

8. 한계점과 후속 연구

시범 데이터는 실제 학교 데이터를 반영한 것이 아니라 분석 구조를 보여주기 위한 가상 데이터이다. 실제 프로젝트에서는 최소 2주 이상 같은 시간대와 같은 기준으로 관찰해야 하며, 관찰자 간 기준 차이를 줄이기 위해 사진 예시와 품목 분류 기준표를 만들어야 한다.

후속 연구로는 다회용기 캠페인 전후 비교, 매점 포장 방식 변경 실험, 급식 메뉴와 잔반량의 관계, 플라스틱 배출량과 학생 인식 변화의 매개효과 분석을 수행할 수 있다.

9. 진로·전공 연결 성찰

이 프로젝트를 통해 나는 쓰레기가 단순히 버려지는 물건이 아니라 학교의 소비 습관, 공간 배치, 제도 설계가 만들어낸 결과라는 점을 이해했다. 데이터는 누군가를 비난하기 위한 도구가 아니라, 더 나은 공동체 행동을 설계하기 위한 근거가 될 수 있다.

에세이 질문: 나는 학교에서 가장 많이 버려지는 플라스틱을 보며 무엇을 불편하다고 느꼈는가? 데이터는 나의 직관과 같았는가, 달랐는가? 나는 이 문제를 개인의 무관심으로 볼 것인가, 학교 시스템의 설계 문제로 볼 것인가?
