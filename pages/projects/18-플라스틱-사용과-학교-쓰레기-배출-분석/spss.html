SPSS 분석 가이드: 플라스틱 사용과 학교 쓰레기 배출 분석

이 가이드는 18_dataset.xlsx 또는 18_dataset.csv를 SPSS에서 불러와 품목별 배출량 분석, 장소별 비교, 상관분석, 회귀분석, 군집분석을 수행하는 절차를 안내한다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 → Excel 또는 CSV 선택 → 첫 행에서 변수 이름 읽기 체크 → 확인.

2. 변수 보기 설정

변수 측도 역할
location_type 명목 장소 유형 독립변수
total_plastic_items 척도 핵심 종속변수
separation_accuracy_pct 척도 분리배출 환경 변수
reusable_bottle_rate_pct 척도 감축 행동 변수
waste_score 척도 위험점수
priority_level 서열 개선 우선순위 등급

3. 분석 절차

분석 단계 SPSS 메뉴 해석 포인트
1. 데이터 불러오기 파일 → 열기 → 데이터 첫 행을 변수명으로 읽어오고, 한글 변수명 대신 영문 변수명 사용
2. 빈도분석 분석 → 기술통계량 → 빈도분석 어느 공간에서 관찰이 많이 이루어졌는지 확인
3. 기술통계 분석 → 기술통계량 → 기술통계 평균보다 높은 공간을 우선 파악
4. 장소유형별 차이 분석 → 평균비교 → 일원배치 분산분석 매점·급식실·교실복도 간 차이 해석
5. 상관분석 분석 → 상관분석 → 이변량 상관계수 거리↑ 정확도↓, 다회용기↑ 배출량↓ 여부 확인
6. 회귀분석 분석 → 회귀분석 → 선형 학생 수를 통제했을 때 구매·다회용기 영향 확인
7. 군집분석 분석 → 분류분석 → K-평균 군집분석 고배출형, 분리배출 취약형, 저위험형으로 명명
8. 시각화 그래프 → 차트 작성기 학생·교사·학교가 직관적으로 볼 수 있게 표현

4. SPSS Syntax 예시

FREQUENCIES VARIABLES=weekday location_type priority_level.

DESCRIPTIVES VARIABLES=total_plastic_items plastic_weight_g separation_accuracy_pct reusable_bottle_rate_pct waste_score reduction_potential_score.

ONEWAY total_plastic_items waste_score BY location_type /STATISTICS DESCRIPTIVES.

CORRELATIONS /VARIABLES=total_plastic_items recycling_bin_distance_m separation_accuracy_pct reusable_bottle_rate_pct reduction_potential_score.

REGRESSION /DEPENDENT total_plastic_items /METHOD=ENTER students_observed single_use_purchase_count reusable_bottle_rate_pct separation_accuracy_pct.

QUICK CLUSTER total_plastic_items separation_accuracy_pct reusable_bottle_rate_pct waste_score /CRITERIA=CLUSTER(3).

5. 보고서 해석 문장 예시

분석 결과, 장소유형에 따라 플라스틱 배출량에 차이가 나타났다. 특히 매점 앞과 급식실 퇴식구의 평균 배출량이 높았으며, 다회용기 사용률이 높은 장소에서는 총 플라스틱 품목 수가 상대적으로 낮았다. 따라서 플라스틱 감축 정책은 학생 개인의 실천만 요구하기보다 매점 포장 방식, 분리수거함 배치, 다회용기 사용 유도 등 학교 시스템 개선과 함께 추진되어야 한다.
