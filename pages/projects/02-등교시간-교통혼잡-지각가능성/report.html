등교 시간 교통 혼잡과 지각 가능성 분석 보고서

학생의 지각은 개인의 게으름만이 아니라 교통 구조의 문제일 수 있는가

1. 프로젝트 개요

본 프로젝트는 용문고 학생들의 지각 현상을 개인의 성실성 문제로만 보지 않고, 버스 지연, 지하철 혼잡, 날씨, 요일, 환승 구조와 같은 교통 시스템 요인이 함께 작용하는지를 데이터로 분석하는 탐구형 프로젝트이다.

분석 단위는 ‘학생 × 날짜 × 등교 경로’이며, 시범 데이터는 10명의 학생을 5일 동안 관찰한 50행 구조로 설계하였다. 실제 수행 시에는 학생 설문과 서울시 버스도착 정보, 서울교통공사 지하철혼잡도, 기상청 단기예보 데이터를 결합한다.

항목 내용
문제의식 학생들의 지각은 개인의 게으름만이 아니라 교통 구조의 문제일 수 있는가
핵심 데이터 버스 도착 정보, 지하철 혼잡도, 날씨, 요일, 학교 도착 시간 설문
분석 방법 기술통계, 요일별 패턴 분석, 상관분석, 회귀분석, 로지스틱 회귀분석
산출물 최적 등교 경로 추천표, 요일별 지각 위험표, 통학 경로 개선 제안서
연결 전공 산업공학, 경영과학, 통계학, 모빌리티, 공공정책

2. 문제 정의 카드

질문 답변
내가 본 문제는 무엇인가? 등교 시간에 같은 학생이라도 요일, 비, 버스 지연, 환승 여부에 따라 학교 도착 시간이 크게 달라지는 문제를 보았다.
왜 중요한가? 지각을 학생 개인의 게으름으로만 판단하면 교통 구조의 불확실성을 보지 못한다. 학생을 비난하기보다 통학 조건을 함께 이해해야 한다.
어떤 데이터를 사용할 것인가? 학생의 집 출발시각, 학교 도착시각, 통학수단, 버스 지연시간, 지하철 혼잡도, 날씨, 요일, 환승 횟수, 체감 혼잡도
어떤 분석을 할 것인가? 요일별 평균 지각시간 비교, 경로별 지각률 비교, 지각시간에 영향을 주는 요인에 대한 상관분석과 회귀분석
누구에게 제안할 것인가? 학교 생활지도부, 담임교사, 학생회, 학부모회, 지역 교통 관련 기관
전공 관심과의 연결 데이터사이언스와 모빌리티 분석을 통해 학생에게 더 공정하고 현실적인 등교 지원 방안을 만들고자 한다.

3. 데이터 설명

이 보고서에 포함된 데이터는 시범용 가상 데이터이다. 다만 변수 구조는 실제 공공데이터와 학생 설문을 결합할 수 있도록 설계하였다. 실제 수업에서는 Sources 시트의 공식 링크를 통해 버스·지하철·날씨 데이터를 확보하고, 학생 설문으로 출발·도착 시간을 수집하면 된다.

구분 주요 변수 측도 활용
학생·날짜 정보 student_id, date, weekday 명목 요일별 패턴 분석
등교 행동 정보 depart_min, arrival_min, travel_time_min 척도 통학시간·지각시간 계산
교통 시스템 정보 bus_delay_min, subway_congestion_pct, transfer_count 척도 교통 구조 요인 분석
날씨 정보 rain_yn, temperature_c 명목/척도 비 오는 날 지각 증가 여부 확인
심리·체감 정보 perceived_crowding 서열 학생이 느끼는 혼잡도 반영
결과 변수 late_min, late_yn, lateness_risk_score 척도/명목 회귀분석 및 추천표 작성

5. 분석 방법

분석 단계 분석 목적 SPSS 메뉴 경로 해석 포인트
기술통계 통학시간, 지각시간, 버스 지연, 혼잡도의 평균 파악 분석 → 기술통계량 → 기술통계 평균과 표준편차로 전체 수준 확인
요일별 패턴 월~금 중 지각이 집중되는 요일 확인 분석 → 평균비교 → 일원배치 분산분석 요일별 평균 지각시간 차이 확인
경로별 비교 버스직행, 지하철중심 등 경로별 지각률 비교 분석 → 평균비교 → 일원배치 분산분석 경로별 안정성 비교

회귀분석 지각시간을 설명하는 핵심 요인 파악 분석 → 회귀분석 → 선형 B값과 유의확률 해석
로지스틱 회귀 지각 여부를 0/1로 예측 분석 → 회귀분석 → 이분형 로지스틱 Exp(B)로 지각 가능성 증가 판단

7. 최적 등교 경로 추천표

상황 추천 경로/행동 근거 기대 효과
평상시 맑은 날 현재 경로 유지 또는 지하철중심 경로 선택 지하철중심 경로의 지각률과 위험점수가 낮음 도착시간 예측 안정성 향상
월요일 평소보다 7~10분 조기 출발 월요일 평균 지각시간이 평일 중 높은 편 주초 교통혼잡 완충
비 오는 날 버스 의존 경로는 8~10분 조기 출발 우천 시 평균 지각시간과 위험점수 증가 날씨 변수에 따른 지각 감소
버스 지연 7분 이상 예상 지하철 또는 도보+지하철 대체 검토 버스 지연시간과 지각시간의 상관이 큼 지연 리스크 분산
환승 2회 이상 가능하면 단순 경로 선택 환승이 많으면 예측 불확실성이 증가 경로 안정성 개선

8. 개선 제안서

제안 대상 제안 내용 근거 데이터 기대 효과 실행 난이도
학교 생활지도부 월요일·우천일에는 지각 판단 시 교통 상황을 함께 고려 요일별 지각률, rain_yn, bus_delay_min 학생을 일방적으로 비난하지 않는 생활지도 중
학생회 학기 초 ‘나의 최적 등교 경로 찾기’ 캠페인 운영 경로별 지각률, 경로 안정성 학생 자율 개선과 데이터 리터러시 향상 중
담임교사 상습 지각 학생에게 개인 비난보다 경로·출발시간 점검표 제공 student_id별 late_min 패턴 맞춤형 등교 코칭 하
학부모회 비 오는 날·월요일 조기 출발 안내 메시지 공유 요일·날씨별 위험점수 가정-학교 공동 지원 하
지역 교통기관 학교 인근 주요 정류장의 등교시간 안내정보 개선 요청 버스 지연시간, 대기시간 대중교통 이용 학생의 불확실성 감소 상

9. 한계점과 후속 연구

첫째, 본 데이터는 시범용 가상 데이터이므로 실제 정책 제안 전에는 최소 2~4주간 학생 설문과 실제 API 데이터를 결합해야 한다. 둘째, 학생의 정확한 거주지를 수집하면 개인정보 위험이 있으므로 행정동 또는 권역 수준으로만 기록해야 한다. 셋째, 실제 버스 도착정보와 날씨 데이터는 API 호출 시점과 등교 시간이 맞아야 하므로 자동 수집 또는 시간대별 수집 계획이 필요하다.

후속 연구에서는 각 학생의 ‘평소 경로’와 ‘대체 경로’를 비교하고, Google My Maps 또는 QGIS를 활용해 경로별 이동시간과 지연 위험을 지도 위에 표시할 수 있다.

10. 나의 진로·전공 관심과의 연결

이 프로젝트는 산업공학의 최적화 사고, 통계학의 예측 분석, 모빌리티 분야의 경로 분석, 공공정책의 학생 지원 관점을 함께 연결한다. 데이터를 통해 학생을 평가하는 것이 아니라, 학생이 더 안정적으로 등교할 수 있는 조건을 설계한다는 점에서 사람 중심 데이터 분석의 의미가 있다.

11. 에세이형 성찰 문장

나는 지각을 개인의 게으름으로만 보았던 시각에서 벗어나, 학생이 매일 통과하는 교통 시스템의 불확실성을 함께 보게 되었다. 데이터는 학생을 단정하기 위한 도구가 아니라, 학생의 생활 조건을 더 정확히 이해하고 개선하기 위한 도구라는 점을 배웠다.
