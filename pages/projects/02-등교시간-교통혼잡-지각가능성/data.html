변수명 변수 설명 측정 단위 SPSS 측도 코딩/비고
obs_id 관측 ID 문자 명목 학생-날짜 단위 관측치
student_id 학생 익명 ID 문자 명목 개인 식별 방지를 위해 익명화
grade 학년 1~3 서열 1=1학년, 2=2학년, 3=3학년
gender 성별 남/여 명목 선택 입력
home_area 거주 권역 행정동 수준 명목 정확한 주소가 아닌 권역만 기록
date 조사일 YYYY-MM-DD 명목 정렬 가능하도록 날짜 형식 권장
weekday 요일 월~금 명목 요일별 패턴 분석
commute_route 통학 경로 유형 범주 명목 버스직행/지하철중심/버스+지하철/도보+마을버스/도보
school_start_min 등교 기준 시각 분 척도 08:00=480으로 코딩
depart_min 집 출발 시각 분 척도 자정 이후 분 단위, 예: 07:25=445
arrival_min 학교 도착 시각 분 척도 자정 이후 분 단위
travel_time_min 통학 소요시간 분 척도 arrival_min - depart_min
late_min 지각 시간 분 척도 max(0, arrival_min - school_start_min)
late_yn 지각 여부 0/1 명목 0=정상, 1=지각
bus_delay_min 버스 지연시간 분 척도 버스 이용 경로만 값이 큼
subway_congestion_pct 지하철 혼잡도 % 척도 지하철 미이용은 0으로 코딩
rain_yn 비 여부 0/1 명목 0=비 안 옴, 1=비 옴
temperature_c 기온 ℃ 척도 등교 시간대 기온
transfer_count 환승 횟수 회 척도 경로 복잡도
walking_time_min 도보 시간 분 척도 정류장/역까지 걷는 시간 포함
bus_wait_min 버스 대기시간 분 척도 버스 미이용은 0
perceived_crowding 체감 혼잡도 1~5 서열 1=매우 여유, 5=매우 혼잡
departure_buffer_min 출발 여유시간 분 척도 등교 기준 대비 계획상 여유
route_stability_score 경로 안정성 점수 0~100 척도 높을수록 도착시간 예측 안정
lateness_risk_score 지각 위험점수 0~100 척도 높을수록 지각 가능성 큼
recommended_action 추천 행동 문자 명목 경로 또는 출발시각 개선 제안

SPSS_Steps

SPSS_Steps: 10행 x 4열

분석 단계 분석 목적 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 Data 시트를 SPSS에서 불러온다. 파일 → 열기 → 데이터 첫 행을 변수명으로 읽고, 변수보기에서 측도 지정
2. 지각 여부 확인 arrival_min과 school_start_min으로 late_min, late_yn 검산 변환 → 변수 계산 late_min = max(0, arrival_min - school_start_min)
3. 기술통계 통학시간, 지연시간, 혼잡도, 위험점수의 평균과 분포 확인 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최솟값, 최댓값 해석
4. 요일별 패턴 요일에 따라 지각시간이 달라지는지 확인 분석 → 평균비교 → 일원배치 분산분석 월·목요일 지각 증가 여부 해석
5. 경로별 비교 통학경로별 평균 통학시간과 지각률 비교 분석 → 평균비교 → 일원배치 분산분석 버스직행, 지하철중심 등 비교
6. 상관분석 출발시각, 버스 지연, 혼잡도와 지각시간의 관계 확인 분석 → 상관분석 → 이변량 상관계수 상관계수의 방향과 크기 해석
7. 선형회귀분석 late_min을 설명하는 주요 요인 파악 분석 → 회귀분석 → 선형 버스 지연, 비, 환승횟수, 출발시각 효과 해석
8. 로지스틱 회귀분석 late_yn=1이 될 가능성을 예측 분석 → 회귀분석 → 이분형 로지스틱 Exp(B)를 통해 지각 가능성 증가 요인 해석
9. 산점도/그래프 지각 위험점수와 경로 안정성을 시각화 그래프 → 차트 작성기 위험이 높은 학생-요일 조합 식별
