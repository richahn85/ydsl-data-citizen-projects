SPSS 분석 가이드

  1. 등교 시간 교통 혼잡과 지각 가능성 분석

1. 데이터 불러오기

파일: 02_dataset.xlsx 또는 02_dataset.csv

SPSS 메뉴: 파일 → 열기 → 데이터 → Excel 또는 CSV 선택 → 첫 행에서 변수 이름 읽어오기 체크

변수 보기에서 student_id, gender, home_area, weekday, commute_route, late_yn, rain_yn은 명목, perceived_crowding은 서열, 나머지 수치 변수는 척도로 지정한다.

2. 변수 계산 확인

late_min과 late_yn은 아래 방식으로 생성할 수 있다. 이미 데이터셋에 포함되어 있지만, 학생들이 직접 계산해보면 변수 생성 원리를 이해할 수 있다.

COMPUTE travel_time_min = arrival_min - depart_min. COMPUTE late_min = arrival_min - school_start_min. IF (late_min < 0) late_min = 0. IF (arrival_min > school_start_min) late_yn = 1. IF (arrival_min <= school_start_min) late_yn = 0. EXECUTE.

3. 기술통계

목적 SPSS 메뉴 사용 변수 해석
전체 등교 상황 파악 분석 → 기술통계량 → 기술통계 travel_time_min, late_min, bus_delay_min, subway_congestion_pct, lateness_risk_score 평균과 표준편차로 등교 불안정성 확인

4. 요일별 패턴 분석

목적: 월~금 중 어느 요일에 지각 가능성이 높은지 확인한다.

SPSS 메뉴: 분석 → 평균비교 → 일원배치 분산분석

종속변수: late_min 또는 travel_time_min / 요인: weekday

해석: 월요일과 목요일 평균이 높다면 주초 혼잡 또는 우천 조건의 영향을 의심할 수 있다.

5. 경로별 비교

목적: 버스직행, 지하철중심, 버스+지하철, 도보+마을버스, 도보 경로 중 어느 경로가 안정적인지 비교한다.

SPSS 메뉴: 분석 → 평균비교 → 일원배치 분산분석

종속변수: late_min, lateness_risk_score, route_stability_score / 요인: commute_route

후속 분석: 평균표를 이용해 최적 등교 경로 추천표를 만든다.

6. 상관분석

목적: 지각시간과 교통·날씨·출발요인의 관련성을 확인한다.

SPSS 메뉴 변수 해석 포인트
분석 → 상관분석 → 이변량 상관계수 late_min, depart_min, bus_delay_min, subway_congestion_pct, rain_yn, transfer_count, perceived_crowding, departure_buffer_min 상관계수가 양수이면 해당 요인이 커질수록 지각시간 증가, 음수이면 감소

7. 선형 회귀분석

목적: 지각시간을 설명하는 주요 변수를 찾는다.

SPSS 메뉴: 분석 → 회귀분석 → 선형

종속변수: late_min

독립변수: depart_min, bus_delay_min, subway_congestion_pct, rain_yn, transfer_count, perceived_crowding, departure_buffer_min

해석: B값이 양수이면 지각시간을 증가시키는 변수이고, 음수이면 지각시간을 줄이는 변수이다. p값이 .05 미만이면 통계적으로 유의하다고 판단할 수 있다.

8. 로지스틱 회귀분석

목적: 지각 여부(late_yn=1)를 예측한다.

SPSS 메뉴: 분석 → 회귀분석 → 이분형 로지스틱

종속변수: late_yn / 독립변수: bus_delay_min, rain_yn, transfer_count, departure_buffer_min, perceived_crowding

해석: Exp(B)가 1보다 크면 지각 가능성을 높이는 변수, 1보다 작으면 지각 가능성을 낮추는 변수로 해석한다.

9. 그래프 작성

그래프 SPSS 메뉴 사용 변수 활용
요일별 평균 지각시간 막대그래프 그래프 → 차트 작성기 weekday, late_min 요일별 위험 요인 설명
경로별 지각률 막대그래프 그래프 → 차트 작성기 commute_route, late_yn 최적 등교 경로 추천
산점도 그래프 → 차트 작성기 route_stability_score, lateness_risk_score 안정성 낮고 위험 높은 경로 식별

10. 보고서 작성 문장 예시

“분석 결과, 비 오는 날과 버스 지연시간은 지각시간 증가와 관련이 있었다. 특히 월요일과 우천일에는 평균 지각시간과 지각위험점수가 함께 높아졌다. 따라서 지각을 개인의 게으름으로만 해석하기보다, 등교 경로의 불확실성을 고려한 조기 출발 안내와 대체 경로 추천이 필요하다.”

* 1. 변수 계산 검산.
COMPUTE travel_time_check = arrival_min - depart_min.
COMPUTE late_min_check = arrival_min - school_start_min.
IF (late_min_check < 0) late_min_check = 0.
IF (arrival_min > school_start_min) late_yn_check = 1.
IF (arrival_min <= school_start_min) late_yn_check = 0.
EXECUTE.

* 2. 기술통계.
DESCRIPTIVES VARIABLES=travel_time_min late_min bus_delay_min subway_congestion_pct
 transfer_count perceived_crowding departure_buffer_min route_stability_score lateness_risk_score
 /STATISTICS=MEAN STDDEV MIN MAX.

* 3. 요일별 평균 비교.
ONEWAY late_min BY weekday
 /STATISTICS DESCRIPTIVES
 /MISSING ANALYSIS.

* 4. 통학 경로별 평균 비교.
ONEWAY late_min BY commute_route
 /STATISTICS DESCRIPTIVES
 /MISSING ANALYSIS.

* 5. 상관분석.
CORRELATIONS
 /VARIABLES=late_min depart_min travel_time_min bus_delay_min subway_congestion_pct rain_yn
 transfer_count perceived_crowding departure_buffer_min
 /PRINT=TWOTAIL NOSIG
 /MISSING=PAIRWISE.

* 6. 선형 회귀분석: 지각시간 예측.
REGRESSION
 /DEPENDENT late_min
 /METHOD=ENTER depart_min bus_delay_min subway_congestion_pct rain_yn transfer_count perceived_crowding departure_buffer_min
 /STATISTICS COEFF OUTS R ANOVA COLLIN TOL.

* 7. 이분형 로지스틱 회귀분석: 지각 여부 예측.
LOGISTIC REGRESSION VARIABLES late_yn
 /METHOD=ENTER bus_delay_min rain_yn transfer_count departure_buffer_min perceived_crowding
 /PRINT=CI(95)
 /CRITERIA=PIN(.05) POUT(.10) ITERATE(20) CUT(.5).

* 8. 그래프는 메뉴 사용 권장:
* 그래프 → 차트 작성기 → 막대그래프: weekday별 late_min 평균
* 그래프 → 차트 작성기 → 막대그래프: commute_route별 late_yn 평균
* 그래프 → 차트 작성기 → 산점도: route_stability_score x lateness_risk_score.
