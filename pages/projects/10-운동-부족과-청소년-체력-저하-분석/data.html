변수명 변수 설명 측정 단위 데이터 유형 SPSS 측도 코딩/값 설명 비고
student_id 익명 학생 ID 문자 문자 명목 S001~S050 개인 식별정보 제거
grade 학년 1~3 숫자 명목 1=1학년, 2=2학년, 3=3학년 분석 시 집단변수
gender 성별 남/여 문자 명목 남, 여 민감정보 최소화
academic_pressure 입시·학업 압박감 1~5점 숫자 서열 1=매우 낮음, 5=매우 높음 설문
daily_sitting_hours 하루 앉아있는 시간 시간 숫자 척도 학교+학원+자습 포함 자기보고
weekday_exercise_min 평일 하루 평균 운동시간 분 숫자 척도 숨이 차는 활동+걷기 제외/포함 기준 명시 필요 설문
weekend_exercise_min 주말 하루 평균 운동시간 분 숫자 척도 토·일 평균 설문
moderate_vigorous_days 최근 7일 중 중강도 이상 신체활동 일수 0~7일 숫자 척도 0~7 청소년건강행태조사형 문항
strength_days 최근 7일 중 근력운동 일수 0~7일 숫자 척도 0~7 팔굽혀펴기/스쿼트 등
pe_class_satisfaction 체육수업 만족도 1~5점 숫자 서열 1=매우 낮음, 5=매우 높음 설문
school_club_participation 스포츠 동아리 참여 여부 0/1 숫자 명목 0=미참여, 1=참여 집단 비교
sleep_hours 평일 평균 수면시간 시간 숫자 척도 소수 1자리 설문
stress_score 스트레스 점수 0~100점 숫자 척도 높을수록 스트레스 높음 문항 합산
bmi 체질량지수 kg/m² 숫자 척도 체중(kg)/키(m)^2 자기보고/측정
paps_cardio_score PAPS 심폐지구력 점수 0~100점 숫자 척도 높을수록 좋음 시범 환산
paps_flexibility_score PAPS 유연성 점수 0~100점 숫자 척도 높을수록 좋음 시범 환산
paps_muscle_score PAPS 근력·근지구력 점수 0~100점 숫자 척도 높을수록 좋음 시범 환산
paps_power_score PAPS 순발력 점수 0~100점 숫자 척도 높을수록 좋음 시범 환산
paps_bmi_score PAPS BMI 건강점수 0~100점 숫자 척도 정상범위에 가까울수록 높음 시범 환산
total_fitness_score 종합 체력점수 0~100점 숫자 척도 가중합 종속/핵심지표
fitness_grade 체력등급 1~5등급 숫자 서열 1=우수, 5=개선 필요 PAPS형 등급
activity_type 신체활동 유형 운동부족형/보통형/활동형 문자 명목 군집/유형 보고서 핵심
fatigue_score 피로감 점수 0~100점 숫자 척도 높을수록 피로 높음 설문
concentration_score 학습 집중도 점수 0~100점 숫자 척도 높을수록 집중도 높음 설문
improvement_priority 개선 우선순위 점수 0~100점 숫자 척도 높을수록 개입 필요 정책 제안 지표

SPSS_Steps

SPSS_Steps: 7행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 빈도분석 학년·성별·운동유형 분포 확인 grade, gender, activity_type, fitness_grade 분석 → 기술통계량 → 빈도분석 운동부족형 학생 비중과 체력등급 분포 확인
2. 기술통계 운동시간·수면·스트레스·체력점수 평균 확인 weekday_exercise_min, sleep_hours, stress_score, total_fitness_score 분석 → 기술통계량 → 기술통계 평균 체력 수준과 운동 부족 정도 파악
3. 상관분석 운동시간과 체력점수·스트레스·집중도의 관계 확인 weekday_exercise_min, moderate_vigorous_days, total_fitness_score, stress_score, concentration_score 분석 → 상관분석 → 이변량 상관계수 운동량이 체력 및 학습 집중과 관련되는지 확인
4. 집단비교 운동유형별 체력·집중도 차이 확인 activity_type, total_fitness_score, concentration_score, stress_score 분석 → 평균비교 → 일원배치 분산분석 운동부족형과 활동형의 평균 차이 해석
5. 회귀분석 체력점수에 영향을 주는 요인 탐색 total_fitness_score / weekday_exercise_min, sleep_hours, daily_sitting_hours, stress_score 분석 → 회귀분석 → 선형 운동시간, 앉아있는 시간, 수면의 설명력 확인
6. 군집분석 학생 건강·활동 유형 분류 weekday_exercise_min, sleep_hours, stress_score, total_fitness_score 분석 → 분류분석 → K-평균 군집분석 체력개선 우선지원군 도출
