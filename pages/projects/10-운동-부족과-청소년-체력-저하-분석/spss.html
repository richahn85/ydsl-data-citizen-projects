1) SPSS 실행 → 파일 → 열기 → 데이터

2) 파일 유형을 Excel 또는 CSV로 선택

3) 10_dataset.xlsx의 Data 시트를 선택하고, 첫 행을 변수명으로 읽어온다.

4) 변수 보기에서 grade, gender, activity_type, fitness_grade는 명목/서열로, 운동시간·점수 변수는 척도로 설정한다.

3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 빈도분석 학년·성별·운동유형 분포 확인 grade, gender, activity_type, fitness_grade 분석 → 기술통계량 → 빈도분석 운동부족형 학생 비중과 체력등급 분포 확인
2. 기술통계 운동시간·수면·스트레스·체력점수 평균 확인 weekday_exercise_min, sleep_hours, stress_score, total_fitness_score 분석 → 기술통계량 → 기술통계 평균 체력 수준과 운동 부족 정도 파악
3. 상관분석 운동시간과 체력점수·스트레스·집중도의 관계 확인 weekday_exercise_min, moderate_vigorous_days, total_fitness_score, stress_score, concentration_score 분석 → 상관분석 → 이변량 상관계수 운동량이 체력 및 학습 집중과 관련되는지 확인
4. 집단비교 운동유형별 체력·집중도 차이 확인 activity_type, total_fitness_score, concentration_score, stress_score 분석 → 평균비교 → 일원배치 분산분석 운동부족형과 활동형의 평균 차이 해석
5. 회귀분석 체력점수에 영향을 주는 요인 탐색 total_fitness_score / weekday_exercise_min, sleep_hours, daily_sitting_hours, stress_score 분석 → 회귀분석 → 선형 운동시간, 앉아있는 시간, 수면의 설명력 확인
6. 군집분석 학생 건강·활동 유형 분류 weekday_exercise_min, sleep_hours, stress_score, total_fitness_score 분석 → 분류분석 → K-평균 군집분석 체력개선 우선지원군 도출

4. 권장 분석 순서

1단계: 빈도분석

activity_type과 fitness_grade를 빈도분석하여 운동부족형 학생 비중과 체력등급 분포를 확인한다.

2단계: 기술통계

weekday_exercise_min, daily_sitting_hours, sleep_hours, stress_score, total_fitness_score의 평균과 표준편차를 확인한다.

3단계: 상관분석

운동시간, 활동일수, 앉아있는 시간, 수면시간, 스트레스와 체력점수·집중도 간 상관관계를 확인한다.

4단계: 일원배치 분산분석

activity_type별 total_fitness_score, stress_score, concentration_score 차이를 확인한다.

5단계: 선형회귀분석

total_fitness_score를 종속변수로 두고 운동시간, 수면, 앉아있는 시간, 스트레스를 독립변수로 투입한다.

6단계: 군집분석

건강·생활 패턴이 비슷한 학생군을 찾고, 체력 개선 우선지원군을 도출한다.

5. 해석 문장 예시

상관분석 결과, 중강도 이상 신체활동 일수는 종합 체력점수와 정적 상관을 보였다. 반면 앉아있는 시간과 스트레스는 체력점수 및 학습 집중도와 부적 상관을 보였다. 이는 운동 부족이 단순히 체력 문제에 그치지 않고 피로감, 스트레스, 집중도와 연결될 수 있음을 의미한다.

일원배치 분산분석 결과, 운동유형에 따라 종합 체력점수에 차이가 나타난다면, 사후분석을 통해 어떤 집단 간 차이가 큰지 확인한다. 특히 운동부족형과 활동형의 차이가 유의하다면 학교 체육·동아리 프로그램의 정책적 필요성을 제시할 수 있다.

6. 보고서 작성 팁

분석 결과를 학생 개인 비난으로 해석하지 않는다.

운동 부족을 학업시간, 좌식시간, 수면, 스트레스가 결합된 생활 구조의 문제로 해석한다.

표와 그래프는 반드시 개선 제안으로 연결한다.

최종 산출물은 ‘학교 체육·동아리 개선 제안서’로 정리한다.

실제 조사 시 익명 설문으로 진행하고, 건강 관련 민감정보는 최소한으로 수집한다.

7. SPSS Syntax 예시

FREQUENCIES VARIABLES=activity_type fitness_grade. DESCRIPTIVES VARIABLES=weekday_exercise_min daily_sitting_hours sleep_hours stress_score total_fitness_score concentration_score. CORRELATIONS /VARIABLES=weekday_exercise_min moderate_vigorous_days daily_sitting_hours sleep_hours stress_score total_fitness_score concentration_score /PRINT=TWOTAIL NOSIG. ONEWAY total_fitness_score stress_score concentration_score BY activity_type /STATISTICS DESCRIPTIVES /POSTHOC=TUKEY. REGRESSION /DEPENDENT total_fitness_score /METHOD=ENTER weekday_exercise_min moderate_vigorous_days daily_sitting_hours sleep_hours stress_score. QUICK CLUSTER weekday_exercise_min sleep_hours stress_score total_fitness_score /CRITERIA=CLUSTER(3) MXITER(10) /SAVE CLUSTER.

* Project 10: Physical Activity Deficit and Adolescent Fitness Decline.
* Import 10_dataset.csv first or open 10_dataset.xlsx Data sheet.

FREQUENCIES VARIABLES=grade gender activity_type fitness_grade
 /ORDER=ANALYSIS.

DESCRIPTIVES VARIABLES=weekday_exercise_min weekend_exercise_min moderate_vigorous_days daily_sitting_hours sleep_hours stress_score total_fitness_score concentration_score
 /STATISTICS=MEAN STDDEV MIN MAX.

CORRELATIONS
 /VARIABLES=weekday_exercise_min moderate_vigorous_days daily_sitting_hours sleep_hours stress_score total_fitness_score concentration_score
 /PRINT=TWOTAIL NOSIG
 /MISSING=PAIRWISE.

ONEWAY total_fitness_score stress_score concentration_score BY activity_type
 /STATISTICS DESCRIPTIVES
 /POSTHOC=TUKEY ALPHA(.05).

REGRESSION
 /DEPENDENT total_fitness_score
 /METHOD=ENTER weekday_exercise_min moderate_vigorous_days daily_sitting_hours sleep_hours stress_score.

QUICK CLUSTER weekday_exercise_min sleep_hours stress_score total_fitness_score
 /CRITERIA=CLUSTER(3) MXITER(10)
 /SAVE CLUSTER.

GRAPH
 /SCATTERPLOT(BIVAR)=weekday_exercise_min WITH total_fitness_score.
