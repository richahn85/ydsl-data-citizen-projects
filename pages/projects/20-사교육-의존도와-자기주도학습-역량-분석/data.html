변수명 변수 설명 측정 단위/코딩 데이터 유형 SPSS 측도 비고
student_id 익명 학생 ID S001~S050 문자 명목 개인식별 불가 코드
grade 학년 1=고1, 2=고2, 3=고3 숫자 서열
gender 성별 1=남, 2=여 숫자 명목 필요 시 분석에서 제외 가능
learning_type 학습 유형 코드 1=자기주도형,2=균형형,3=사교육의존형,4=방치형 숫자 명목 군집분석 결과 대체용
learning_type_name 학습 유형 이름 문자 문자 명목
private_hours_week 주당 사교육 시간 시간/주 숫자 척도 학원, 과외, 유료 인터넷 강좌 포함
private_cost_month_10k 월 사교육비 만원/월 숫자 척도 응답 부담을 낮추기 위해 구간형으로도 가능
private_subject_count 사교육 참여 과목 수 개 숫자 척도
self_study_hours_week 주당 자기주도학습 시간 시간/주 숫자 척도 스스로 계획하고 수행한 학습
goal_setting 목표 설정 역량 1~5점 숫자 서열 1=전혀 그렇지 않다, 5=매우 그렇다
planning 학습 계획 실행 1~5점 숫자 서열
self_monitoring 학습 중 자기점검 1~5점 숫자 서열
error_review 오답·실패 복기 1~5점 숫자 서열
resource_management 학습 자원 관리 1~5점 숫자 서열
autonomous_motivation 자율적 학습동기 1~5점 숫자 서열
private_dependence_item 사교육 의존 문항 1~5점 숫자 서열 높을수록 사교육 의존
parent_pressure_item 부모 압박 체감 1~5점 숫자 서열
purposeless_tutoring 목적 없는 사교육 참여 1~5점 숫자 서열
career_clarity_item 진로 명확성 1~5점 숫자 서열
perceived_effect 사교육 효과 자기평가 1~5점 숫자 서열
private_stress_item 사교육으로 인한 스트레스 1~5점 숫자 서열
sleep_hours 평일 평균 수면시간 시간/일 숫자 척도
overall_stress 전반적 스트레스 1~5점 숫자 서열
self_directed_score 자기주도학습 종합점수 0~100점 숫자 척도 6개 자기주도 문항 기반 지수
dependency_score 사교육 의존도 점수 0~100점 숫자 척도 시간, 비용, 과목 수, 압박감, 목적성 기반
academic_change_score 최근 성적 변화 자기평가 -10~+15점 숫자 척도 전년 또는 직전 시험 대비 주관적 변화
support_priority_score 지원 우선순위 점수 0~100점 숫자 척도 개입 필요도
support_recommendation 추천 지원 방식 문자 문자 명목 자기주도학습 코칭 등

SPSS_Steps

SPSS_Steps: 7행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 빈도분석 학년, 성별, 학습 유형 분포 확인 grade, gender, learning_type 분석 → 기술통계량 → 빈도분석 특정 학년이나 유형에 응답이 치우쳤는지 확인
2. 기술통계 사교육 시간, 자기주도학습 점수, 의존도 점수의 평균과 표준편차 확인 private_hours_week, self_study_hours_week, self_directed_score, dependency_score 분석 → 기술통계량 → 기술통계 사교육 시간이 높은 학생이 실제로 자기주도성이 높은지 첫 인상 확인
3. 상관분석 사교육 시간·의존도·자기주도학습·성적 변화의 관계 확인 private_hours_week, dependency_score, self_directed_score, academic_change_score 분석 → 상관분석 → 이변량 상관계수 단순 사교육 시간보다 의존도 점수가 더 중요한지 비교
4. 일원분산분석 학습 유형별 자기주도학습 점수 차이 검정 learning_type, self_directed_score 분석 → 평균비교 → 일원배치 분산분석 유형별 평균 차이가 통계적으로 유의한지 확인
5. 회귀분석 자기주도학습 점수를 설명하는 요인 탐색 self_directed_score, private_hours_week, dependency_score, career_clarity_item, sleep_hours, overall_stress 분석 → 회귀분석 → 선형 사교육 양보다 진로 명확성, 자율동기, 자기학습 시간이 더 설명력이 있는지 확인
6. 군집분석 학생을 학습지원 유형으로 분류 private_hours_week, self_study_hours_week, dependency_score, self_directed_score 분석 → 분류분석 → K-평균 군집분석 자기주도형, 균형형, 사교육의존형, 방치형을 도출
