3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 빈도분석 학년, 성별, 학습 유형 분포 확인 grade, gender, learning_type 분석 → 기술통계량 → 빈도분석 특정 학년이나 유형에 응답이 치우쳤는지 확인
2. 기술통계 사교육 시간, 자기주도학습 점수, 의존도 점수의 평균과 표준편차 확인 private_hours_week, self_study_hours_week, self_directed_score, dependency_score 분석 → 기술통계량 → 기술통계 사교육 시간이 높은 학생이 실제로 자기주도성이 높은지 첫 인상 확인
3. 상관분석 사교육 시간·의존도·자기주도학습·성적 변화의 관계 확인 private_hours_week, dependency_score, self_directed_score, academic_change_score 분석 → 상관분석 → 이변량 상관계수 단순 사교육 시간보다 의존도 점수가 더 중요한지 비교
4. 일원분산분석 학습 유형별 자기주도학습 점수 차이 검정 learning_type, self_directed_score 분석 → 평균비교 → 일원배치 분산분석 유형별 평균 차이가 통계적으로 유의한지 확인
5. 회귀분석 자기주도학습 점수를 설명하는 요인 탐색 self_directed_score, private_hours_week, dependency_score, career_clarity_item, sleep_hours, overall_stress 분석 → 회귀분석 → 선형 사교육 양보다 진로 명확성, 자율동기, 자기학습 시간이 더 설명력이 있는지 확인
6. 군집분석 학생을 학습지원 유형으로 분류 private_hours_week, self_study_hours_week, dependency_score, self_directed_score 분석 → 분류분석 → K-평균 군집분석 자기주도형, 균형형, 사교육의존형, 방치형을 도출

4. 해석 예시

상관분석에서 private_hours_week와 self_directed_score의 상관이 약하거나 음(-)의 방향이면, 사교육 시간이 많은 것이 곧 자기주도성이 높다는 뜻은 아님을 해석한다.

dependency_score와 self_directed_score의 상관이 음(-)의 방향이면, 사교육 ‘양’보다 사교육에 의존하는 학습 방식이 자기주도성 저하와 더 관련이 있을 수 있음을 설명한다.

분산분석에서는 자기주도형, 균형형, 사교육의존형, 방치형의 평균 자기주도점수를 비교한다. 평균 차이가 크면 학생별 맞춤 지원전략이 필요하다는 결론으로 연결한다.

5. 그래프 작성

그래프 → 차트 작성기에서 산점도를 선택하고 X축에 private_hours_week 또는 dependency_score, Y축에 self_directed_score를 넣는다. 학습 유형별 차이는 막대그래프로 표현한다.

6. SPSS Syntax 예시

GET DATA /TYPE=XLSX /FILE='20_dataset.xlsx' /SHEET=name 'Data' /READNAMES=ON.

FREQUENCIES VARIABLES=grade gender learning_type support_recommendation.

DESCRIPTIVES VARIABLES=private_hours_week private_cost_month_10k self_study_hours_week self_directed_score dependency_score academic_change_score.

CORRELATIONS /VARIABLES=private_hours_week self_study_hours_week dependency_score self_directed_score academic_change_score career_clarity_item overall_stress /PRINT=TWOTAIL NOSIG.

ONEWAY self_directed_score BY learning_type /STATISTICS DESCRIPTIVES /POSTHOC=TUKEY ALPHA(0.05).

REGRESSION /DEPENDENT self_directed_score /METHOD=ENTER private_hours_week dependency_score self_study_hours_week career_clarity_item sleep_hours overall_stress.

QUICK CLUSTER private_hours_week self_study_hours_week dependency_score self_directed_score /CRITERIA=CLUSTER(4) MXITER(10) /SAVE CLUSTER.
