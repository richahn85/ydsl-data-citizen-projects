student_id 익명 학생 ID 문자 명목 S001~S050 개인정보 보호를 위해 익명 처리
grade 학년 1~3 서열 1=1학년, 2=2학년, 3=3학년 학년별 스트레스 차이 분석
gender 성별 1/2 명목 1=남학생, 2=여학생 민감 정보이므로 선택응답 가능
exam_period 시험기간 여부 0/1 명목 0=평상시, 1=시험기간 시험 스트레스 효과 분석
sleep_hours 평일 수면시간 시간 척도 4.0~8.5 수면 부족 요인
exercise_min 하루 운동시간 분 척도 0~90 스트레스 완충 요인
academy_hours 하루 학원/사교육 시간 시간 척도 0~6 시간압박 요인
study_hours 하루 자기주도 학습시간 시간 척도 1~8 학습부담과 구분
smartphone_min 하루 스마트폰 사용시간 분 척도 40~380 디지털 생활 요인
career_anxiety_score 진로불안 평균점수 1~5 척도 career_q1~q3 평균 진로 관련 심리요인
peer_relation_score 친구관계 평균점수 1~5 척도 peer_q1~q3 평균 관계 보호요인
support_score 가정·학교 지원감 평균점수 1~5 척도 support_q1~q3 평균 사회적 지지 요인
pressure_score 성적압박 점수 1~5 척도 1=낮음, 5=높음 입시 압박 요인
stress_q1~stress_q5 스트레스 문항 1~5 서열/척도 1=전혀 그렇지 않다~5=매우 그렇다 평균값 계산 가능
stress_score 스트레스 평균점수 1~5 척도 stress_q1~q5 평균 핵심 종속변수
risk_index 스트레스 위험지수 0~100 척도 수면·시험·불안·지원감 기반 가중점수 위험신호 체크리스트
stress_level 스트레스 수준 1~3 서열 1=안정, 2=주의, 3=고위험 집단 비교 및 세그먼트
high_stress 고스트레스 여부 0/1 명목 0=아니오, 1=예 로지스틱 회귀분석
warning_signal 위험 신호 문자 명목 안정/주의/고위험 개선 제안 분류

peer_q3 학교에서 소속감을 느낀다 1~5 리커트 높을수록 관계 양호
support_q1 가족은 나의 고민을 들어준다 1~5 리커트 높을수록 지원감 높음
support_q2 학교에는 도움을 요청할 수 있는 선생님이나 상담 자원이 있다 1~5 리커트 높을수록 지원감 높음
support_q3 나는 힘들 때 도움을 받을 수 있다고 느낀다 1~5 리커트 높을수록 지원감 높음

SPSS_Steps

SPSS_Steps: 10행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 가져오기 전체 변수 파일 → 열기 → 데이터 문자형 ID와 범주형 변수 측도 확인
2. 빈도분석 학년, 시험기간, 스트레스 수준 분포 확인 grade, exam_period, stress_level 분석 → 기술통계량 → 빈도분석 고위험 학생 비율 확인
3. 기술통계 수면, 운동, 학원, 스마트폰, 스트레스 평균 확인 sleep_hours, exercise_min, academy_hours, smartphone_min, stress_score 분석 → 기술통계량 → 기술통계 평균과 표준편차로 전체 상태 진단
4. 독립표본 t검정 시험기간 여부에 따른 스트레스 차이 확인 exam_period, stress_score 분석 → 평균비교 → 독립표본 t 검정 시험기간 효과가 통계적으로 유의한지 확인
5. 상관분석 스트레스와 관련 요인의 방향·강도 확인 sleep_hours, academy_hours, smartphone_min, career_anxiety_score, peer_relation_score, support_score, stress_score 분석 → 상관분석 → 이변량 상관계수 수면은 음의 상관, 진로불안은 양의 상관인지 확인
6. 회귀분석 스트레스 점수 예측요인 확인 stress_score, sleep_hours, exercise_min, academy_hours, career_anxiety_score, peer_relation_score, support_score, exam_period 분석 → 회귀분석 → 선형 표준화계수 β와 유의확률로 중요 변수 해석
7. 로지스틱 회귀 고스트레스 여부 예측 high_stress와 주요 독립변수 분석 → 회귀분석 → 이분형 로지스틱 Exp(B)로 위험요인 해석
8. 군집분석 학생 스트레스 유형 분류 sleep_hours, academy_hours, career_anxiety_score, peer_relation_score, support_score, stress_score 분석 → 분류분석 → K-평균 군집분석 고위험·주의·안정 유형으로 해석
9. 그래프 작성 결과 시각화 sleep_hours, stress_score, exam_period 그래프 → 차트 작성기 수면-스트레스 산점도와 시험기간별 막대그래프 제시
