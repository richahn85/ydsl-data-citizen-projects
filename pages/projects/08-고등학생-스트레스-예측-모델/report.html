고등학생 스트레스 예측 모델

시험기간·수면·운동·학원시간·진로불안·친구관계 데이터를 활용한 스트레스 위험 신호 체크리스트

1. 프로젝트 개요

문제의식: 학생 스트레스는 단순히 개인의 성격이나 의지 문제가 아니라, 시험기간, 수면 부족, 운동 부족, 학원시간, 진로불안, 친구관계, 가정·학교의 지원감이 함께 작동하는 생활 시스템의 결과일 수 있다.

왜 중요한가: 스트레스를 성적관리의 부수적 현상으로만 보면 고위험 신호를 놓치기 쉽다. 이 프로젝트는 학생을 성적 데이터로만 보지 않고 삶의 조건을 함께 보는 People First 관점의 데이터 분석이다.

연결 전공: 심리학, 보건학, AI, 교육학, 데이터사이언스

2. 문제 정의 카드

질문 답변
내가 본 문제는 무엇인가? 시험기간이 되면 학생들의 불안, 피로, 예민함이 눈에 띄게 늘어나지만 그 원인이 명확히 분석되지 않는다.
왜 중요한가? 스트레스가 누적되면 학습 몰입, 친구관계, 수면, 건강이 함께 흔들릴 수 있다.
어떤 데이터를 사용할 것인가? 수면시간, 운동시간, 학원시간, 학습시간, 스마트폰 사용시간, 진로불안, 친구관계, 지원감, 스트레스 설문.
어떤 분석을 할 것인가? 기술통계, t검정, 상관분석, 회귀분석, 로지스틱 회귀, 군집분석, 위험지수 개발.
누구에게 제안할 것인가? 학교 상담부, 담임교사, 학년부, 학부모회, 학생자치회.
전공 관심과 어떻게 연결되는가? 심리·보건·AI를 결합해 학생의 위험 신호를 조기 발견하고 지원하는 모델로 확장할 수 있다.

3. 데이터 설명

본 시범 데이터는 익명 학생 50명의 설문 응답을 가정해 만든 시범용 가상 데이터다. 실제 연구에서는 학교 자체 익명 설문과 질병관리청 청소년건강행태조사, 청소년통계 자료를 결합해 전국 기준과 비교할 수 있다.

4. 설문 문항 설계

모든 핵심 심리 문항은 1=전혀 그렇지 않다, 2=그렇지 않다, 3=보통이다, 4=그렇다, 5=매우 그렇다로 코딩한다. 친구관계와 지원감 문항은 점수가 높을수록 보호요인이 강한 것으로 해석한다.

stress_q2 해야 할 일이 많아 마음이 쉴 틈이 없다고 느낀다 1~5 리커트

5. 분석 방법

분석 단계 분석 목적 SPSS 메뉴 경로
1. 데이터 불러오기 CSV/XLSX를 SPSS로 가져오기 파일 → 열기 → 데이터
2. 빈도분석 학년, 시험기간, 스트레스 수준 분포 확인 분석 → 기술통계량 → 빈도분석
3. 기술통계 수면, 운동, 학원, 스마트폰, 스트레스 평균 확인 분석 → 기술통계량 → 기술통계
4. 독립표본 t검정 시험기간 여부에 따른 스트레스 차이 확인 분석 → 평균비교 → 독립표본 t 검정
5. 상관분석 스트레스와 관련 요인의 방향·강도 확인 분석 → 상관분석 → 이변량 상관계수
6. 회귀분석 스트레스 점수 예측요인 확인 분석 → 회귀분석 → 선형
7. 로지스틱 회귀 고스트레스 여부 예측 분석 → 회귀분석 → 이분형 로지스틱

6. 분석 결과 예시

시범 데이터의 전체 평균 스트레스 점수는 4.04점이며, 고스트레스 학생 비율은 92.0%로 나타났다. 시험기간 평균 스트레스는 4.36점, 평상시 평균 스트레스는 3.75점으로 시험기간의 부담이 더 높게 나타났다.

상관분석에서는 진로불안과 스트레스의 상관이 r=0.51, 수면시간과 스트레스의 상관이 r=-0.44, 지원감과 스트레스의 상관이 r=-0.19로 나타났다. 이는 스트레스가 단일 원인보다 생활·심리·관계 요인의 결합으로 형성된다는 점을 보여준다.

9. 한계점과 후속 연구

본 데이터는 시범용 가상 데이터이므로 실제 학생의 심리상태를 진단하는 데 사용할 수 없다. 실제 연구에서는 익명성, 자발적 참여, 상담 연계 기준, 개인정보 보호를 명확히 해야 한다. 후속 연구에서는 2~4주간 반복 측정 자료를 수집해 스트레스 변화 패턴을 분석할 수 있다.

10. 나의 진로·전공 관심과의 연결

이 프로젝트는 학생 스트레스를 개인의 나약함으로 판단하지 않고, 수면, 학업, 관계, 진로, 지원체계가 결합된 시스템 문제로 이해하려는 시도다. 앞으로 심리학, 보건학, 교육학, AI를 결합해 학생을 더 빨리 돕는 예측모델을 설계하고 싶다.

11. 에세이형 성찰 문장

나는 스트레스를 참고 견뎌야 하는 개인의 문제로만 보았지만, 데이터를 분석하면서 스트레스가 학생을 둘러싼 시간표, 관계, 진로불안, 수면 조건이 함께 만든 결과일 수 있음을 알게 되었다. 데이터는 사람을 평가하기 위한 도구가 아니라, 사람이 더 무너지기 전에 돕기 위한 신호를 발견하는 도구가 되어야 한다.
