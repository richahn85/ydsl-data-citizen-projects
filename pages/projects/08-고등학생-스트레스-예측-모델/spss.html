SPSS 분석 가이드: 고등학생 스트레스 예측 모델

이 문서는 08_dataset.xlsx 또는 08_dataset.csv를 SPSS에서 불러와 스트레스 예측모델을 분석하는 절차를 안내한다.

1. 데이터 불러오기

SPSS → 파일 → 열기 → 데이터 → 08_dataset.xlsx 또는 08_dataset.csv 선택. Excel 파일은 첫 행을 변수명으로 읽어오도록 체크한다.

2. 변수 측도 설정

변수 권장 SPSS 측도
student_id 명목
grade 서열
gender, exam_period, high_stress 명목
sleep_hours, exercise_min, academy_hours, stress_score, risk_index 척도
stress_level 서열

3. 분석 절차

분석 단계 목적 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 가져오기 파일 → 열기 → 데이터 문자형 ID와 범주형 변수 측도 확인
2. 빈도분석 학년, 시험기간, 스트레스 수준 분포 확인 분석 → 기술통계량 → 빈도분석 고위험 학생 비율 확인
3. 기술통계 수면, 운동, 학원, 스마트폰, 스트레스 평균 확인 분석 → 기술통계량 → 기술통계 평균과 표준편차로 전체 상태 진단
4. 독립표본 t검정 시험기간 여부에 따른 스트레스 차이 확인 분석 → 평균비교 → 독립표본 t 검정 시험기간 효과가 통계적으로 유의한지 확인
5. 상관분석 스트레스와 관련 요인의 방향·강도 확인 분석 → 상관분석 → 이변량 상관계수 수면은 음의 상관, 진로불안은 양의 상관인지 확인
6. 회귀분석 스트레스 점수 예측요인 확인 분석 → 회귀분석 → 선형 표준화계수 β와 유의확률로 중요 변수 해석
7. 로지스틱 회귀 고스트레스 여부 예측 분석 → 회귀분석 → 이분형 로지스틱 Exp(B)로 위험요인 해석
8. 군집분석 학생 스트레스 유형 분류 분석 → 분류분석 → K-평균 군집분석 고위험·주의·안정 유형으로 해석
9. 그래프 작성 결과 시각화 그래프 → 차트 작성기 수면-스트레스 산점도와 시험기간별 막대그래프 제시

4. Syntax 예시

FREQUENCIES VARIABLES=grade gender exam_period stress_level high_stress.

DESCRIPTIVES VARIABLES=sleep_hours exercise_min academy_hours smartphone_min career_anxiety_score peer_relation_score support_score stress_score risk_index.

T-TEST GROUPS=exam_period(0 1) /VARIABLES=stress_score.

CORRELATIONS /VARIABLES=sleep_hours exercise_min academy_hours smartphone_min career_anxiety_score peer_relation_score support_score stress_score.

REGRESSION /DEPENDENT stress_score /METHOD=ENTER sleep_hours exercise_min academy_hours smartphone_min career_anxiety_score peer_relation_score support_score exam_period grade.

LOGISTIC REGRESSION VARIABLES high_stress /METHOD=ENTER sleep_hours exercise_min academy_hours career_anxiety_score peer_relation_score support_score exam_period grade.

QUICK CLUSTER sleep_hours academy_hours career_anxiety_score peer_relation_score support_score stress_score /CRITERIA=CLUSTER(3).

5. 보고서 해석 문장 예시

상관분석 결과 진로불안은 스트레스와 정적 상관을 보였고, 수면시간과 지원감은 스트레스와 부적 상관을 보였다. 따라서 스트레스 완화 정책은 단순히 공부시간을 줄이는 문제가 아니라 수면 회복, 진로불안 완화, 관계·지원체계 강화가 함께 이루어져야 한다.
