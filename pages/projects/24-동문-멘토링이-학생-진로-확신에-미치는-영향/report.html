24번 주제: 동문 멘토링이 학생 진로 확신에 미치는 영향

프로젝트 산출물: 용문고 진로 멘토링 개선안 / 분석 방법: 전후 비교, 만족도 분석, 회귀분석

데이터 성격: 학교·동문회가 실제로 수집할 수 있는 학생 사전·사후 설문과 동문 멘토링 운영 기록을 반영한 시범용 가상 데이터

1. 핵심 요약

이 프로젝트는 동문 멘토링 전후에 학생의 진로 확신, 전공 이해도, 진로 불안, 실행계획이 어떻게 변화하는지 분석한다. 단순히 멘토링을 많이 했는가가 아니라, 동문과 학생의 관심 분야가 얼마나 잘 맞았는지, 학생이 멘토링 이후 어떤 행동을 했는지를 함께 본다.

시범 데이터에서 멘토링 참여집단의 평균 진로 확신 증가폭은 1.66점, 비교집단은 0.21점으로 나타났다. 이는 동문 멘토링이 정보 전달을 넘어 학생의 자기이해와 진로 실행계획 형성에 영향을 줄 수 있음을 보여준다.

2. 문제의식

학생들은 진로를 선택하라고 요구받지만, 실제 직무와 전공의 삶을 가까이서 만날 기회는 충분하지 않다. 학교 설명회나 인터넷 정보는 도움이 되지만, “그 길을 먼저 걸어간 사람”이 들려주는 현실적 경험은 학생의 불안을 낮추고 진로 확신을 높일 수 있다.

핵심 질문: 멘토링 전후 학생의 진로 확신은 어떻게 달라지는가? 어떤 멘토링 조건이 진로 확신 상승에 가장 큰 영향을 주는가?

3. 문제 정의 카드

질문 답변
내가 본 문제는 무엇인가? 진로 정보는 많지만 학생마다 접근 수준과 실제 직무 이해도는 다르다.
왜 중요한가? 진로 선택은 입시 전략을 넘어 학생의 삶의 방향을 정하는 결정이기 때문이다.
어떤 데이터를 사용할 것인가? 사전·사후 진로 확신 설문, 멘토링 횟수, 분야 일치도, 만족도, 희망 전공 변화, 실행계획 점수
어떤 분석을 할 것인가? 대응표본 t검정, 참여/비교집단 차이분석, 상관분석, 회귀분석, 로지스틱 회귀분석
누구에게 제안할 것인가? 용문고, 동문회, 학부모회, 진로진학부, 용문포럼
전공 관심과 어떻게 연결되는가? 교육학·HRD·심리학·경영학 관점에서 사람의 성장과 네트워크 기반 학습을 분석한다.

6. 분석 방법

분석 단계 분석 목적 SPSS 메뉴 해석 포인트
1. 데이터 불러오기 Excel/CSV 데이터를 SPSS로 불러오기 파일 → 열기 → 데이터 첫 행을 변수명으로 읽고 측도를 확인한다.
2. 변수계산 confidence_gain, clarity_gain, anxiety_reduction 계산 변환 → 변수 계산 사후-사전 변화량과 불안 감소량을 생성한다.
3. 기술통계 진로 확신·불안·만족도의 평균 파악 분석 → 기술통계량 → 기술통계 평균 변화가 긍정적인지 확인한다.
4. 대응표본 t검정 멘토링 전후 진로 확신 변화 검정 분석 → 평균비교 → 대응표본 t 검정 사후 점수가 사전보다 유의하게 높아졌는지 본다.
5. 독립표본 t검정 참여집단과 비교집단의 변화량 비교 분석 → 평균비교 → 독립표본 t 검정 멘토링 참여 여부에 따른 차이를 확인한다.
6. 상관분석 멘토링 횟수·일치도·만족도와 변화량 관계 확인 분석 → 상관분석 → 이변량 상관계수 횟수와 질적 매칭이 성과와 관련되는지 본다.
7. 회귀분석 진로 확신 변화 예측 분석 → 회귀분석 → 선형 어떤 요인이 진로 확신 상승을 가장 잘 설명하는지 본다.
8. 로지스틱 회귀분석 고상승 학생 예측 분석 → 회귀분석 → 이분형 로지스틱 고효과 멘토링 조건을 도출한다.
9. 그래프 작성 전후 변화, 유형별 차이 시각화 그래프 → 차트 작성기 보고서용 핵심 그래프를 만든다.

9. 한계점과 후속 연구

본 시범 데이터는 실제 학생과 동문의 응답이 아니라 분석 절차를 설명하기 위한 가상 데이터다. 실제 연구에서는 개인정보 보호 동의, 익명화, 멘토링 전후 동일 학생 매칭, 비교집단 설정이 필요하다.

후속 연구에서는 동문 멘토의 산업·직무 다양성, 멘토링 지속기간, 학생의 포트폴리오 변화, 대학 전공 선택 결과까지 추적하면 더 강한 분석이 가능하다.

10. 진로·전공 관심과의 연결

이 프로젝트는 교육학과 HRD의 관점에서 멘토링이라는 학습 경험이 학생의 자기효능감과 진로 확신에 미치는 영향을 분석한다. 경영학적으로는 동문 네트워크라는 학교의 무형자산을 어떻게 후배 성장 시스템으로 전환할 수 있는지 탐구한다.

11. 에세이형 성찰 문장

나는 진로 확신이 단순히 정보를 많이 얻는다고 생기는 것이 아니라, 나와 비슷한 고민을 지나간 사람과의 대화를 통해 형성될 수 있음을 알게 되었다. 데이터는 멘토링의 효과가 감동적인 경험담에만 머무르지 않고 측정 가능한 변화로 나타날 수 있음을 보여주었다.
