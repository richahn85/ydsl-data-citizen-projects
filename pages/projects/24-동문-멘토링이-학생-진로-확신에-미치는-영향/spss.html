24번 SPSS 분석 가이드: 동문 멘토링이 학생 진로 확신에 미치는 영향

분석 목표: 멘토링 전후 학생의 진로 확신 변화가 유의한지 확인하고, 어떤 요인이 변화량을 설명하는지 분석한다.

1. 데이터 불러오기와 변수 확인

① SPSS 실행 → 파일 → 열기 → 데이터 → 24_dataset.xlsx 선택 → Data 시트 선택 → 첫 행에서 변수 이름 읽기 체크

② 변수 보기에서 student_id, interest_field, participation_group, mentoring_type, recommendation_type은 명목, grade는 서열, 점수형 변수는 척도로 지정한다.

2. 변수 계산

변환 → 변수 계산에서 confidence_gain = post_career_confidence - pre_career_confidence를 만든다. 같은 방식으로 clarity_gain과 anxiety_reduction을 만든다.

3. 분석 절차표

단계 목적 메뉴 사용 변수 해석
1. 데이터 불러오기 Excel/CSV 데이터를 SPSS로 불러오기 파일 → 열기 → 데이터 전체 변수 첫 행을 변수명으로 읽고 측도를 확인한다.
2. 변수계산 confidence_gain, clarity_gain, anxiety_reduction 계산 변환 → 변수 계산 pre/post 변수 사후-사전 변화량과 불안 감소량을 생성한다.
3. 기술통계 진로 확신·불안·만족도의 평균 파악 분석 → 기술통계량 → 기술통계 pre/post, gain, satisfaction 평균 변화가 긍정적인지 확인한다.
4. 대응표본 t검정 멘토링 전후 진로 확신 변화 검정 분석 → 평균비교 → 대응표본 t 검정 pre_career_confidence, post_career_confidence 사후 점수가 사전보다 유의하게 높아졌는지 본다.
5. 독립표본 t검정 참여집단과 비교집단의 변화량 비교 분석 → 평균비교 → 독립표본 t 검정 participation_group, confidence_gain 멘토링 참여 여부에 따른 차이를 확인한다.
6. 상관분석 멘토링 횟수·일치도·만족도와 변화량 관계 확인 분석 → 상관분석 → 이변량 상관계수 mentoring_count, mentor_field_match, satisfaction, confidence_gain 횟수와 질적 매칭이 성과와 관련되는지 본다.
7. 회귀분석 진로 확신 변화 예측 분석 → 회귀분석 → 선형 confidence_gain, mentoring_count, mentor_field_match, satisfaction, reflection_score 어떤 요인이 진로 확신 상승을 가장 잘 설명하는지 본다.
8. 로지스틱 회귀분석 고상승 학생 예측 분석 → 회귀분석 → 이분형 로지스틱 high_gain_yn, mentoring_count, mentor_field_match 등 고효과 멘토링 조건을 도출한다.
9. 그래프 작성 전후 변화, 유형별 차이 시각화 그래프 → 차트 작성기 pre/post, mentoring_type, confidence_gain 보고서용 핵심 그래프를 만든다.

4. 보고서 해석 문장 예시

대응표본 t검정 결과 사후 진로 확신 평균이 사전 진로 확신 평균보다 높게 나타났다면, “동문 멘토링 후 학생들의 진로 확신은 향상된 것으로 해석할 수 있다”라고 서술한다.

회귀분석에서 mentoring_count와 mentor_field_match가 유의하다면, “멘토링의 양뿐 아니라 학생 관심 분야와 동문 전문성의 일치도가 중요하다”라고 해석한다.

* Project 24: Alumni mentoring and career confidence.
GET DATA /TYPE=XLSX /FILE='24_dataset.xlsx' /SHEET=name 'Data' /READNAMES=ON.
EXECUTE.

VARIABLE LEVEL grade(ORDINAL) mentoring_count(SCALE) mentoring_total_minutes(SCALE)
 pre_career_confidence(SCALE) post_career_confidence(SCALE) confidence_gain(SCALE)
 pre_major_clarity(SCALE) post_major_clarity(SCALE) clarity_gain(SCALE)
 pre_career_anxiety(SCALE) post_career_anxiety(SCALE) anxiety_reduction(SCALE)
 mentoring_satisfaction(SCALE) career_action_plan_score(SCALE) reflection_score(SCALE)
 portfolio_update_yn(NOMINAL) major_change_yn(NOMINAL) high_gain_yn(NOMINAL).

* If needed, recompute change variables.
COMPUTE confidence_gain = post_career_confidence - pre_career_confidence.
COMPUTE clarity_gain = post_major_clarity - pre_major_clarity.
COMPUTE anxiety_reduction = pre_career_anxiety - post_career_anxiety.
IF (confidence_gain >= .8) high_gain_yn = 1.
IF (confidence_gain < .8) high_gain_yn = 0.
EXECUTE.

DESCRIPTIVES VARIABLES=pre_career_confidence post_career_confidence confidence_gain mentoring_count mentor_field_match mentoring_satisfaction.

T-TEST PAIRS=pre_career_confidence WITH post_career_confidence (PAIRED).
T-TEST GROUPS=participation_group('참여' '비교') /VARIABLES=confidence_gain.

CORRELATIONS /VARIABLES=mentoring_count mentor_field_match mentoring_satisfaction reflection_score confidence_gain.

REGRESSION /DEPENDENT confidence_gain /METHOD=ENTER mentoring_count mentor_field_match mentoring_satisfaction reflection_score pre_career_confidence.

LOGISTIC REGRESSION VARIABLES high_gain_yn /METHOD=ENTER mentoring_count mentor_field_match mentoring_satisfaction reflection_score.
