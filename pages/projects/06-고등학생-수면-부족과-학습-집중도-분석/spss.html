3. 분석 절차 요약

분석 단계 SPSS 메뉴 핵심 해석
1. 데이터 불러오기 파일 → 열기 → 데이터 Data 시트 첫 행을 변수명으로 인식
2. 기술통계 분석 → 기술통계량 → 기술통계 수면 부족 정도와 집중도 수준의 기본 분포 확인
3. 빈도분석 분석 → 기술통계량 → 빈도분석 어떤 수면 유형이 많은지 확인
4. 상관분석 분석 → 상관분석 → 이변량 상관계수 수면시간과 집중도는 양의 상관, 스마트폰·스트레스는 음의 상관 기대
5. 집단 비교 분석 → 평균비교 → 독립표본 t 검정 수면 부족 집단의 집중도가 낮은지 확인
6. 회귀분석 분석 → 회귀분석 → 선형 어떤 요인이 집중도에 가장 큰 영향을 주는지 확인
7. 군집분석 분석 → 분류분석 → K-평균 군집분석 균형수면형, 스마트폰야행형, 학습과부하형 등으로 해석
8. 그래프 작성 그래프 → 차트 작성기 산점도와 유형별 평균 막대그래프 작성

4. 상관분석 해석

수면시간과 집중도 점수의 상관계수가 양수이면 수면시간이 길수록 집중도가 높아지는 경향이 있다는 의미이다. 스마트폰 사용시간과 집중도 점수의 상관계수가 음수이면 잠들기 전 스마트폰 사용이 많을수록 집중도가 낮아지는 경향이 있다는 의미이다.

5. 회귀분석 해석

종속변수는 concentration_score로 설정하고, 독립변수에는 weekday_sleep_hours, smartphone_before_sleep_min, stress_score, study_hours_after_school을 투입한다. 유의확률 p값이 .05 미만이면 해당 변수가 집중도 예측에 통계적으로 유의하다고 해석한다.

6. 군집분석 해석

K-평균 군집분석에서 군집 수를 4 또는 5로 설정한다. 각 군집의 평균 수면시간, 스마트폰 사용시간, 스트레스, 집중도 점수를 비교한 뒤 균형수면형, 스마트폰야행형, 학습과부하형, 주말보충형 등으로 이름을 붙인다.

7. 보고서 작성 문장 예시

상관분석 결과, 평일 수면시간과 집중도 점수는 정적 상관을 보였다(r=0.866). 이는 학생들의 학습 집중도가 단순한 의지의 문제가 아니라 수면시간과 관련될 수 있음을 시사한다. 따라서 학교는 공부시간 확대만 강조하기보다 수면 관리와 스마트폰 사용 습관 개선을 함께 지도할 필요가 있다.

* Project 06: High School Sleep Deficit and Learning Concentration.
GET DATA
  /TYPE=XLSX
  /FILE='06_dataset.xlsx'
  /SHEET=name 'Data'
  /READNAMES=ON.
EXECUTE.

* Derived variable: short sleep group.
COMPUTE short_sleep_group = 0.
IF (weekday_sleep_hours < 6) short_sleep_group = 1.
VARIABLE LABELS short_sleep_group '6시간 미만 수면집단(1=예, 0=아니오)'.
VALUE LABELS short_sleep_group 0 '6시간 이상' 1 '6시간 미만'.
EXECUTE.

FREQUENCIES VARIABLES=grade gender sleep_type recommendation_priority.
DESCRIPTIVES VARIABLES=weekday_sleep_hours weekend_sleep_hours smartphone_before_sleep_min stress_score concentration_score learning_risk_index.
CORRELATIONS
  /VARIABLES=weekday_sleep_hours smartphone_before_sleep_min stress_score concentration_score
  /PRINT=TWOTAIL NOSIG.
T-TEST GROUPS=short_sleep_group(0 1)
  /VARIABLES=concentration_score stress_score morning_sleepiness.
REGRESSION
  /DEPENDENT concentration_score
  /METHOD=ENTER weekday_sleep_hours smartphone_before_sleep_min stress_score study_hours_after_school academy_hours physical_activity_min.
QUICK CLUSTER weekday_sleep_hours smartphone_before_sleep_min sleep_quality morning_sleepiness stress_score
  /CRITERIA=CLUSTER(5) MXITER(10)
  /SAVE CLUSTER.
