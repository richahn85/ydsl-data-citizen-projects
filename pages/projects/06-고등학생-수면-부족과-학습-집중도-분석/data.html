변수명 변수 설명 측정 단위/값 SPSS 측도 실제 데이터 출처 비고
student_id 익명 학생 ID 문자 명목 시범 설문 개인 식별정보는 포함하지 않음
grade 학년 1~3 서열 학생 설문 1=1학년, 2=2학년, 3=3학년
gender 성별 남/여 명목 학생 설문 익명 처리
weekday_sleep_hours 평일 평균 수면 시간 시간 척도 학생 설문 최근 1주 기준
weekend_sleep_hours 주말 평균 수면 시간 시간 척도 학생 설문 최근 1주 기준
sleep_deficit_hours 권장 기준 대비 부족 수면 시간 척도 계산 변수 7.5시간 - 평일 수면시간, 음수는 0
weekend_catchup_hours 주말 보충수면 시간 시간 척도 계산 변수 주말 수면시간 - 평일 수면시간
bedtime_min 취침 시각 분 척도 학생 설문 자정 이후는 24시 이후 분으로 입력
wake_time_min 기상 시각 분 척도 학생 설문 00:00 기준 분 단위
smartphone_before_sleep_min 취침 전 스마트폰 사용 시간 분 척도 학생 설문 잠들기 전 2시간 이내 기준
study_hours_after_school 방과후 자기학습 시간 시간 척도 학생 설문 학교 수업 외 자기학습
academy_hours 학원·과외 시간 시간 척도 학생 설문 하루 평균
caffeine_after6_cups 오후 6시 이후 카페인 섭취량 잔 척도 학생 설문 커피/에너지음료/카페인 음료
physical_activity_min 하루 신체활동 시간 분 척도 학생 설문 걷기 포함 가능
breakfast_frequency 아침식사 빈도 1~5 서열 학생 설문 1=거의 안 먹음, 5=매일 먹음
sleep_quality 수면의 질 자기평가 1~5 서열 학생 설문 1=매우 나쁨, 5=매우 좋음
perceived_sleep_sufficiency 수면충분감 1~5 서열 학생 설문 1=전혀 충분하지 않음, 5=매우 충분함
morning_sleepiness 오전 졸림 정도 1~5 서열 학생 설문 1=전혀 졸리지 않음, 5=매우 졸림
stress_score 스트레스 정도 1~5 서열 학생 설문 1=매우 낮음, 5=매우 높음
focus_self_eval 학습 집중도 자기평가 1~5 서열 학생 설문 1=매우 낮음, 5=매우 높음
concentration_score 학습 집중도 점수 0~100 척도 계산/설문 혼합 집중도 자기평가와 수면 관련 변수를 반영한 시범 점수
academic_self_rating 학업 수행 자기평가 1~5 서열 학생 설문 성적 원자료 대신 자기평가 사용
late_night_smartphone 취침 전 스마트폰 고사용 여부 0/1 명목 계산 변수 90분 이상=1
sleep_type 수면 유형 문자 명목 군집 해석 변수 균형수면형/스마트폰야행형/학습과부하형/주말보충형/일반관리형
cluster_suggested 군집 번호 1~5 명목 군집분석 결과 대체 변수 SPSS K-평균 군집분석 결과와 비교
learning_risk_index 학습 집중 위험지수 0~100 척도 계산 변수 높을수록 집중도 관리 필요
recommendation_priority 개입 우선순위 낮음/보통/높음 서열 계산 변수 위험지수 기준

SPSS_Steps

SPSS_Steps: 9행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 XLSX/CSV를 SPSS로 불러오고 변수 측도를 확인한다. 전체 변수 파일 → 열기 → 데이터 Data 시트 첫 행을 변수명으로 인식
2. 기술통계 평균 수면시간, 스마트폰 사용시간, 집중도 평균을 파악한다. weekday_sleep_hours, smartphone_before_sleep_min, concentration_score, stress_score 분석 → 기술통계량 → 기술통계 수면 부족 정도와 집중도 수준의 기본 분포 확인
3. 빈도분석 수면 유형과 개입 우선순위 분포를 확인한다. sleep_type, recommendation_priority, grade, gender 분석 → 기술통계량 → 빈도분석 어떤 수면 유형이 많은지 확인
4. 상관분석 수면시간·스마트폰 사용·스트레스와 집중도의 관련성을 본다. weekday_sleep_hours, smartphone_before_sleep_min, stress_score, concentration_score 분석 → 상관분석 → 이변량 상관계수 수면시간과 집중도는 양의 상관, 스마트폰·스트레스는 음의 상관 기대
5. 집단 비교 6시간 미만 수면집단과 6시간 이상 집단의 집중도 차이를 본다. short_sleep_group, concentration_score 분석 → 평균비교 → 독립표본 t 검정 수면 부족 집단의 집중도가 낮은지 확인
6. 회귀분석 집중도를 설명하는 주요 요인을 검증한다. 종속: concentration_score / 독립: weekday_sleep_hours, smartphone_before_sleep_min, stress_score, study_hours_after_school 분석 → 회귀분석 → 선형 어떤 요인이 집중도에 가장 큰 영향을 주는지 확인
