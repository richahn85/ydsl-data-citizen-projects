career_clarity 진로 명확성 1~5점 서열 1=매우 낮음, 5=매우 높음 학생 설문

skill_ai AI 역량 관심도 1~5점 서열 AI·SW 관련 학습 관심 학생 설문
skill_data 데이터 역량 관심도 1~5점 서열 데이터 분석 학습 관심 학생 설문
skill_business 비즈니스 역량 관심도 1~5점 서열 경영·창업·마케팅 관심 학생 설문
skill_communication 커뮤니케이션 역량 관심도 1~5점 서열 발표·글쓰기·대인관계 관심 학생 설문
alumni_id 익명 동문 ID 문자 명목 개인정보 제거 동문 번호 동문 설문
alumni_cohort 동문 기수 숫자 서열 예: 23기, 33기 동문 설문
alumni_major 동문 전공 계열 범주 명목 동문의 대학 전공 계열 동문 설문
alumni_industry 동문 종사 산업 범주 명목 동문이 현재 일하는 산업 분야 동문 설문
alumni_job_field 동문 직무 범주 명목 동문의 현재 직무 분야 동문 설문
mentoring_area 멘토링 가능 분야 문자 명목 동문이 도울 수 있는 분야 동문 설문
monthly_available_hours 월 멘토링 가능 시간 시간 척도 월 기준 멘토링 제공 가능 시간 동문 설문
online_possible 온라인 멘토링 가능 여부 0/1 명목 0=불가능, 1=가능 동문 설문
mentor_network_degree 동문 네트워크 연결도 점수 척도 동문이 연결 가능한 멘토/분야의 수를 반영 동문회 내부자료/시범값
industry_match 산업 일치 여부 0/1 명목 학생 관심 산업과 동문 산업 일치 여부 계산 변수
major_match 전공 일치 여부 0/1 명목 학생 희망 전공과 동문 전공 일치 여부 계산 변수
career_match 직무 일치 여부 0/1 명목 학생 희망 직무와 동문 직무 일치 여부 계산 변수
skill_match_score 역량 관심-멘토링 분야 적합도 0~100 환산 일부 척도 학생 관심 역량과 동문 멘토링 분야의 적합도 계산 변수
availability_score 동문 멘토링 가능성 점수 0~10점 척도 월 가능 시간과 온라인 가능 여부 반영 계산 변수
matching_score 동문-학생 매칭 적합도 0~100점 척도 산업·전공·직무·역량·가용성·필요도 가중합 계산 변수
tie_strength_score 관계 강도 점수 0~100점 척도 매칭점수와 동문 연결도, 학생 필요도 결합 계산 변수
expected_career_help 예상 진로 도움 정도 1~5점 척도 매칭 시 학생에게 기대되는 도움 정도 계산 변수
student_match_rank 학생별 후보 순위 1~5 서열 학생별 매칭점수 순위 계산 변수
recommended_pair 추천 매칭 여부 0/1 명목 0=비추천, 1=우선 추천 계산 변수
recommended_action 추천 지원 방식 범주 명목 1:1 멘토링/그룹 멘토링/정보세션 계산 변수

SPSS_Steps

SPSS_Steps: 10행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 학생-동문 후보 매칭 데이터 확인 23_dataset.csv 또는 Data 시트 파일 → 열기 → 데이터 문자형 변수와 숫자형 변수를 구분한다.
2. 빈도분석 학생 관심 산업과 동문 산업의 분포 파악 industry_interest, alumni_industry, recommended_action 분석 → 기술통계량 → 빈도분석 관심 수요와 공급이 어떤 산업에 몰려 있는지 본다.
3. 기술통계 매칭점수와 관계강도 수준 확인 matching_score, tie_strength_score, expected_career_help 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최대/최소로 매칭 품질을 진단한다.
4. 교차분석 관심 산업과 추천 방식의 관계 확인 industry_interest × recommended_action 분석 → 기술통계량 → 교차분석 특정 산업 관심 학생에게 어떤 지원 방식이 필요한지 본다.
5. 상관분석 매칭점수와 예상 도움 정도의 관련성 확인 matching_score, tie_strength_score, expected_career_help, mentor_network_degree 분석 → 상관분석 → 이변량 상관계수 매칭점수가 높을수록 예상 진로 도움도 커지는지 확인한다.
6. 평균 비교 추천 여부에 따른 매칭점수 차이 검증 recommended_pair, matching_score 분석 → 평균비교 → 독립표본 t 검정 추천 후보와 비추천 후보의 점수 차이가 충분히 큰지 확인한다.
7. 회귀분석 예상 진로 도움 정도를 설명하는 요인 탐색 expected_career_help ← matching_score, mentor_network_degree, career_clarity, mentoring_need 분석 → 회귀분석 → 선형 무엇이 멘토링 효과를 높이는지 해석한다.
8. 군집분석 학생-멘토 매칭 유형 분류 matching_score, tie_strength_score, mentor_network_degree, mentoring_need 분석 → 분류분석 → K-평균 군집분석 고적합 1:1형, 그룹멘토링형, 정보세션형으로 분류한다.
9. 그래프 작성 네트워크형 산출물 제작 matching_score, alumni_id, student_id, industry_interest 그래프 → 차트 작성기 상위 매칭쌍과 산업별 부족 분야를 시각화한다.
