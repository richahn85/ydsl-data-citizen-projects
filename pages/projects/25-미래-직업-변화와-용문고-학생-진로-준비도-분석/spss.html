25번 SPSS 분석 가이드 미래 직업 변화와 Career Readiness 분석

1. 데이터 불러오기

SPSS에서 파일 → 열기 → 데이터 → 25_dataset.xlsx 또는 25_dataset.csv를 선택한다. Excel 파일을 사용할 경우 첫 행을 변수명으로 읽어오도록 설정한다.

2. 변수 보기 설정

student_id, hope_job, job_family, hope_major, gap_level, roadmap_recommendation은 명목 또는 서열로, 각 점수형 변수는 척도로 설정한다.

3. 권장 분석 순서

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 빈도분석 희망 직업·직업군 분포 확인 hope_job, job_family, gap_level 분석 → 기술통계량 → 빈도분석 학생들이 어떤 직업군에 몰려 있는지 확인
2. 기술통계 Career Readiness와 역량 갭 평균 확인 career_readiness_score, skill_gap_score, ai_era_fit_score 분석 → 기술통계량 → 기술통계 평균·표준편차·최댓값·최솟값 확인
3. 교차분석 직업군별 역량 갭 등급 차이 확인 job_family × gap_level 분석 → 기술통계량 → 교차분석 고위험 갭이 많은 직업군 파악
4. 일원배치 분산분석 직업군별 평균 준비도 차이 검증 job_family, career_readiness_score 분석 → 평균비교 → 일원배치 분산분석 직업군에 따라 준비도 차이가 있는지 확인
5. 상관분석 AI 노출도·준비도·갭 간 관계 확인 ai_exposure_score, career_readiness_score, skill_gap_score 분석 → 상관분석 → 이변량 상관계수 AI 노출도가 높을수록 갭이 커지는지 탐색
6. 회귀분석 AI-Era Fit Score 예측 요인 확인 ai_era_fit_score ← AI·데이터·포트폴리오·정보접근성 분석 → 회귀분석 → 선형 어떤 역량이 적합도에 큰 영향을 주는지 해석
7. 군집분석 학생 진로 준비 유형 분류 ai_literacy_score, data_literacy_score, portfolio_score, skill_gap_score 분석 → 분류분석 → K-평균 군집분석 AI준비형, 정보탐색형, 포트폴리오보완형 등으로 명명
8. 그래프 로드맵과 역량 갭 시각화 job_family, skill_gap_score, roadmap_recommendation 그래프 → 차트 작성기 보고서용 핵심 시각화 작성

4. 결과 해석 문장 예시

직업군별 역량 갭의 평균을 비교한 결과, 특정 직업군에서 AI·데이터 역량 보완 필요성이 높게 나타났다.

AI 노출도가 높은 직업을 희망하는 학생이라도 AI 리터러시와 포트폴리오 수준이 높으면 AI-Era Fit Score는 높게 나타날 수 있다.

따라서 진로지도는 직업명 중심이 아니라 필요한 역량과 학생의 현재 준비도 사이의 차이를 줄이는 방향으로 설계되어야 한다.

* Project 25: 미래 직업 변화와 용문고 학생 Career Readiness 분석.
* CSV 파일을 SPSS에서 불러온 뒤 변수 측도를 설정한다.

GET DATA
  /TYPE=TXT
  /FILE='25_dataset.csv'
  /ENCODING='UTF8'
  /DELCASE=LINE
  /DELIMITERS=","
  /ARRANGEMENT=DELIMITED
  /FIRSTCASE=2
  /VARIABLES=
  student_id A8
  grade F1.0
  hope_job A30
  job_family A30
  hope_major A30
  future_growth_score F8.1
  ai_exposure_score F8.1
  skill_requirement_score F8.1
  human_centricity_score F8.1
  ai_literacy_score F8.1
  data_literacy_score F8.1
  creativity_score F8.1
  communication_score F8.1
  problem_solving_score F8.1
  adaptability_score F8.1
  career_info_access_score F8.1
  project_experience_count F8.0
  portfolio_score F8.1
  english_global_score F8.1
  career_anxiety_score F8.1
  career_readiness_score F8.1
  skill_gap_score F8.1
  ai_era_fit_score F8.1
  gap_level A10
  roadmap_recommendation A40.
EXECUTE.

FREQUENCIES VARIABLES=hope_job job_family gap_level roadmap_recommendation.
DESCRIPTIVES VARIABLES=career_readiness_score skill_gap_score ai_era_fit_score ai_exposure_score future_growth_score.
CROSSTABS /TABLES=job_family BY gap_level /CELLS=COUNT ROW.
ONEWAY career_readiness_score BY job_family /STATISTICS DESCRIPTIVES.
CORRELATIONS /VARIABLES=ai_exposure_score future_growth_score career_readiness_score skill_gap_score ai_era_fit_score
  /PRINT=TWOTAIL NOSIG.
REGRESSION
  /DEPENDENT ai_era_fit_score
  /METHOD=ENTER ai_literacy_score data_literacy_score problem_solving_score communication_score portfolio_score english_global_score career_info_access_score.
QUICK CLUSTER ai_literacy_score data_literacy_score portfolio_score skill_gap_score
  /CRITERIA=CLUSTER(3) MXITER(10)
  /SAVE CLUSTER.
