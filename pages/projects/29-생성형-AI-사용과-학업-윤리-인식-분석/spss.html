29번 SPSS 분석 가이드 생성형 AI 사용과 학업 윤리 인식 분석

이 문서는 29_dataset.xlsx 또는 29_dataset.csv를 SPSS에서 불러와 분석하는 절차를 안내한다. 모든 데이터는 시범용 가상 데이터이며, 실제 수업에서는 익명 설문으로 교체한다.

1. SPSS 데이터 불러오기

1) SPSS 실행 → 파일 → 열기 → 데이터 → 29_dataset.xlsx 선택

2) Excel 파일 읽기 창에서 ‘데이터 첫 행에서 변수 이름 읽어오기’를 체크한다.

3) 변수 보기에서 student_id와 ai_use_type은 명목, Likert 문항은 서열, 점수 변수는 척도로 설정한다.

2. 변수 계산

윤리적 AI 사용 점수와 학업 윤리 인식 점수, 오남용 위험점수는 데이터셋에 이미 포함되어 있다. 실제 설문에서 직접 계산하려면 아래 Syntax를 참고한다.

COMPUTE ethical_use_score = MEAN(disclosure_intention, citation_habit, fact_check_habit, privacy_awareness, plagiarism_awareness). COMPUTE academic_integrity_score = 20(0.25disclosure_intention + 0.25citation_habit + 0.20fact_check_habit + 0.15privacy_awareness + 0.15plagiarism_awareness). COMPUTE misuse_risk_score = 8boundary_confusion + 6assignment_pressure + 5peer_ai_norm + 2.2genai_use_freq_week + 8*(5-ethical_use_score) - 16. IF (misuse_risk_score < 0) misuse_risk_score = 0. IF (misuse_risk_score > 100) misuse_risk_score = 100. IF (misuse_risk_score >= 55) high_misuse_risk = 1. IF (misuse_risk_score < 55) high_misuse_risk = 0. EXECUTE.

3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 Excel/CSV를 SPSS로 불러와 변수 측도 확인 전체 변수 파일 → 열기 → 데이터 문자형 ID와 명목/서열/척도 구분을 확인
2. 빈도분석 AI 사용 유형과 고위험 여부 분포 확인 ai_use_type, high_misuse_risk, main_ai_purpose 분석 → 기술통계량 → 빈도분석 어떤 유형 학생이 많은지 파악
3. 기술통계 윤리점수와 오남용 위험점수의 평균 수준 확인 ethical_use_score, academic_integrity_score, misuse_risk_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최소·최대값 확인
4. 상관분석 AI 윤리 인식과 오남용 위험의 관계 확인 teacher_rule_clarity, citation_habit, fact_check_habit, boundary_confusion, misuse_risk_score 분석 → 상관분석 → 이변량 상관계수 규칙 명확성·검증 습관이 높을수록 위험이 낮은지 해석
5. 집단비교 AI 사용 유형별 윤리점수 차이 확인 ai_use_type, academic_integrity_score 분석 → 평균비교 → 일원배치 분산분석 Ethical active user와 Boundary-confused user 차이 확인
6. 회귀분석 오남용 위험점수 예측 요인 확인 misuse_risk_score, teacher_rule_clarity, ai_guidance_experience, citation_habit, fact_check_habit, assignment_pressure 분석 → 회귀분석 → 선형 오남용 위험을 낮추는 핵심 요인 도출
7. 로지스틱 회귀 고위험 학생 여부 예측 high_misuse_risk와 주요 예측변수 분석 → 회귀분석 → 이분형 로지스틱 고위험 가능성을 높이는 요인 해석
8. 군집분석 AI 사용·윤리 인식 유형 분류 genai_use_freq_week, teacher_rule_clarity, citation_habit, fact_check_habit, boundary_confusion 분석 → 분류분석 → K-평균 군집분석 학생 유형별 맞춤 교육안 설계
9. 시각화 AI 윤리 격차를 그래프로 설명 academic_integrity_score, misuse_risk_score, ai_use_type 그래프 → 차트 작성기 산점도와 막대그래프로 결과 전달

4. 핵심 해석 기준

• teacher_rule_clarity와 ai_guidance_experience가 높을수록 misuse_risk_score가 낮아지는지 확인한다.

• disclosure_intention, citation_habit, fact_check_habit이 academic_integrity_score를 높이는지 확인한다.

• high_misuse_risk가 1인 학생이 어떤 조건에서 많이 나타나는지 로지스틱 회귀분석으로 확인한다.

• AI 사용 빈도 자체보다 사용 방식과 학교 안내의 명확성이 더 중요한지 해석한다.

5. 보고서 작성 문장 예시

생성형 AI 사용과 학업 윤리 인식 간의 관계를 분석한 결과, AI 사용 빈도 자체보다 AI 사용 과정 공개, 출처 기록, 사실 검증 습관이 학업 윤리 인식과 더 밀접하게 연결되어 있었다. 이는 학교 AI 윤리 교육이 단순 금지보다 ‘어떻게 책임 있게 사용할 것인가’를 중심으로 설계되어야 함을 시사한다.

* Project 29: Generative AI Use and Academic Integrity Perception.
GET DATA
  /TYPE=XLSX
  /FILE='29_dataset.xlsx'
  /SHEET=name 'Data'
  /READNAMES=ON.
EXECUTE.

VARIABLE LEVEL student_id gender main_ai_purpose ai_use_type (NOMINAL).
VARIABLE LEVEL grade home_digital_access teacher_rule_clarity ai_guidance_experience disclosure_intention citation_habit fact_check_habit privacy_awareness plagiarism_awareness boundary_confusion ai_detector_trust assignment_pressure peer_ai_norm learning_gain_self_eval (ORDINAL).
VARIABLE LEVEL genai_use_freq_week ethical_use_score academic_integrity_score misuse_risk_score (SCALE).

FREQUENCIES VARIABLES=main_ai_purpose ai_use_type high_misuse_risk.
DESCRIPTIVES VARIABLES=genai_use_freq_week ethical_use_score academic_integrity_score misuse_risk_score.

CORRELATIONS
  /VARIABLES=teacher_rule_clarity ai_guidance_experience citation_habit fact_check_habit boundary_confusion assignment_pressure peer_ai_norm academic_integrity_score misuse_risk_score
  /PRINT=TWOTAIL NOSIG.

ONEWAY academic_integrity_score misuse_risk_score BY ai_use_type
  /STATISTICS DESCRIPTIVES
  /POSTHOC=TUKEY ALPHA(.05).

REGRESSION
  /DEPENDENT misuse_risk_score
  /METHOD=ENTER genai_use_freq_week teacher_rule_clarity ai_guidance_experience citation_habit fact_check_habit boundary_confusion assignment_pressure peer_ai_norm.

LOGISTIC REGRESSION VARIABLES high_misuse_risk
  /METHOD=ENTER genai_use_freq_week teacher_rule_clarity ai_guidance_experience citation_habit fact_check_habit boundary_confusion assignment_pressure peer_ai_norm
  /PRINT=CI(95).

QUICK CLUSTER genai_use_freq_week teacher_rule_clarity citation_habit fact_check_habit boundary_confusion
  /CRITERIA=CLUSTER(3) MXITER(10)
  /SAVE CLUSTER.
