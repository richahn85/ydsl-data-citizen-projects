변수명 변수 설명 측정 단위/범위 SPSS 측도 코딩/계산 방식 비고
student_id 익명 학생 ID 문자 명목 S001~S050 개인 식별 불가
grade 학년 1~3 서열 1=1학년, 2=2학년, 3=3학년
gender 성별 M/F 명목 M=남, F=여 선택/익명 처리
genai_use_freq_week 주간 생성형 AI 사용 횟수 회/주 척도 0~14
home_digital_access 가정 디지털 접근성 1~5점 서열 1=매우 낮음~5=매우 높음
paid_ai_tool 유료 AI 도구 접근 여부 0/1 명목 0=없음, 1=있음
main_ai_purpose 주요 AI 사용 목적 범주 명목 Brainstorm/Summary/Writing/Translation/Coding/NotUsed
teacher_rule_clarity 교사가 제시한 AI 사용 규칙 명확성 1~5점 서열 1=전혀 명확하지 않음~5=매우 명확
ai_guidance_experience AI 활용 교육 경험 1~5점 서열 1=전혀 없음~5=충분함
disclosure_intention AI 사용 사실을 밝히려는 의향 1~5점 서열 1=전혀 아님~5=매우 그렇다
citation_habit AI 활용 내용 출처·과정 기록 습관 1~5점 서열 1=전혀 없음~5=항상 기록
fact_check_habit AI 답변 검증 습관 1~5점 서열 1=검증하지 않음~5=항상 검증
privacy_awareness 개인정보 입력 위험 인식 1~5점 서열 1=낮음~5=높음
plagiarism_awareness 표절·대필 위험 인식 1~5점 서열 1=낮음~5=높음
boundary_confusion AI 활용과 부정행위 경계 혼란 1~5점 서열 1=혼란 낮음~5=혼란 높음
ai_detector_trust AI 탐지 도구 신뢰도 1~5점 서열 1=불신~5=매우 신뢰
assignment_pressure 과제 압박감 1~5점 서열 1=낮음~5=높음
peer_ai_norm 친구들의 AI 사용 일반화 정도 1~5점 서열 1=낮음~5=높음
learning_gain_self_eval AI 활용 학습효과 자기평가 1~5점 서열 1=낮음~5=높음
ethical_use_score 윤리적 AI 사용 점수 1~5점 척도 관련 5개 문항 평균 계산 변수
academic_integrity_score 학업 윤리 인식 점수 0~100점 척도 가중합 환산 계산 변수
misuse_risk_score AI 오남용 위험점수 0~100점 척도 경계혼란·압박·검증 부족 등 가중합 계산 변수
high_misuse_risk AI 오남용 고위험 여부 0/1 명목 0=일반, 1=고위험 55점 이상
ai_use_type AI 사용 윤리 유형 범주 명목 Non/Low user, Practical user, Ethical active user, Boundary-confused user 분류 변수

Q9 친구들이 AI를 활용하는 방식은 나의 사용 방식에 영향을 준다. 1~5 peer_ai_norm
Q10 과제 압박이 높을 때 AI에 과도하게 의존하고 싶은 마음이 커진다. 1~5 assignment_pressure
Q11 AI 사용 여부를 탐지 도구만으로 판단하는 것은 충분하다고 생각한다. 1~5 ai_detector_trust
Q12 AI를 적절히 사용하면 내 학습 이해에 도움이 된다. 1~5 learning_gain_self_eval

SPSS_Steps

SPSS_Steps: 10행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 Excel/CSV를 SPSS로 불러와 변수 측도 확인 전체 변수 파일 → 열기 → 데이터 문자형 ID와 명목/서열/척도 구분을 확인
2. 빈도분석 AI 사용 유형과 고위험 여부 분포 확인 ai_use_type, high_misuse_risk, main_ai_purpose 분석 → 기술통계량 → 빈도분석 어떤 유형 학생이 많은지 파악
3. 기술통계 윤리점수와 오남용 위험점수의 평균 수준 확인 ethical_use_score, academic_integrity_score, misuse_risk_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최소·최대값 확인
4. 상관분석 AI 윤리 인식과 오남용 위험의 관계 확인 teacher_rule_clarity, citation_habit, fact_check_habit, boundary_confusion, misuse_risk_score 분석 → 상관분석 → 이변량 상관계수 규칙 명확성·검증 습관이 높을수록 위험이 낮은지 해석
5. 집단비교 AI 사용 유형별 윤리점수 차이 확인 ai_use_type, academic_integrity_score 분석 → 평균비교 → 일원배치 분산분석 Ethical active user와 Boundary-confused user 차이 확인
6. 회귀분석 오남용 위험점수 예측 요인 확인 misuse_risk_score, teacher_rule_clarity, ai_guidance_experience, citation_habit, fact_check_habit, assignment_pressure 분석 → 회귀분석 → 선형 오남용 위험을 낮추는 핵심 요인 도출
7. 로지스틱 회귀 고위험 학생 여부 예측 high_misuse_risk와 주요 예측변수 분석 → 회귀분석 → 이분형 로지스틱 고위험 가능성을 높이는 요인 해석
8. 군집분석 AI 사용·윤리 인식 유형 분류 genai_use_freq_week, teacher_rule_clarity, citation_habit, fact_check_habit, boundary_confusion 분석 → 분류분석 → K-평균 군집분석 학생 유형별 맞춤 교육안 설계
9. 시각화 AI 윤리 격차를 그래프로 설명 academic_integrity_score, misuse_risk_score, ai_use_type 그래프 → 차트 작성기 산점도와 막대그래프로 결과 전달
