SPSS_Steps

SPSS_Steps: 8행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 빈도분석 소비 세그먼트와 추천 프로그램 분포 파악 consumer_segment_label, recommendation 분석 → 기술통계량 → 빈도분석 어떤 학생 소비 유형이 많은지 확인한다.
2. 기술통계 소비액·방문빈도·기여점수의 평균과 분산 확인 weekly_local_spend_krw, local_spend_share_pct, local_contribution_score 분석 → 기술통계량 → 기술통계 지역상권 기여 수준의 평균과 차이를 본다.
3. 상관분석 지역 소비비중과 기여점수의 관련성 확인 local_spend_share_pct, weekly_local_spend_krw, local_contribution_score 분석 → 상관분석 → 이변량 상관계수 지역 소비비중이 높을수록 기여점수가 높아지는지 해석한다.
4. 일원배치 분산분석 소비 유형별 지역상권 기여점수 차이 검증 consumer_segment_label, local_contribution_score 분석 → 평균비교 → 일원배치 분산분석 소비 유형별 평균 차이가 통계적으로 유의한지 확인한다.
5. 회귀분석 지역상권 기여점수를 예측하는 요인 탐색 local_spend_share_pct, after_school_visit_wk, teen_discount_need_1_5, co_creation_intention_1_5 분석 → 회귀분석 → 선형 방문빈도·지역 소비비중·참여의향 중 중요한 요인을 파악한다.
6. 군집분석 학생 소비 패턴을 데이터 기반으로 유형화 convenience_freq_wk, cafe_freq_wk, snack_freq_wk, study_space_freq_mo, local_spend_share_pct 분석 → 분류분석 → K-평균 군집분석 실속형, 카페형, 온라인형 등 군집명을 붙이고 맞춤 제안을 만든다.
7. 시각화 상권 구역별 기여도와 소비 유형 표현 lon, lat, local_contribution_score, zone_name 그래프 → 차트 작성기 → 산점도 학교 주변 어느 구역에서 학생 소비 기여도가 높은지 지도처럼 표현한다.
