3. 권장 분석 절차

분석 단계 SPSS 메뉴 해석 포인트
1. 빈도분석 분석 → 기술통계량 → 빈도분석 어떤 학생 소비 유형이 많은지 확인한다.
2. 기술통계 분석 → 기술통계량 → 기술통계 지역상권 기여 수준의 평균과 차이를 본다.
3. 상관분석 분석 → 상관분석 → 이변량 상관계수 지역 소비비중이 높을수록 기여점수가 높아지는지 해석한다.
4. 일원배치 분산분석 분석 → 평균비교 → 일원배치 분산분석 소비 유형별 평균 차이가 통계적으로 유의한지 확인한다.
5. 회귀분석 분석 → 회귀분석 → 선형 방문빈도·지역 소비비중·참여의향 중 중요한 요인을 파악한다.
6. 군집분석 분석 → 분류분석 → K-평균 군집분석 실속형, 카페형, 온라인형 등 군집명을 붙이고 맞춤 제안을 만든다.
7. 시각화 그래프 → 차트 작성기 → 산점도 학교 주변 어느 구역에서 학생 소비 기여도가 높은지 지도처럼 표현한다.

4. 결과 해석 예시

소비 유형별 평균 지역상권 기여점수를 비교했을 때, 지역 소비비중과 방문 빈도가 높은 유형은 지역상권 기여점수가 높게 나타날 가능성이 크다. 온라인 중심형 학생은 지역상권 기여점수가 낮더라도 쿠폰, 픽업, 학생 친화 정보 제공을 통해 지역 상권으로 유입될 수 있다.

5. SPSS Syntax 예시

GET DATA /TYPE=XLSX /FILE="27_dataset.xlsx" /SHEET=name "Data" /READNAMES=ON. EXECUTE. FREQUENCIES VARIABLES=consumer_segment_label recommendation. DESCRIPTIVES VARIABLES=weekly_local_spend_krw local_spend_share_pct local_contribution_score. CORRELATIONS /VARIABLES=local_spend_share_pct weekly_local_spend_krw after_school_visit_wk local_contribution_score. ONEWAY local_contribution_score BY consumer_segment_label /STATISTICS DESCRIPTIVES. REGRESSION /DEPENDENT local_contribution_score /METHOD=ENTER local_spend_share_pct weekly_local_spend_krw after_school_visit_wk teen_discount_need_1_5 co_creation_intention_1_5. QUICK CLUSTER convenience_freq_wk cafe_freq_wk snack_freq_wk study_space_freq_mo local_spend_share_pct /CRITERIA=CLUSTER(5) MXITER(10).
