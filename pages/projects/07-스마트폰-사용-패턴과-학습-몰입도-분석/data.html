변수명 변수 설명 측정 단위 SPSS 측도 코딩/값 비고
student_id 익명 학생 ID 문자 명목 S001~S050 개인 식별이 불가능한 익명 코드
grade 학년 1/2/3 서열 1=1학년, 2=2학년, 3=3학년 학년별 차이 분석
gender 성별 남/여 명목 남, 여 인구통계학적 변수
weekday_phone_min 평일 하루 스마트폰 총 사용시간 분 척도 0 이상 정수 자기보고 또는 스크린타임 기록
weekend_phone_min 주말 하루 스마트폰 총 사용시간 분 척도 0 이상 정수 평일 대비 사용량 비교
study_app_min 학습 앱/검색/강의 사용시간 분 척도 0 이상 정수 생산적 사용 지표
sns_min SNS 사용시간 분 척도 0 이상 정수 사회적 사용 지표
shortform_min 숏폼 영상 사용시간 분 척도 0 이상 정수 주의분산 가능성 지표
game_min 모바일 게임 사용시간 분 척도 0 이상 정수 오락 사용 지표
messenger_min 메신저 사용시간 분 척도 0 이상 정수 친구 소통 사용 지표
info_search_min 정보검색 사용시간 분 척도 0 이상 정수 학습·정보 탐색 보조
bedtime_phone_min 취침 전 스마트폰 사용시간 분 척도 0 이상 정수 수면 및 다음날 집중도 영향
night_check_freq 밤중 스마트폰 확인 횟수 회 척도 0 이상 정수 수면 방해 가능성 지표
notification_off 공부 중 알림 차단 여부 0/1 명목 0=아니오, 1=예 자기통제 행동
app_switch_count 공부 중 앱 전환 횟수 회 척도 0 이상 정수 주의 전환 빈도
learning_time_min 하루 자기주도 학습시간 분 척도 0 이상 정수 학습 투입시간
sleep_hours 평일 평균 수면시간 시간 척도 소수 1자리 통제변수
study_app_ratio 학습 앱 사용 비율 비율 척도 study_app_min / weekday_phone_min 사용 방식 핵심 지표
distraction_score 주의분산 점수 0~100 척도 높을수록 분산 큼 종속/매개 변수
self_control_score 디지털 자기조절 점수 0~100 척도 높을수록 자기조절 높음 핵심 예측 변수
concentration_score 학습 집중도 점수 0~100 척도 높을수록 집중도 높음 주요 종속변수
flow_score 학습 몰입도 점수 0~100 척도 높을수록 몰입도 높음 주요 종속변수
assignment_delay_score 과제 지연 정도 1~5 서열 1=낮음, 5=높음 학습행동 결과
overuse_risk_score 스마트폰 과의존 위험점수 0~100 척도 높을수록 위험 높음 위험군 분류 보조
phone_type_code 스마트폰 사용 유형 코드 1~5 명목 1=학습도구형,2=균형형,3=SNS·숏폼형,4=게임형,5=고위험분산형 군집/분류 결과
phone_type 스마트폰 사용 유형 문자 명목 학습도구형 등 보고서 해석용
high_immersion 고몰입 여부 0/1 명목 0=보통 이하, 1=고몰입 로지스틱 회귀 종속변수

SPSS_Steps

SPSS_Steps: 11행 x 5열 / 미리보기 10행 x 5열

SPSS Analysis Steps
분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러오기 전체 변수 파일 → 열기 → 데이터 문자형/수치형 변수와 측도 확인
2. 빈도분석 성별·학년·사용 유형 분포 확인 grade, gender, phone_type_code 분석 → 기술통계량 → 빈도분석 표본 구성과 유형별 빈도 확인
3. 기술통계 사용시간·집중도·자기조절 평균 확인 weekday_phone_min, shortform_min, bedtime_phone_min, concentration_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최대/최소값 확인
4. 상관분석 사용 방식과 집중도의 관련성 탐색 study_app_ratio, shortform_min, bedtime_phone_min, sleep_hours, concentration_score 분석 → 상관분석 → 이변량 상관계수 전체 사용시간보다 사용 방식이 더 중요한지 확인
5. 회귀분석 학습 집중도 예측 요인 분석 종속: concentration_score / 독립: study_app_ratio, shortform_min, bedtime_phone_min, sleep_hours, self_control_score 분석 → 회귀분석 → 선형 어떤 요인이 집중도를 설명하는지 확인
6. 군집분석 학생의 스마트폰 사용 유형 분류 study_app_ratio, sns_min, shortform_min, game_min, bedtime_phone_min, self_control_score 분석 → 분류분석 → K-평균 군집분석 학습도구형, 균형형, SNS·숏폼형, 게임형, 고위험분산형 도출
7. 분산분석 사용 유형별 집중도 차이 검증 독립: phone_type_code / 종속: concentration_score, flow_score 분석 → 평균비교 → 일원배치 분산분석 유형별 평균 차이가 유의한지 확인
8. 로지스틱 회귀 고몰입 여부 예측 종속: high_immersion / 독립: 사용 방식 변수 분석 → 회귀분석 → 이분형 로지스틱 고몰입 가능성을 높이는 요인 확인
