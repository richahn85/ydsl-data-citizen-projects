SPSS 분석 가이드

이 가이드는 07_dataset.xlsx 또는 07_dataset.csv를 SPSS에서 불러와 분석하는 절차를 설명한다. 분석의 핵심은 총 사용시간보다 사용 방식이 학습 몰입도를 더 잘 설명하는지 확인하는 것이다.

1. 데이터 불러오기와 변수 확인

SPSS에서 파일 → 열기 → 데이터를 선택한 뒤 07_dataset.xlsx의 Data 시트를 불러온다. 첫 행을 변수명으로 읽고, 변수 보기에서 grade, gender, phone_type_code, high_immersion은 명목 또는 서열, 사용시간·점수 변수는 척도로 설정한다.

2. 분석 단계별 메뉴

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러오기 전체 변수 파일 → 열기 → 데이터 문자형/수치형 변수와 측도 확인
2. 빈도분석 성별·학년·사용 유형 분포 확인 grade, gender, phone_type_code 분석 → 기술통계량 → 빈도분석 표본 구성과 유형별 빈도 확인
3. 기술통계 사용시간·집중도·자기조절 평균 확인 weekday_phone_min, shortform_min, bedtime_phone_min, concentration_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최대/최소값 확인
4. 상관분석 사용 방식과 집중도의 관련성 탐색 study_app_ratio, shortform_min, bedtime_phone_min, sleep_hours, concentration_score 분석 → 상관분석 → 이변량 상관계수 전체 사용시간보다 사용 방식이 더 중요한지 확인
5. 회귀분석 학습 집중도 예측 요인 분석 종속: concentration_score / 독립: study_app_ratio, shortform_min, bedtime_phone_min, sleep_hours, self_control_score 분석 → 회귀분석 → 선형 어떤 요인이 집중도를 설명하는지 확인
6. 군집분석 학생의 스마트폰 사용 유형 분류 study_app_ratio, sns_min, shortform_min, game_min, bedtime_phone_min, self_control_score 분석 → 분류분석 → K-평균 군집분석 학습도구형, 균형형, SNS·숏폼형, 게임형, 고위험분산형 도출
7. 분산분석 사용 유형별 집중도 차이 검증 독립: phone_type_code / 종속: concentration_score, flow_score 분석 → 평균비교 → 일원배치 분산분석 유형별 평균 차이가 유의한지 확인
8. 로지스틱 회귀 고몰입 여부 예측 종속: high_immersion / 독립: 사용 방식 변수 분석 → 회귀분석 → 이분형 로지스틱 고몰입 가능성을 높이는 요인 확인
9. 그래프 작성 핵심 결과 시각화 study_app_ratio, concentration_score, phone_type_code 그래프 → 차트 작성기 산점도·막대그래프를 보고서에 삽입

3. 권장 해석 순서

• 기술통계로 전체 사용시간과 집중도 평균을 먼저 확인한다.

• 상관분석으로 총 사용시간, 학습 앱 비율, 숏폼 사용시간, 취침 전 사용시간이 집중도와 어떤 관련이 있는지 확인한다.

• 회귀분석으로 집중도에 영향을 미치는 변수를 동시에 투입한다.

• K-평균 군집분석으로 학생 사용 유형을 분류하고 유형별 집중도 차이를 분산분석으로 검증한다.

• 결과를 학생 비난이 아니라 디지털 습관 개선안으로 연결한다.

4. SPSS Syntax 예시

GET DATA /TYPE=XLSX /FILE='07_dataset.xlsx' /SHEET=name 'Data' /READNAMES=ON.

EXECUTE.

FREQUENCIES VARIABLES=grade gender phone_type_code high_immersion.

DESCRIPTIVES VARIABLES=weekday_phone_min weekend_phone_min study_app_ratio shortform_min bedtime_phone_min sleep_hours concentration_score flow_score distraction_score self_control_score overuse_risk_score.

CORRELATIONS /VARIABLES=weekday_phone_min study_app_ratio shortform_min game_min bedtime_phone_min sleep_hours self_control_score concentration_score flow_score.

REGRESSION /DEPENDENT concentration_score /METHOD=ENTER study_app_ratio shortform_min bedtime_phone_min sleep_hours self_control_score weekday_phone_min.

QUICK CLUSTER study_app_ratio sns_min shortform_min game_min bedtime_phone_min self_control_score /CRITERIA=CLUSTER(5) MXITER(10) /SAVE CLUSTER.

ONEWAY concentration_score flow_score BY phone_type_code /STATISTICS DESCRIPTIVES /POSTHOC=TUKEY.

LOGISTIC REGRESSION VARIABLES high_immersion /METHOD=ENTER study_app_ratio shortform_min bedtime_phone_min sleep_hours self_control_score overuse_risk_score.
