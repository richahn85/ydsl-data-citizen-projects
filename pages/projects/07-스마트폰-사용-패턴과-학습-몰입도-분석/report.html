학생 유형별 디지털 습관 개선안

본 보고서는 시범용 가상 데이터를 활용한 고등학생 데이터 기반 문제해결 프로젝트 예시이다. 실제 수업에서는 익명 설문, 스크린타임 기록, 청소년건강행태조사 등 공식 통계와 학생 자체 조사를 결합하여 분석을 고도화할 수 있다.

1. 프로젝트 개요

항목 내용
문제의식 스마트폰 사용 시간이 아니라 사용 방식이 학습 몰입도를 더 크게 좌우할 수 있는가
핵심 데이터 스마트폰 총 사용시간, 앱 유형별 사용시간, 취침 전 사용시간, 학습시간, 수면시간, 집중도 자기평가
분석 방법 빈도분석, 기술통계, 상관분석, 회귀분석, 군집분석, 분산분석, 로지스틱 회귀
산출물 학생 유형별 디지털 습관 개선안
연결 전공 컴퓨터과학, 심리학, HCI, 교육공학

2. 문제 정의 카드

질문 답변
내가 본 문제 많은 학생이 스마트폰을 오래 사용하지만, 어떤 학생은 학습 도구로 활용하고 어떤 학생은 숏폼·SNS·게임으로 인해 집중이 흐트러진다.
왜 중요한가 학생을 단순히 “스마트폰을 많이 쓰는 학생”으로 판단하면 실제 학습 습관의 차이를 놓친다.
사용 데이터 익명 설문, 앱 유형별 사용시간, 취침 전 사용시간, 수면시간, 집중도 자기평가, 자기조절 행동
분석 방법 상관분석으로 관련성을 확인하고, 회귀분석으로 집중도 예측 요인을 찾으며, 군집분석으로 사용 유형을 분류한다.
제안 대상 학교, 담임교사, 학생자치회, 학부모회, 상담부
전공 연결 컴퓨터과학과 심리학을 연결하여 디지털 환경이 인간의 주의력과 학습 행동에 미치는 영향을 분석한다.

3. 데이터 설계

분석 단위는 익명 학생 1명이며, 시범 데이터는 총 50명 응답으로 구성하였다. 실제 조사에서는 Google Forms 또는 학교 자체 설문으로 수집한 뒤 SPSS에서 분석한다.

5. 분석 방법

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러오기 전체 변수 파일 → 열기 → 데이터
2. 빈도분석 성별·학년·사용 유형 분포 확인 grade, gender, phone_type_code 분석 → 기술통계량 → 빈도분석
3. 기술통계 사용시간·집중도·자기조절 평균 확인 weekday_phone_min, shortform_min, bedtime_phone_min, concentration_score 분석 → 기술통계량 → 기술통계
4. 상관분석 사용 방식과 집중도의 관련성 탐색 study_app_ratio, shortform_min, bedtime_phone_min, sleep_hours, concentration_score 분석 → 상관분석 → 이변량 상관계수
5. 회귀분석 학습 집중도 예측 요인 분석 종속: concentration_score / 독립: study_app_ratio, shortform_min, bedtime_phone_min, sleep_hours, self_control_score 분석 → 회귀분석 → 선형
6. 군집분석 학생의 스마트폰 사용 유형 분류 study_app_ratio, sns_min, shortform_min, game_min, bedtime_phone_min, self_control_score 분석 → 분류분석 → K-평균 군집분석
7. 분산분석 사용 유형별 집중도 차이 검증 독립: phone_type_code / 종속: concentration_score, flow_score 분석 → 평균비교 → 일원배치 분산분석
8. 로지스틱 회귀 고몰입 여부 예측 종속: high_immersion / 독립: 사용 방식 변수 분석 → 회귀분석 → 이분형 로지스틱
9. 그래프 작성 핵심 결과 시각화 study_app_ratio, concentration_score, phone_type_code 그래프 → 차트 작성기

8. 한계점과 후속 연구

본 시범 데이터는 가상 데이터이므로 실제 학생의 생활을 완전히 대표하지 않는다. 실제 연구에서는 스크린타임 캡처, 익명 설문, 학습기록, 수면기록을 2주 이상 수집하여 시간대별 사용 패턴까지 분석하는 것이 바람직하다. 또한 자기보고식 집중도는 주관성이 있으므로 교사 관찰, 자습 집중시간 측정, 과제 제출 지연 기록과 함께 검증하면 더 신뢰로운 결과를 얻을 수 있다.

9. 나의 진로·전공 관심과 연결

이 프로젝트는 컴퓨터과학, 심리학, HCI, 교육공학과 연결된다. 스마트폰을 단순히 나쁜 도구로 보는 것이 아니라, 앱 설계와 알림 구조, 사용 맥락이 학생의 주의력과 학습 몰입에 어떤 영향을 미치는지 분석한다. 나아가 학생의 삶을 더 잘 돕는 디지털 환경을 설계하는 문제로 확장할 수 있다.

성찰 문장: 나는 스마트폰 사용 시간을 줄이는 것만이 답이라고 생각했지만, 데이터를 보며 사용 시간보다 사용 방식과 자기조절 전략이 더 중요할 수 있다는 점을 이해하게 되었다.
