변수명 변수 설명 측정 단위 데이터 유형 SPSS 측도 실제 데이터 출처 비고
quarter 관찰 분기 분기 문자 명목 시범 생성/서울시 상권분석서비스 연도·분기 구조 2024Q1 형식
area_id 학교 주변 상권 구역 ID ID 문자 명목 시범 생성/상권영역 데이터 A01~A10
area_name 상권 구역명 명칭 문자 명목 시범 생성 학생 보고서용 별칭
business_type 주요 업종 유형 범주 문자 명목 소상공인시장진흥공단 상가업소정보/서울시 점포-상권 업종분류
location_type 학교 기준 위치 유형 범주 문자 명목 현장조사 정문, 후문, 역세권 등
distance_to_school_m 학교에서 상권까지 거리 m 숫자 척도 지도/API 또는 현장 측정 지도거리
latitude 위도 도 숫자 척도 상권영역/상가업소정보 지도 시각화
longitude 경도 도 숫자 척도 상권영역/상가업소정보 지도 시각화
store_count 점포 수 개 숫자 척도 서울시 점포-상권/상가업소정보 상권 규모
opening_count 개업 점포 수 개 숫자 척도 서울시 점포-상권 변화 정도
closing_count 폐업 점포 수 개 숫자 척도 서울시 점포-상권 위험 정도
survival_rate_pct 점포 생존율 % 숫자 척도 서울시 상권분석서비스/계산 변수 높을수록 안정
estimated_sales_million_krw 추정 매출 백만원 숫자 척도 서울시 추정매출-상권 단위 환산
floating_population 추정 유동인구 명 숫자 척도 서울시 추정유동인구/생활인구 기간별 합산
student_visit_share_pct 학생 방문 비중 % 숫자 척도 학생 설문/현장조사 청소년 영향
rent_index 임대료 부담 지수 지수 숫자 척도 서울시 상권분석서비스/현장 인터뷰 높을수록 부담
vacancy_risk_score 공실·폐업 위험점수 0~100 숫자 척도 계산 변수 높을수록 위험
youth_friendly_score 청소년 친화 점수 0~100 숫자 척도 학생 설문/계산 변수 높을수록 친화
resident_life_score 지역생활 기여 점수 0~100 숫자 척도 설문/현장조사 주민 생활성
commercial_change_index 상권 변화 지수 0~100 숫자 척도 계산 변수 개업, 폐업, 매출 변화
improvement_priority_score 개선 우선순위 점수 0~100 숫자 척도 계산 변수 높을수록 정책 제안 우선
commercial_type 상권 유형 분류 범주 문자 명목 군집/규칙 기반 분류 분석 후 유형화
business_type_eng 영문 업종 유형 범주 문자 명목 시각화용 변환 변수 그래프 한글 깨짐 방지

SPSS_Steps

SPSS_Steps: 8행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 입력 분기별 상권 데이터를 SPSS로 불러오기 quarter, area_id, business_type 등 파일 → 열기 → 데이터 문자 변수와 숫자 변수 측도 확인
2. 기술통계 상권의 평균 점포수·매출·유동인구 파악 store_count, estimated_sales_million_krw, floating_population 분석 → 기술통계량 → 기술통계 평균과 표준편차가 큰 지점 확인
3. 업종별 비교 업종에 따라 생존율·청소년 친화성이 다른지 확인 business_type, survival_rate_pct, youth_friendly_score 분석 → 평균비교 → 일원배치 분산분석 문구·학용품, 분식·카페, 학원가 차이 해석
4. 상관분석 유동인구·학생방문비중·매출·폐업위험의 관계 파악 floating_population, student_visit_share_pct, estimated_sales_million_krw, vacancy_risk_score 분석 → 상관분석 → 이변량 상관계수 상권 활성화와 폐업위험의 관계 해석
5. 회귀분석 상권 매출을 설명하는 요인 확인 estimated_sales_million_krw, store_count, floating_population, student_visit_share_pct, rent_index 분석 → 회귀분석 → 선형 학생 방문 비중과 유동인구가 매출에 미치는 영향 해석
6. 군집분석 상권을 유형별로 분류 survival_rate_pct, vacancy_risk_score, youth_friendly_score, resident_life_score 분석 → 분류분석 → K-평균 군집분석 청소년친화 안정상권, 공실위험상권 등 명명
7. 시각화 상권 변화와 개선 우선순위 제시 latitude, longitude, improvement_priority_score 그래프 → 차트 작성기 / Google My Maps 점 크기와 색으로 위험도 표현
