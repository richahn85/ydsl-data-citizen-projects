3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 입력 분기별 상권 데이터를 SPSS로 불러오기 quarter, area_id, business_type 등 파일 → 열기 → 데이터 문자 변수와 숫자 변수 측도 확인
2. 기술통계 상권의 평균 점포수·매출·유동인구 파악 store_count, estimated_sales_million_krw, floating_population 분석 → 기술통계량 → 기술통계 평균과 표준편차가 큰 지점 확인
3. 업종별 비교 업종에 따라 생존율·청소년 친화성이 다른지 확인 business_type, survival_rate_pct, youth_friendly_score 분석 → 평균비교 → 일원배치 분산분석 문구·학용품, 분식·카페, 학원가 차이 해석
4. 상관분석 유동인구·학생방문비중·매출·폐업위험의 관계 파악 floating_population, student_visit_share_pct, estimated_sales_million_krw, vacancy_risk_score 분석 → 상관분석 → 이변량 상관계수 상권 활성화와 폐업위험의 관계 해석
5. 회귀분석 상권 매출을 설명하는 요인 확인 estimated_sales_million_krw, store_count, floating_population, student_visit_share_pct, rent_index 분석 → 회귀분석 → 선형 학생 방문 비중과 유동인구가 매출에 미치는 영향 해석
6. 군집분석 상권을 유형별로 분류 survival_rate_pct, vacancy_risk_score, youth_friendly_score, resident_life_score 분석 → 분류분석 → K-평균 군집분석 청소년친화 안정상권, 공실위험상권 등 명명
7. 시각화 상권 변화와 개선 우선순위 제시 latitude, longitude, improvement_priority_score 그래프 → 차트 작성기 / Google My Maps 점 크기와 색으로 위험도 표현

4. 주요 해석 문장 예시

기술통계 결과, 학교와 가까운 상권일수록 학생 방문 비중이 높고 청소년 친화 점수가 높게 나타나는 경향이 있었다.

상관분석 결과, 유동인구와 추정 매출 간에는 정적 관계가 나타났으며, 폐업 점포 수와 공실 위험점수도 함께 증가하는 경향을 보였다.

군집분석 결과, 학교 주변 상권은 청소년친화 안정상권, 공실·폐업 위험상권, 변화·성장상권, 지역주민 생활상권으로 구분할 수 있었다.

5. SPSS Syntax 예시

아래 Syntax는 CSV 파일을 불러온 뒤 기술통계, 상관분석, 회귀분석, 군집분석을 수행하는 예시다. 파일 경로는 사용자의 저장 위치에 맞게 수정한다.

GET DATA /TYPE=TXT /FILE="26_dataset.csv" /DELCASE=LINE /DELIMITERS="," /ARRANGEMENT=DELIMITED /FIRSTCASE=2 /VARIABLES=quarter A8 area_id A4 area_name A40 business_type A20 location_type A20 distance_to_school_m F8.0 latitude F10.6 longitude F10.6 store_count F8.0 opening_count F8.0 closing_count F8.0 survival_rate_pct F8.1 estimated_sales_million_krw F8.1 floating_population F8.0 student_visit_share_pct F8.1 rent_index F8.1 vacancy_risk_score F8.1 youth_friendly_score F8.1 resident_life_score F8.1 commercial_change_index F8.1 improvement_priority_score F8.1 commercial_type A30. EXECUTE. DESCRIPTIVES VARIABLES=store_count estimated_sales_million_krw floating_population survival_rate_pct vacancy_risk_score youth_friendly_score improvement_priority_score. ONEWAY survival_rate_pct youth_friendly_score BY business_type. CORRELATIONS /VARIABLES=floating_population estimated_sales_million_krw student_visit_share_pct survival_rate_pct vacancy_risk_score. REGRESSION /DEPENDENT estimated_sales_million_krw /METHOD=ENTER store_count floating_population student_visit_share_pct rent_index. QUICK CLUSTER survival_rate_pct vacancy_risk_score youth_friendly_score resident_life_score /CRITERIA=CLUSTERS(4).
