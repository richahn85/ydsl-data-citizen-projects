26번 주제: 학교 주변 상권 변화 분석

학생과 지역주민의 생활을 반영하는 청소년 친화형 지역상권 제안

1. 프로젝트 개요

문제의식: 학교 주변 상권은 학생과 지역주민의 생활을 어떻게 반영하는가? 단순히 점포가 많고 매출이 높은 상권이 좋은 상권은 아니다. 학생이 안전하게 이용할 수 있고, 지역주민의 생활을 지탱하며, 소상공인이 지속가능하게 생존할 수 있는 상권인지 함께 보아야 한다.

이 프로젝트는 점포 유형, 유동인구, 추정 매출, 개업·폐업 데이터, 학생 설문과 현장조사를 결합해 학교 주변 상권의 변화를 분석하고, 청소년 친화형 지역상권 제안을 만드는 것을 목표로 한다.

질문 답변
내가 본 문제 학교 주변에 학생이 자주 가는 분식·카페는 늘지만 문구점·생활 점포는 줄어드는 듯하다.
왜 중요한가 상권 변화는 학생 생활 편의, 지역경제, 안전한 방과후 공간과 연결된다.
사용 데이터 점포 유형, 개업·폐업, 추정매출, 유동인구, 학교와 거리, 학생 설문, 현장 관찰
분석 방법 상권 변화 시각화, 업종별 생존율 분석, 상관·회귀분석, 군집분석, 개선 우선순위 점수화
제안 대상 학교, 동문회, 지역상인회, 성북구청 지역경제과, 청소년 관련 부서
전공 연결 경영학, 경제학, 도시사회학, 데이터사이언스, 지역혁신

2. 데이터 설계

분석 단위는 “분기 × 학교 주변 상권 구역”이다. 10개 상권 구역을 5개 분기로 관찰한 50행의 시범 데이터를 만들었다. 실제 분석에서는 서울시 상권분석서비스, 소상공인시장진흥공단 상가업소정보, 학생 설문, 현장조사를 결합한다.

4. 분석 방법

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 입력 분기별 상권 데이터를 SPSS로 불러오기 quarter, area_id, business_type 등 파일 → 열기 → 데이터 문자 변수와 숫자 변수 측도 확인
2. 기술통계 상권의 평균 점포수·매출·유동인구 파악 store_count, estimated_sales_million_krw, floating_population 분석 → 기술통계량 → 기술통계 평균과 표준편차가 큰 지점 확인
3. 업종별 비교 업종에 따라 생존율·청소년 친화성이 다른지 확인 business_type, survival_rate_pct, youth_friendly_score 분석 → 평균비교 → 일원배치 분산분석 문구·학용품, 분식·카페, 학원가 차이 해석
4. 상관분석 유동인구·학생방문비중·매출·폐업위험의 관계 파악 floating_population, student_visit_share_pct, estimated_sales_million_krw, vacancy_risk_score 분석 → 상관분석 → 이변량 상관계수 상권 활성화와 폐업위험의 관계 해석
5. 회귀분석 상권 매출을 설명하는 요인 확인 estimated_sales_million_krw, store_count, floating_population, student_visit_share_pct, rent_index 분석 → 회귀분석 → 선형 학생 방문 비중과 유동인구가 매출에 미치는 영향 해석
6. 군집분석 상권을 유형별로 분류 survival_rate_pct, vacancy_risk_score, youth_friendly_score, resident_life_score 분석 → 분류분석 → K-평균 군집분석 청소년친화 안정상권, 공실위험상권 등 명명
7. 시각화 상권 변화와 개선 우선순위 제시 latitude, longitude, improvement_priority_score 그래프 → 차트 작성기 / Google My Maps 점 크기와 색으로 위험도 표현

5. 시범 분석 결과

시범 데이터 분석 결과, 학생 방문 비중이 높은 분식·카페와 편의점·간편식 상권은 청소년 친화 점수가 높게 나타났지만, 임대료 부담과 폐업 점수가 높은 구역에서는 개선 우선순위가 높게 나타났다. 이는 학교 주변 상권을 단순 소비공간이 아니라 학생 생활권과 지역경제가 만나는 공간으로 보아야 함을 보여준다.

7. 한계점과 후속 연구

본 데이터는 공식 데이터 구조를 반영한 시범용 가상 데이터이므로 실제 정책 제안에는 공식 자료 다운로드와 현장조사가 필요하다. 특히 학교 주변 상권은 카드매출, 유동인구, 임대료, 학생 체감 인식이 서로 다르게 나타날 수 있다. 후속 연구에서는 실제 학생 설문과 상인 인터뷰를 추가해 데이터가 포착하지 못한 생활의 맥락을 보완해야 한다.

8. 진로·전공 관심과의 연결

이 프로젝트는 경영학의 상권 분석, 경제학의 지역경제 이해, 도시사회학의 생활권 분석, 데이터사이언스의 예측과 시각화를 결합한다. 학생은 “어떤 가게가 돈을 많이 버는가”를 넘어서 “어떤 상권이 사람을 살리는가”라는 질문으로 나아갈 수 있다.
