SPSS 분석 가이드: 고령층 디지털 소외 문제 분석

이 가이드는 14_dataset.xlsx 또는 14_dataset.csv를 SPSS에서 불러와 분석하는 절차를 안내한다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 → 14_dataset.xlsx 선택 → Data 시트 선택 → 첫 행에서 변수 이름 읽어오기를 체크한다. CSV를 사용할 경우 UTF-8 또는 한국어 인코딩을 확인한다.

2. 변수 보기 설정

변수 측도 값 레이블
respondent_id 명목 E001~E050
district 명목 성북구
dong 명목 돈암1동 등
age 척도 60~86
age_group 서열 60-64세~80세 이상
gender 명목 남/여
living_alone 명목 0=아니오, 1=예
education_level 서열 1=초졸 이하,2=중졸,3=고졸,4=대졸 이상
smartphone_own 명목 0=없음, 1=있음
kiosk_use_freq_month 척도 0~12
mobile_app_use_freq_month 척도 0~20
kiosk_difficulty 서열 1=매우 쉬움, 5=매우 어려움
app_difficulty 서열 1=매우 쉬움, 5=매우 어려움
online_reservation_difficulty 서열 1=매우 쉬움, 5=매우 어려움
digital_confidence 서열 1=매우 낮음, 5=매우 높음
digital_literacy_score 척도 0~100

3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV 또는 XLSX 파일을 SPSS로 불러온다. respondent_id~support_priority_rank 파일 → 열기 → 데이터 첫 행을 변수명으로 읽고 측도를 확인한다.
2. 빈도분석 응답자 특성과 위험등급 분포를 확인한다. age_group, gender, living_alone, exclusion_level 분석 → 기술통계량 → 빈도분석 고위험 집단이 어느 연령대·독거 여부에 집중되는지 확인한다.
3. 기술통계 핵심 변수의 평균과 분산을 파악한다. kiosk_difficulty, app_difficulty, digital_literacy_score, digital_exclusion_score 분석 → 기술통계량 → 기술통계 평균 소외점수와 변수별 차이를 확인한다.
4. 교육 경험별 차이 분석 디지털 교육 경험이 소외점수 차이와 관련되는지 본다. digital_education_experience, digital_exclusion_score 분석 → 평균비교 → 독립표본 t 검정 교육 경험이 있는 집단의 위험점수가 낮은지 해석한다.
5. 연령대별 차이 분석 연령대별 디지털 소외 위험 차이를 검증한다. age_group, digital_exclusion_score 분석 → 평균비교 → 일원배치 분산분석 80세 이상 집단의 위험점수가 유의하게 높은지 확인한다.
6. 상관분석 디지털 소외와 관련된 주요 요인을 탐색한다. kiosk_difficulty, app_difficulty, digital_confidence, nearest_digital_center_min, digital_exclusion_score 분석 → 상관분석 → 이변량 상관계수 어려움·두려움은 양의 상관, 자신감·도움 가능성은 음의 상관이 예상된다.
7. 회귀분석 소외점수를 예측하는 핵심 변수를 파악한다. digital_exclusion_score, age, kiosk_difficulty, app_difficulty, education_experience, help_available_score 분석 → 회귀분석 → 선형 가장 영향력이 큰 변수를 찾아 프로그램 설계 근거로 사용한다.
8. 군집분석 고령층을 지원유형별로 분류한다. kiosk_difficulty, app_difficulty, digital_confidence, access_gap_score, digital_literacy_score 분석 → 분류분석 → K-평균 군집분석 방문상담형·체험교육형·자립보완형 등으로 해석한다.
9. 시각화 위험집단과 교육 인프라 취약성을 그림으로 보여준다. digital_exclusion_score, nearest_digital_center_min, access_gap_score 그래프 → 차트 작성기 지원 우선지역·우선대상을 직관적으로 제시한다.

4. Syntax 예시

GET DATA /TYPE=XLSX /FILE='14_dataset.xlsx' /SHEET=name 'Data' /READNAMES=ON. EXECUTE. FREQUENCIES VARIABLES=age_group gender living_alone exclusion_level. DESCRIPTIVES VARIABLES=kiosk_difficulty app_difficulty digital_confidence digital_literacy_score digital_exclusion_score. T-TEST GROUPS=digital_education_experience(0 1) /VARIABLES=digital_exclusion_score. ONEWAY digital_exclusion_score BY age_group /STATISTICS DESCRIPTIVES. CORRELATIONS /VARIABLES=kiosk_difficulty app_difficulty digital_confidence nearest_digital_center_min help_available_score digital_exclusion_score. REGRESSION /DEPENDENT digital_exclusion_score /METHOD=ENTER age kiosk_difficulty app_difficulty online_reservation_difficulty digital_education_experience help_available_score nearest_digital_center_min. QUICK CLUSTER kiosk_difficulty app_difficulty digital_confidence access_gap_score digital_literacy_score /CRITERIA=CLUSTER(3) MXITER(10) /SAVE CLUSTER.

5. 해석 문장 예시

교육 경험이 없는 집단의 평균 디지털 소외 위험점수가 교육 경험이 있는 집단보다 높게 나타났다면, 이는 고령층 디지털 교육이 단순 취미 활동이 아니라 생활서비스 접근성을 높이는 지원정책이 될 수 있음을 의미한다.

키오스크 사용 어려움과 디지털 소외 위험점수가 높은 양의 상관을 보인다면, 음식 주문·병원 접수·은행 업무와 같은 생활밀착형 키오스크 체험교육을 우선 설계해야 한다.
