SPSS 분석 가이드 용문고 통학길 보행 안전 위험 예측 모델

이 문서는 01_dataset.xlsx 또는 01_dataset.csv를 SPSS에서 불러와 위험점수 분석, 지도형 산점도, 군집분석까지 수행하기 위한 단계별 안내서입니다.

주의: 본 데이터는 시범용 가상 데이터입니다. 실제 프로젝트에서는 공식 공공데이터와 학생 현장조사 데이터를 동일한 변수 구조로 입력해 분석합니다.

1. 파일 불러오기

1) SPSS 실행 → 파일 → 열기 → 데이터

2) 파일 유형을 Excel 또는 CSV로 선택

3) 01_dataset.xlsx의 Data 시트를 선택하고, ‘첫 행에서 변수 이름 읽어오기’를 체크

4) 변수 보기에서 측도 설정: route_name, road_type, risk_grade는 명목/서열, risk_score와 사고·보행량 변수는 척도

2. 변수 측도 점검

SPSS 측도 해당 변수
명목 obs_id, segment_id, route_name, district, road_type, time_slot, school_zone
서열 visibility_risk_1_5, slope_risk_1_5, student_flow_score_1_5, risk_grade, priority_group
척도 latitude, longitude, distance_m, accident_3yr, ped_accident_3yr, floating_pop_2h, sidewalk_width_m, risk_score

3. 분석 절차 요약

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 XLSX 또는 CSV를 SPSS로 가져오기 전체 변수 파일 → 열기 → 데이터 첫 행을 변수명으로 인식하고 변수보기에서 측도 확인
2. 빈도분석 위험등급·도로유형의 분포 확인 risk_grade, road_type, time_slot 분석 → 기술통계량 → 빈도분석 고위험 구간이 몇 개인지 파악
3. 기술통계 주요 위험요인의 평균·최댓값 확인 accident_3yr, ped_accident_3yr, floating_pop_2h, sidewalk_width_m, risk_score 분석 → 기술통계량 → 기술통계 어떤 변수가 전체 위험을 높이는지 기초 진단
4. 상관분석 위험점수와 요인 간 관계 확인 risk_score, accident_3yr, ped_accident_3yr, floating_pop_2h, illegal_parking_30min, sidewalk_width_m 분석 → 상관분석 → 이변량 상관계수 위험점수와 가장 관련이 큰 변수를 찾음
5. 회귀분석 위험점수를 설명하는 핵심 요인 검증 risk_score 종속변수 / 사고·보행량·정류장·보도폭·주정차 독립변수 분석 → 회귀분석 → 선형 위험점수에 유의한 영향을 주는 변수 확인
6. 군집분석 위험구간 유형 분류 accident_score, exposure_score, environment_score, school_score 분석 → 분류분석 → K-평균 군집분석 사고위험형, 노출위험형, 환경개선형 등으로 명명
7. 지도형 산점도 위험지도 시각화 longitude, latitude, risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 학교 주변의 위험점 분포를 시각적으로 설명
8. 개선 우선순위 도출 구간별 정책 제안 연결 priority_group, risk_grade, cluster_suggestion 데이터 → 케이스 정렬 1순위 구간부터 학교·구청 제안서 작성

4. 위험등급 빈도분석

메뉴: 분석 → 기술통계량 → 빈도분석

변수: risk_grade, priority_group

해석: 매우 높음 또는 높음 등급이 많으면 학교 주변 통학로 중 집중 관리 구간이 많다는 의미입니다.

5. 기술통계

메뉴: 분석 → 기술통계량 → 기술통계

변수: accident_3yr, ped_accident_3yr, floating_pop_2h, sidewalk_width_m, illegal_parking_30min, risk_score

해석: 평균, 최솟값, 최댓값을 확인해 통학로 전체의 위험 수준과 극단적으로 위험한 구간을 파악합니다.

6. 상관분석

메뉴: 분석 → 상관분석 → 이변량 상관계수

변수: risk_score, accident_3yr, ped_accident_3yr, floating_pop_2h, illegal_parking_30min, sidewalk_width_m

해석: risk_score와 양의 상관이 큰 변수는 위험을 높이는 요인입니다. sidewalk_width_m은 보도 폭이 넓을수록 위험이 낮아지는 음의 관계를 기대할 수 있습니다.

7. 회귀분석

메뉴: 분석 → 회귀분석 → 선형

종속변수: risk_score

독립변수: accident_3yr, ped_accident_3yr, floating_pop_2h, bus_stop_count_100m, crosswalk_count_100m, sidewalk_width_m, illegal_parking_30min

해석: 유의확률 p < .05인 변수는 위험점수를 설명하는 주요 변수입니다. 회귀계수가 양수이면 위험을 높이는 방향, 음수이면 위험을 낮추는 방향입니다.

8. K-평균 군집분석

메뉴: 분석 → 분류분석 → K-평균 군집분석

변수: accident_score, exposure_score, environment_score, school_score

군집 수: 3개 또는 4개로 설정

해석: 군집 중심값을 보고 사고위험형, 보행량 집중형, 환경개선형, 저위험 관찰형 등으로 이름을 붙입니다.

9. 지도형 산점도

메뉴: 그래프 → 차트 작성기 → 산점도

X축: longitude, Y축: latitude, 그룹/색상: risk_grade 또는 priority_group

해석: 실제 지도는 아니지만 좌표 기반으로 위험 구간의 공간 분포를 확인할 수 있습니다. 최종 제출용은 Google My Maps 또는 QGIS로 제작하면 좋습니다.

10. SPSS Syntax 예시

GET DATA /TYPE=XLSX /FILE='01_dataset.xlsx' /SHEET=name 'Data' /READNAMES=ON. EXECUTE.

FREQUENCIES VARIABLES=risk_grade priority_group road_type.

DESCRIPTIVES VARIABLES=accident_3yr ped_accident_3yr floating_pop_2h sidewalk_width_m illegal_parking_30min risk_score /STATISTICS=MEAN STDDEV MIN MAX.

CORRELATIONS /VARIABLES=risk_score accident_3yr ped_accident_3yr floating_pop_2h illegal_parking_30min sidewalk_width_m /PRINT=TWOTAIL NOSIG /MISSING=PAIRWISE.

REGRESSION /DEPENDENT risk_score /METHOD=ENTER accident_3yr ped_accident_3yr floating_pop_2h bus_stop_count_100m crosswalk_count_100m sidewalk_width_m illegal_parking_30min.

QUICK CLUSTER accident_score exposure_score environment_score school_score /CRITERIA=CLUSTER(3) MXITER(10) /SAVE CLUSTER.
