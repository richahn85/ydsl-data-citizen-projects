변수명 변수 설명 측정 단위 데이터 유형 SPSS 측도 실제 데이터 출처 비고
obs_id 관찰 단위 ID 문자 문자 명목 시범용 가상 데이터 각 행은 통학로 구간 관찰값
segment_id 도로 구간 ID 문자 문자 명목 현장 구간 설정 동일 구간을 여러 시간대에 관찰 가능
route_name 도로 구간명 문자 문자 명목 학생 현장조사 학생이 이해할 수 있는 구간명
district 자치구 문자 문자 명목 서울시 주소/행정구역 본 예시는 성북구
observed_date 관찰일 YYYY-MM-DD 날짜 명목 학생 현장조사 SPSS에서는 문자열 또는 날짜로 처리
time_slot 관찰 시간대 문자 문자 명목 학생 현장조사 등교/하교 시간대 구분
latitude 위도 WGS84 숫자 척도 서울시 공간데이터/현장 좌표 실제 데이터 교체 시 공식 좌표 사용
longitude 경도 WGS84 숫자 척도 서울시 공간데이터/현장 좌표 실제 데이터 교체 시 공식 좌표 사용
distance_m 학교 정문과의 거리 m 숫자 척도 좌표 거리계산 Google My Maps/QGIS 활용 가능
road_type 도로 유형 분류 문자 명목 현장조사/도로자료 주간선도로, 이면도로 등
crosswalk_count_100m 반경 100m 내 횡단보도 수 개 숫자 척도 서울시 대로변 횡단보도 위치정보, 자치구별 횡단보도 자료 공간 결합 후 산출
signal_count_100m 반경 100m 내 신호등 수 개 숫자 척도 서울시 자치구별 신호등 및 횡단보도 위치 및 현황 공간 결합 후 산출
bus_stop_count_100m 반경 100m 내 버스정류장 수 개 숫자 척도 서울시 버스정류소 위치정보 공간 결합 후 산출
accident_3yr 최근 3년 전체 교통사고 건수 건 숫자 척도 TAAS 교통사고 GIS 지도 조회 후 구간별 집계
ped_accident_3yr 최근 3년 보행자 사고 건수 건 숫자 척도 TAAS 교통사고 GIS, 보행자 교통사고 다발지역 API 보행자 사고 중심
severe_accident_3yr 최근 3년 중상 이상 사고 건수 건 숫자 척도 한국도로교통공단 보행자 사고 다발지역 API 위험 가중치 반영
floating_pop_2h 등하교 2시간 추정 보행량 명 숫자 척도 학생 현장조사/생활인구 보조자료 학생 10분 관찰값을 환산
sidewalk_width_m 보도 폭 m 숫자 척도 학생 현장조사 줄자/지도 거리측정
lane_count 차로 수 개 숫자 척도 현장조사/도로자료 차량 노출위험
illegal_parking_30min 30분 관찰 불법주정차 수 대 숫자 척도 학생 현장조사 사진 기록 병행
visibility_risk_1_5 시야 방해 위험 1~5점 숫자 서열 학생 현장조사 1 낮음, 5 매우 높음
slope_risk_1_5 경사 위험 1~5점 숫자 서열 현장조사/경사도 자료 1 낮음, 5 매우 높음
school_zone 학교 근접/보호구역 여부 0/1 숫자 명목 현장조사/보호구역 자료 0 아니오, 1 예
student_flow_score_1_5 학생 통행 집중도 1~5점 숫자 서열 학생 현장조사 보행량을 5단계로 환산
accident_score 사고위험 점수 0~100 숫자 척도 계산변수 사고건수·보행자사고·중상사고 가중합
exposure_score 노출위험 점수 0~100 숫자 척도 계산변수 보행량·버스정류장·횡단보도 반영
environment_score 도로환경위험 점수 0~100 숫자 척도 계산변수 보도폭, 차로수, 주정차, 시야, 경사 반영
school_score 통학집중위험 점수 0~100 숫자 척도 계산변수 학교 거리, 학생집중도, 보호구역 반영
risk_score 최종 위험점수 0~100 숫자 척도 계산변수 0.35사고+0.25노출+0.25환경+0.15통학
risk_grade 위험등급 문자 문자 서열 계산변수 매우 높음/높음/보통/낮음
priority_group 개선 우선순위 문자 문자 서열 계산변수 1순위/2순위/3순위/관찰
cluster_suggestion 군집 해석 제안 문자 문자 명목 계산변수 K-평균 군집분석 후 비교 가능

SPSS_Steps

SPSS_Steps: 9행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 XLSX 또는 CSV를 SPSS로 가져오기 전체 변수 파일 → 열기 → 데이터 첫 행을 변수명으로 인식하고 변수보기에서 측도 확인
2. 빈도분석 위험등급·도로유형의 분포 확인 risk_grade, road_type, time_slot 분석 → 기술통계량 → 빈도분석 고위험 구간이 몇 개인지 파악
3. 기술통계 주요 위험요인의 평균·최댓값 확인 accident_3yr, ped_accident_3yr, floating_pop_2h, sidewalk_width_m, risk_score 분석 → 기술통계량 → 기술통계 어떤 변수가 전체 위험을 높이는지 기초 진단
4. 상관분석 위험점수와 요인 간 관계 확인 risk_score, accident_3yr, ped_accident_3yr, floating_pop_2h, illegal_parking_30min, sidewalk_width_m 분석 → 상관분석 → 이변량 상관계수 위험점수와 가장 관련이 큰 변수를 찾음
5. 회귀분석 위험점수를 설명하는 핵심 요인 검증 risk_score 종속변수 / 사고·보행량·정류장·보도폭·주정차 독립변수 분석 → 회귀분석 → 선형 위험점수에 유의한 영향을 주는 변수 확인
6. 군집분석 위험구간 유형 분류 accident_score, exposure_score, environment_score, school_score 분석 → 분류분석 → K-평균 군집분석 사고위험형, 노출위험형, 환경개선형 등으로 명명
7. 지도형 산점도 위험지도 시각화 longitude, latitude, risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 학교 주변의 위험점 분포를 시각적으로 설명
8. 개선 우선순위 도출 구간별 정책 제안 연결 priority_group, risk_grade, cluster_suggestion 데이터 → 케이스 정렬 1순위 구간부터 학교·구청 제안서 작성
