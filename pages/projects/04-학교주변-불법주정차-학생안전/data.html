변수명 변수 설명 측정 단위 SPSS 측도 비고
segment_id 도로 구간 ID 문자 명목 분석 단위. 한 행은 하나의 관찰 구간
segment_name 도로 구간명 문자 명목 학생이 이해하기 쉬운 구간명
road_type 도로 유형 범주 명목 school_gate, alley, bus_stop, arterial, commercial 등
lat 위도 WGS84 척도 지도 시각화용 시범 좌표
lon 경도 WGS84 척도 지도 시각화용 시범 좌표
x_m 학교 기준 X좌표 m 척도 SPSS 산점도 지도용 상대좌표
y_m 학교 기준 Y좌표 m 척도 SPSS 산점도 지도용 상대좌표
distance_school_m 학교와의 거리 m 척도 학교에서 가까울수록 통학 노출도가 높다고 가정
observation_time 관찰 시간대 범주 명목 등교/하교/혼합 관찰 구분
observed_illegal_vehicles_30min 30분 관찰 불법주정차 차량 수 대 척도 학생 현장 관찰 또는 사진 기록으로 수집 가능
enforcement_cases_1yr 최근 1년 단속 건수 건 척도 서울시 불법주정차 단속 정보 대체 변수
fixed_cctv_100m 반경 100m 단속 CCTV 여부 0/1 명목 0=없음, 1=있음
crosswalk_count_100m 반경 100m 횡단보도 수 개 척도 횡단보도 위치자료로 산출
bus_stop_count_100m 반경 100m 버스정류장 수 개 척도 버스정류소 위치자료로 산출
accident_count_3yr 최근 3년 교통사고 건수 건 척도 TAAS 사고조회 또는 시범용 값
pedestrian_accident_3yr 최근 3년 보행자 사고 건수 건 척도 보행자 사고 위험 반영
student_flow_30min 30분 학생 보행량 명 척도 현장 관찰로 수집 가능
sidewalk_width_m 보도 폭 m 척도 좁을수록 위험 증가
road_width_m 도로 폭 m 척도 차량 공간 규모
lane_count 차로 수 개 척도 큰길/이면도로 구분 보조
blind_spot_score 시야 방해 점수 1~5 서열 불법주정차로 횡단 전 시야가 얼마나 가려지는가
traffic_volume_score 차량 통행량 점수 1~5 서열 현장 관찰 기반 추정
speed_limit_kmh 제한속도 km/h 척도 속도 환경 변수
parking_density_score 불법주정차 밀도 점수 0~100 척도 관찰 차량수와 단속건수를 표준화한 점수
accident_score 사고위험 점수 0~100 척도 사고건수와 보행자 사고건수 표준화 점수
visibility_score 시야방해 위험 점수 0~100 척도 시야 방해와 보도 폭 협소를 반영
exposure_score 학생노출 점수 0~100 척도 학생 보행량, 버스정류장, 횡단보도 수 반영
near_school_score 학교근접 점수 0~100 척도 학교에 가까울수록 높음
safety_risk_score 종합 안전위험 점수 0~100 척도 가중치 종합 점수
risk_grade 위험등급 범주 서열 낮음/보통/높음/매우 높음
risk_cluster 위험유형 범주 명목 군집분석 해석용 유형
proposed_action 개선 제안 문자 명목 구간별 우선 개선안
priority_rank 개선 우선순위 순위 서열 위험점수 기준 순위

SPSS_Steps

SPSS_Steps: 8행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 확인 결측치와 이상값 확인 모든 변수 분석 → 기술통계량 → 빈도분석/기술통계 불법주정차 차량 수, 사고건수, 학생 보행량의 범위 확인
2. 위험점수 기술통계 전체 위험 수준 파악 safety_risk_score, parking_density_score, visibility_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최댓값으로 위험의 분산 확인
3. 위험등급 빈도분석 매우 높음/높음 구간 비율 확인 risk_grade 분석 → 기술통계량 → 빈도분석 우선 개선 대상 규모 파악
4. 상관분석 불법주정차와 시야방해·사고의 관계 확인 observed_illegal_vehicles_30min, enforcement_cases_1yr, blind_spot_score, accident_count_3yr, safety_risk_score 분석 → 상관분석 → 이변량 상관계수 불법주정차가 많을수록 위험점수가 증가하는지 확인
5. 회귀분석 종합 위험점수의 설명 요인 확인 종속: safety_risk_score / 독립: observed_illegal_vehicles_30min, blind_spot_score, student_flow_30min, accident_count_3yr, sidewalk_width_m 분석 → 회귀분석 → 선형 어떤 요인이 위험점수를 가장 많이 설명하는지 비교
6. 군집분석 위험구간 유형 분류 parking_density_score, visibility_score, exposure_score, accident_score 분석 → 분류분석 → K-평균 군집분석 불법주정차·시야방해형, 학생노출형, 사고이력형 등으로 해석
7. 지도 시각화 위험구간 공간분포 표시 x_m, y_m, safety_risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 학교 기준 좌표 위에 위험점수를 점 크기로 표시
