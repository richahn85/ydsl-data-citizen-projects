이 문서는 04_dataset.xlsx 또는 04_dataset.csv를 SPSS로 불러와 분석하는 절차를 안내한다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 → 04_dataset.xlsx 선택 → 첫 행에서 변수 이름 읽어오기 체크 → Data 시트 선택

CSV를 사용할 경우 파일 유형을 CSV로 선택하고 UTF-8 또는 CP949 인코딩을 확인한다.

3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 확인 결측치와 이상값 확인 모든 변수 분석 → 기술통계량 → 빈도분석/기술통계 불법주정차 차량 수, 사고건수, 학생 보행량의 범위 확인
2. 위험점수 기술통계 전체 위험 수준 파악 safety_risk_score, parking_density_score, visibility_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최댓값으로 위험의 분산 확인
3. 위험등급 빈도분석 매우 높음/높음 구간 비율 확인 risk_grade 분석 → 기술통계량 → 빈도분석 우선 개선 대상 규모 파악
4. 상관분석 불법주정차와 시야방해·사고의 관계 확인 observed_illegal_vehicles_30min, enforcement_cases_1yr, blind_spot_score, accident_count_3yr, safety_risk_score 분석 → 상관분석 → 이변량 상관계수 불법주정차가 많을수록 위험점수가 증가하는지 확인
5. 회귀분석 종합 위험점수의 설명 요인 확인 종속: safety_risk_score / 독립: observed_illegal_vehicles_30min, blind_spot_score, student_flow_30min, accident_count_3yr, sidewalk_width_m 분석 → 회귀분석 → 선형 어떤 요인이 위험점수를 가장 많이 설명하는지 비교
6. 군집분석 위험구간 유형 분류 parking_density_score, visibility_score, exposure_score, accident_score 분석 → 분류분석 → K-평균 군집분석 불법주정차·시야방해형, 학생노출형, 사고이력형 등으로 해석
7. 지도 시각화 위험구간 공간분포 표시 x_m, y_m, safety_risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 학교 기준 좌표 위에 위험점수를 점 크기로 표시

4. 위험점수 계산 Syntax 예시

  • 불법주정차와 학생 안전 위험 분석 예시.

DESCRIPTIVES VARIABLES=observed_illegal_vehicles_30min enforcement_cases_1yr blind_spot_score student_flow_30min safety_risk_score. FREQUENCIES VARIABLES=risk_grade risk_cluster. CORRELATIONS /VARIABLES=observed_illegal_vehicles_30min enforcement_cases_1yr blind_spot_score accident_count_3yr safety_risk_score /PRINT=TWOTAIL NOSIG. REGRESSION /DEPENDENT safety_risk_score /METHOD=ENTER observed_illegal_vehicles_30min blind_spot_score student_flow_30min accident_count_3yr sidewalk_width_m. QUICK CLUSTER parking_density_score visibility_score exposure_score accident_score /CRITERIA=CLUSTER(4) MXITER(10) /SAVE CLUSTER. GRAPH /SCATTERPLOT(BIVAR)=x_m WITH y_m.

5. 결과 해석 문장 예시

상관분석에서 observed_illegal_vehicles_30min과 safety_risk_score의 상관이 양(+)의 방향으로 나타나면, 불법주정차 차량이 많은 구간일수록 종합 위험점수가 높아지는 경향이 있다고 해석할 수 있다.

회귀분석에서 blind_spot_score의 표준화계수가 크게 나타나면, 단순히 주차 차량의 수보다 학생의 시야가 얼마나 가려지는지가 안전위험을 설명하는 핵심 요인일 수 있다.

군집분석 결과 “불법주정차·시야방해형” 구간은 집중 단속과 시야확보 시설을 함께 제안하고, “학생노출·혼잡형” 구간은 통학지도 인력과 보행동선 분리를 우선 제안한다.

* Project 04: Illegal Parking and Student Safety around School.
* Open 04_dataset.csv or 04_dataset.xlsx first.

DESCRIPTIVES VARIABLES=observed_illegal_vehicles_30min enforcement_cases_1yr blind_spot_score student_flow_30min safety_risk_score
  /STATISTICS=MEAN STDDEV MIN MAX.

FREQUENCIES VARIABLES=risk_grade risk_cluster.

CORRELATIONS
  /VARIABLES=observed_illegal_vehicles_30min enforcement_cases_1yr blind_spot_score accident_count_3yr pedestrian_accident_3yr safety_risk_score
  /PRINT=TWOTAIL NOSIG.

REGRESSION
  /DEPENDENT safety_risk_score
  /METHOD=ENTER observed_illegal_vehicles_30min blind_spot_score student_flow_30min accident_count_3yr sidewalk_width_m.

QUICK CLUSTER parking_density_score visibility_score exposure_score accident_score
  /CRITERIA=CLUSTER(4) MXITER(10)
  /SAVE CLUSTER.

GRAPH /SCATTERPLOT(BIVAR)=x_m WITH y_m.
