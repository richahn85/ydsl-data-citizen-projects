SPSS 분석 가이드 19번 AI 학습 도구 사용 격차 분석

1. 데이터 불러오기

1) SPSS 실행 → 파일 → 열기 → 데이터 → 19_dataset.xlsx 또는 19_dataset.csv 선택

2) Excel 파일을 불러올 경우 ‘첫 행에서 변수 이름 읽어오기’를 체크한다.

3) 변수 보기에서 student_id는 명목, Likert 문항은 서열, 점수 변수는 척도로 지정한다.

2. 분석 절차 요약

단계 메뉴 경로 사용 변수 해석 포인트
빈도분석 분석 → 기술통계량 → 빈도분석 grade, paid_ai_access, user_type AI 접근성 및 사용 유형 분포 확인
기술통계 분석 → 기술통계량 → 기술통계 ai_literacy_score, gap_risk_score 평균과 표준편차로 전체 수준 파악
t검정 분석 → 평균비교 → 독립표본 t 검정 paid_ai_access, ai_literacy_score 유료 AI 접근 여부에 따른 차이 확인
분산분석 분석 → 평균비교 → 일원배치 분산분석 user_type, ai_literacy_score 사용 유형별 리터러시 차이 확인
상관분석 분석 → 상관분석 → 이변량 상관계수 home_digital_env, ai4_output_verification, ai_literacy_score 환경·검증습관과 리터러시 관계 확인
회귀분석 분석 → 회귀분석 → 선형 ai_literacy_score와 예측변수 AI 리터러시를 설명하는 핵심 요인 도출
군집분석 분석 → 분류분석 → K-평균 군집분석 weekly_ai_sessions, prompt, verification 등 학생 유형 분류

3. 보고서 해석 문장 예시

AI 리터러시 점수는 가정 디지털 환경, 교사의 안내 수준, 프롬프트 작성 역량, AI 답변 검증 습관과 관련이 있는 것으로 나타났다. 따라서 AI 학습 격차는 개인의 관심 부족만이 아니라 학교와 가정이 제공하는 학습 조건의 차이와도 연결된다.

4. 그래프 작성

그래프 → 차트 작성기에서 user_type별 ai_literacy_score 평균 막대그래프, home_digital_env와 ai_literacy_score 산점도, ai7_teacher_guidance와 ai10_learning_gain 산점도를 작성한다.

5. 정책 제안 연결

분석 결과는 반드시 교육 제안으로 연결한다. 기초형 학생에게는 AI 접근·사용법 교육, 과제해결형 학생에게는 출처검증·윤리교육, 고급활용형 학생에게는 심화 프로젝트와 동료 멘토링을 제안한다.
