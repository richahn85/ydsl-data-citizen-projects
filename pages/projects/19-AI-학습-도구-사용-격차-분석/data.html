변수명 변수 설명 측정 단위/코딩 데이터 유형 SPSS 측도 실제 데이터 출처 비고
student_id 익명 학생 ID S001~S050 문자 명목 학생 설문 개인식별정보 제거
grade 학년 1=1학년, 2=2학년, 3=3학년 정수 서열 학생 설문 집단 비교 변수
home_digital_env 가정 디지털 환경 수준 1=매우 낮음~5=매우 높음 정수 서열 학생 설문/디지털정보격차 실태조사 참고 접근성 지표
device_access 개인 학습기기 접근성 1=낮음~5=높음 정수 서열 학생 설문 스마트폰·태블릿·노트북 보유 및 사용 가능성
internet_stability 인터넷 안정성 1=불안정~5=안정 정수 서열 학생 설문 온라인 AI 도구 사용 조건
private_study_space 집중 가능한 학습공간 1=없음~5=충분 정수 서열 학생 설문 가정 내 학습환경
paid_ai_access 유료 AI 도구 접근 여부 0=없음, 1=있음 정수 명목 학생 설문 유료 기능 접근 격차
ai_experience_months AI 학습 도구 사용 경험 개월 정수 척도 학생 설문 사용 기간
weekly_ai_sessions 주간 AI 학습 도구 사용 횟수 회/주 정수 척도 학생 설문 활용 빈도
ai_tool_diversity 활용하는 AI 도구 수 개 정수 척도 학생 설문 도구 다양성
ai1_access AI 도구 접근 용이성 문항 1~5점 정수 서열 학생 설문 Likert
ai2_use_frequency AI 학습 도구 사용 빈도 문항 1~5점 정수 서열 학생 설문 Likert
ai3_prompt_skill 프롬프트 작성 역량 문항 1~5점 정수 서열 학생 설문 Likert
ai4_output_verification AI 답변 검증 습관 문항 1~5점 정수 서열 학생 설문 Likert
ai5_ethics_awareness AI 윤리 인식 문항 1~5점 정수 서열 학생 설문 Likert
ai6_confidence AI 활용 자신감 문항 1~5점 정수 서열 학생 설문 Likert
ai7_teacher_guidance 교사의 AI 활용 안내 수준 1~5점 정수 서열 학생 설문 Likert
ai8_home_support 가정 내 AI 활용 지원 수준 1~5점 정수 서열 학생 설문 Likert
ai9_peer_sharing 친구와 AI 활용법 공유 수준 1~5점 정수 서열 학생 설문 Likert
ai10_learning_gain AI 활용 후 학습효과 자기평가 1~5점 정수 서열 학생 설문 Likert
ai11_overreliance_risk AI 의존 위험 자기평가 1~5점 정수 서열 학생 설문 높을수록 의존 위험 높음
ai12_future_interest AI 기반 학습 탐구 관심 1~5점 정수 서열 학생 설문 Likert
ai_literacy_score AI 리터러시 종합점수 0~100점 실수 척도 파생변수 접근·활용·검증·윤리·자신감 결합
learning_outcome_score 학습 성과 자기평가 점수 0~100점 실수 척도 파생변수/학생 설문 AI 사용의 학습효과
gap_risk_score AI 학습 격차 위험점수 0~100점 실수 척도 파생변수 높을수록 지원 필요
user_type AI 학습 도구 사용 유형 1=비사용·기초형,2=과제해결형,3=비판적검증형,4=고급활용형 정수 명목 군집/분류 결과 사용 유형 분류
recommended_support 추천 지원 방식 1=기초교육,2=프롬프트워크숍,3=검증·윤리교육,4=동료멘토링,5=심화프로젝트 정수 명목 분석 결과 교육 프로그램 설계

Survey_Items: 13행 x 5열

문항번호 설문 문항 척도/코딩 연결 변수 해석 방향
AI1 나는 학교나 집에서 AI 학습 도구에 쉽게 접근할 수 있다. 1=전혀 그렇지 않다~5=매우 그렇다 ai1_access 높을수록 접근성 높음
AI2 나는 학습 과정에서 AI 도구를 자주 사용한다. 1~5점 ai2_use_frequency 높을수록 사용 빈도 높음
AI3 나는 AI에게 질문할 때 조건, 목적, 형식을 구체적으로 제시한다. 1~5점 ai3_prompt_skill 높을수록 프롬프트 역량 높음
AI4 나는 AI 답변을 그대로 믿지 않고 다른 자료와 비교해 확인한다. 1~5점 ai4_output_verification 높을수록 검증 습관 좋음
AI5 나는 AI 사용에서 표절, 개인정보, 편향 문제를 의식한다. 1~5점 ai5_ethics_awareness 높을수록 윤리 인식 높음
AI6 나는 AI 도구를 활용해 스스로 학습 문제를 해결할 자신이 있다. 1~5점 ai6_confidence 높을수록 자신감 높음
AI7 학교 수업이나 교사로부터 AI 학습 도구 활용 방법을 안내받았다. 1~5점 ai7_teacher_guidance 높을수록 학교 지원 높음
AI8 가정에서 AI·디지털 학습 도구 사용에 필요한 도움을 받을 수 있다. 1~5점 ai8_home_support 높을수록 가정 지원 높음
AI9 나는 친구들과 AI 활용법이나 좋은 질문 방법을 공유한다. 1~5점 ai9_peer_sharing 높을수록 동료학습 높음
AI10 AI를 활용하면 어려운 개념을 이해하는 데 도움이 된다. 1~5점 ai10_learning_gain 높을수록 학습효과 높음
AI11 나는 AI가 답을 대신해주면 스스로 생각하는 시간이 줄어든다. 1~5점 ai11_overreliance_risk 높을수록 의존 위험 높음
AI12 앞으로 AI를 활용해 더 깊은 탐구 프로젝트를 해보고 싶다. 1~5점 ai12_future_interest 높을수록 심화 관심 높음

SPSS_Steps

SPSS_Steps: 10행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX 데이터를 SPSS로 가져오기 전체 변수 파일 → 열기 → 데이터 변수 보기에서 명목/서열/척도 확인
2. 빈도분석 학년, 유료 AI 접근, 사용 유형 분포 파악 grade, paid_ai_access, user_type, recommended_support 분석 → 기술통계량 → 빈도분석 AI 접근성이 특정 집단에 치우치는지 확인
3. 기술통계 AI 리터러시·격차위험 평균과 분포 파악 ai_literacy_score, gap_risk_score, learning_outcome_score 분석 → 기술통계량 → 기술통계 전체 평균과 표준편차로 격차 수준 해석
4. 독립표본 t검정 유료 AI 접근 여부에 따른 AI 리터러시 차이 검정 paid_ai_access, ai_literacy_score 분석 → 평균비교 → 독립표본 t 검정 유료 도구 접근이 학습 격차와 관련 있는지 확인
5. 일원배치 분산분석 사용 유형별 AI 리터러시·학습효과 차이 검정 user_type, ai_literacy_score, learning_outcome_score 분석 → 평균비교 → 일원배치 분산분석 고급활용형과 비사용형의 차이 확인
6. 상관분석 가정 디지털 환경·교사지도·검증습관과 AI 리터러시 관계 확인 home_digital_env, ai7_teacher_guidance, ai4_output_verification, ai_literacy_score 분석 → 상관분석 → 이변량 상관계수 환경과 역량의 관련성 확인
