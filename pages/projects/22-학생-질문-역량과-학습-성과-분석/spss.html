22번 SPSS 분석 가이드: 학생 질문 역량과 학습 성과 분석

이 문서는 22_dataset.xlsx 또는 22_dataset.csv를 SPSS에서 불러와 질문 역량과 학습성과의 관계를 분석하는 절차를 안내한다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 → 22_dataset.xlsx 또는 22_dataset.csv 선택 → 첫 행을 변수명으로 읽어오기 체크.

2. 변수 보기 설정

student_id, class_id, question_profile은 명목, grade는 서열, 점수형 변수는 척도로 설정한다. high_performer는 0=보통/보완 필요, 1=고성과로 값 레이블을 지정한다.

3. 분석 단계별 메뉴

단계 메뉴 투입 변수 해석
빈도분석 분석 → 기술통계량 → 빈도분석 grade, question_profile, high_performer 질문 유형과 성과 집단 분포 확인
기술통계 분석 → 기술통계량 → 기술통계 question_total, avg_question_depth, question_competency_score, learning_outcome_score 평균과 표준편차 확인
신뢰도 분석 분석 → 척도 → 신뢰도 분석 survey_q1~survey_q11 설문 척도의 일관성 확인
상관분석 분석 → 상관분석 → 이변량 상관계수 question_competency_score, learning_outcome_score 등 질문 역량과 성과 관계 확인
회귀분석 분석 → 회귀분석 → 선형 종속: learning_outcome_score / 독립: question_total, avg_question_depth, diversity, reflection, effort 학습성과 예측요인 확인
ANOVA 분석 → 평균비교 → 일원배치 분산분석 question_profile, learning_outcome_score 질문 유형별 평균 성과 비교
K-평균 군집 분석 → 분류분석 → K-평균 군집분석 question_total, avg_question_depth, diversity, reflection 질문 패턴 유형화

4. 결과 해석 문장 예시

상관분석 결과 질문 역량 종합점수와 학습성과 종합점수 간 정적 관계가 나타났다. 이는 질문을 많이 하는 것 자체보다 질문의 깊이, 다양성, 성찰 기록이 함께 작동할 때 학습성과가 높아질 수 있음을 시사한다.

분산분석 결과 질문 유형별 학습성과 평균이 다르게 나타난다면, 침묵형 학생에게는 질문 시작 문장 카드가 필요하고, 사실확인형 학생에게는 분석·비판 질문으로 확장하는 훈련이 필요하다고 제안할 수 있다.

5. 보고서 작성 팁

결과를 “질문 많이 한 학생이 우수하다”로 단순화하지 않는다. 좋은 보고서는 질문의 양, 질, 다양성, 성찰을 구분하고 각 학생에게 필요한 지원 전략을 제안해야 한다.

* Project 22: Student Question Competency and Learning Outcomes.
GET DATA /TYPE=XLSX /FILE='22_dataset.xlsx' /SHEET=name 'Data' /READNAMES=ON.
EXECUTE.

VARIABLE LABELS
 question_total '수업 중 질문 총횟수'
 avg_question_depth '평균 질문 깊이 1~5점'
 question_diversity_index '질문 다양성 지수 0~100'
 self_reflection_score '자기성찰 기록 점수'
 question_competency_score '질문 역량 종합점수'
 learning_outcome_score '학습성과 종합점수'
 high_performer '고성과 여부'.
VALUE LABELS high_performer 0 '보통/보완 필요' 1 '고성과'.
EXECUTE.

FREQUENCIES VARIABLES=grade question_profile high_performer.
DESCRIPTIVES VARIABLES=question_total avg_question_depth question_diversity_index self_reflection_score question_competency_score learning_outcome_score.

RELIABILITY
 /VARIABLES=survey_q1_ask_frequency survey_q2_clarify survey_q3_analyze survey_q4_connect survey_q5_challenge survey_q6_meta survey_q7_listen survey_q8_feedback survey_q9_reflection survey_q10_confidence survey_q11_ai_prompt
 /SCALE('Question Competency Survey') ALL
 /MODEL=ALPHA.

CORRELATIONS
 /VARIABLES=question_total avg_question_depth question_diversity_index self_reflection_score question_competency_score learning_outcome_score
 /PRINT=TWOTAIL NOSIG.

REGRESSION
 /DEPENDENT learning_outcome_score
 /METHOD=ENTER question_total avg_question_depth question_diversity_index self_reflection_score study_effort_score teacher_feedback_score.

ONEWAY learning_outcome_score BY question_profile
 /STATISTICS DESCRIPTIVES
 /POSTHOC=TUKEY ALPHA(0.05).

QUICK CLUSTER question_total avg_question_depth question_diversity_index self_reflection_score
 /CRITERIA=CLUSTER(3) MXITER(10)
 /SAVE CLUSTER.
