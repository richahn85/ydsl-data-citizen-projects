SPSS 분석 가이드

  1. 진로 정보 격차 분석

1. 데이터 불러오기

SPSS에서 파일 → 열기 → 데이터를 선택한 뒤 21_dataset.xlsx의 Data 시트를 불러온다. 변수 보기에서 student_id, gender, desired_major_area 등은 명목, 1~5점 문항은 서열, 지수와 점수는 척도로 설정한다.

2. 변수 계산

진로정보 접근성 지수와 격차 위험점수는 이미 데이터셋에 포함되어 있으나, 실제 연구에서는 다음과 같이 여러 정보 접근성 변수를 표준화하거나 평균화하여 지수를 만들 수 있다.

예: information_access_index = 공식 사이트 활용도, 전공 이해도, 직업 이해도, 상담 경험, 동문 멘토링 경험, 가정 지원감 등을 종합한 0~100점 지수

3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 빈도분석 학년·희망 전공·정보 부족 유형의 분포 확인 grade, desired_major_area, primary_gap_type 분석 → 기술통계량 → 빈도분석 어느 전공·직업 분야에서 정보 부족이 집중되는지 확인
2. 기술통계 진로정보 접근성 지수와 격차 위험점수의 평균·표준편차 확인 information_access_index, career_gap_risk_score 분석 → 기술통계량 → 기술통계 전체 학생의 정보 접근성 수준 진단
3. 그룹 비교 동문 멘토링 경험 여부에 따라 진로 확신이 다른지 검정 alumni_mentoring_experience, career_confidence 분석 → 평균비교 → 독립표본 t 검정 멘토링 경험 학생의 진로 확신이 유의하게 높은지 확인
4. 상관분석 정보 접근성 요인과 진로 확신·불안의 관련성 확인 information_access_index, career_confidence, career_anxiety 분석 → 상관분석 → 이변량 상관계수 정보 접근성이 높을수록 진로 확신이 높고 불안이 낮은지 확인
5. 회귀분석 진로 확신을 설명하는 핵심 요인 파악 career_confidence ~ mentor_availability, teacher_guidance_score, parent_job_info_access, major_info_understanding 분석 → 회귀분석 → 선형 전공 이해, 동문 멘토 접근성, 교사 지도 중 핵심 요인 도출
6. 로지스틱 회귀 고위험 정보격차 학생을 예측 high_gap_yn ~ counseling_count, mentor_availability, parent_job_info_access, career_net_use 분석 → 회귀분석 → 이분형 로지스틱 지원이 필요한 학생 유형 예측
7. K-평균 군집분석 진로정보 이용 유형 분류 information_access_index, career_confidence, career_anxiety, mentor_availability 분석 → 분류분석 → K-평균 군집분석 정보취약형·정보탐색형·고확신형 등으로 구분
8. 시각화 격차 위험과 접근성의 관계 표현 information_access_index, career_gap_risk_score 그래프 → 차트 작성기 위험 학생군을 직관적으로 제시

4. 결과 해석 문장 예시

상관분석 결과 진로정보 접근성 지수는 진로 확신과 정적 관계를 보이고, 진로 불안과는 부적 관계를 보이는 것으로 해석할 수 있다. 이는 정보를 충분히 얻는 학생일수록 진로 선택에 대한 심리적 안정감이 높을 가능성을 보여준다.

5. 제출 산출물

학생은 분석 결과를 바탕으로 ① 진로정보 접근성 진단표, ② 정보취약 학생 유형 분류표, ③ 동문 기반 진로 멘토링 플랫폼 제안서, ④ 개인 성찰문을 제출한다.

* Project 21: Career information gap analysis.
GET DATA /TYPE=XLSX
  /FILE='21_dataset.xlsx'
  /SHEET=name 'Data'
  /READNAMES=on.
EXECUTE.

* Descriptive statistics.
FREQUENCIES VARIABLES=grade desired_major_area primary_gap_type recommended_support.
DESCRIPTIVES VARIABLES=information_access_index career_gap_risk_score career_confidence career_anxiety.

* Group comparison: alumni mentoring experience and career confidence.
T-TEST GROUPS=alumni_mentoring_experience(0 1)
  /VARIABLES=career_confidence information_access_index career_gap_risk_score
  /CRITERIA=CI(.95).

* Correlation analysis.
CORRELATIONS
  /VARIABLES=information_access_index career_gap_risk_score career_confidence career_anxiety mentor_availability teacher_guidance_score parent_job_info_access
  /PRINT=TWOTAIL NOSIG.

* Multiple regression: factors predicting career confidence.
REGRESSION
  /DEPENDENT career_confidence
  /METHOD=ENTER major_info_understanding job_info_understanding mentor_availability teacher_guidance_score parent_job_info_access career_net_use worknet_use.

* Logistic regression: high career information gap risk.
LOGISTIC REGRESSION VARIABLES high_gap_yn
  /METHOD=ENTER career_counseling_count mentor_availability parent_job_info_access career_net_use teacher_guidance_score
  /PRINT=CI(95).

* K-means clustering.
QUICK CLUSTER information_access_index career_confidence career_anxiety mentor_availability
  /CRITERIA=CLUSTER(4) MXITER(10)
  /SAVE CLUSTER.
