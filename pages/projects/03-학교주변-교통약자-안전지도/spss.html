SPSS 분석 가이드: 학교 주변 교통 약자 안전지도 만들기

이 문서는 03_dataset.xlsx 또는 03_dataset.csv를 SPSS로 불러와 교통약자 위험구간을 분석하는 절차를 안내한다. 분석 목적은 교통약자 위험점수가 높은 구간을 찾고, 위험을 만드는 핵심 요인을 파악하며, 개선 우선순위를 도출하는 것이다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 → 03_dataset.xlsx 또는 03_dataset.csv 선택. XLSX를 사용할 때는 Data 시트를 선택하고 첫 행을 변수 이름으로 읽어온다.

2. 변수 측도 설정

변수 유형 예시 SPSS 측도
문자형 ID·유형 segment_id, route_type, road_type, risk_type 명목
위험등급·순위 risk_grade, priority_rank 서열
점수·거리·개수 vulnerable_risk_score, distance_school_m, accident_count_3yr 척도

3. 분석 절차

단계 목적 SPSS 메뉴 주요 변수 해석
1. 데이터 불러오기 XLSX/CSV를 SPSS로 불러와 변수 측도 확인 파일 → 열기 → 데이터 전체 변수 첫 행을 변수명으로 읽고, 범주형은 명목/서열, 점수형은 척도로 설정
2. 기술통계 위험점수와 보행환경 변수의 평균·최대값 확인 분석 → 기술통계량 → 기술통계 vulnerable_risk_score, environment_score, exposure_score 어떤 변수의 편차가 큰지 확인
3. 빈도분석 위험등급과 위험유형 분포 확인 분석 → 기술통계량 → 빈도분석 risk_grade, risk_type, route_type 높음/매우 높음 구간 비율 확인
4. 상관분석 교통약자 위험점수와 주요 위험요인 관계 확인 분석 → 상관분석 → 이변량 상관계수 vulnerable_risk_score, exposure_score, environment_score, accident_score, infrastructure_lack_score 위험점수와 가장 관련이 큰 요인 파악
5. 회귀분석 위험점수를 설명하는 주요 요인 검증 분석 → 회귀분석 → 선형 종속: vulnerable_risk_score / 독립: exposure_score, environment_score, accident_score, infrastructure_lack_score, facility_score 표준화계수 Beta로 영향력 비교
6. 군집분석 구간을 위험유형별로 분류 분석 → 분류분석 → K-평균 군집분석 exposure_score, environment_score, accident_score, infrastructure_lack_score, facility_score 노출형·환경취약형·사고이력형 등 정책유형 분류
7. 지도 시각화 학교 주변 위험구간을 좌표 기반으로 표시 그래프 → 차트 작성기 → 산점도 x_m_from_school, y_m_from_school, vulnerable_risk_score, risk_grade 점 크기/색으로 위험 수준 표시
8. 우선순위표 작성 상위 10개 개선대상 구간 도출 데이터 → 케이스 정렬 priority_rank, segment_id, risk_grade, recommended_action 성북구청·경찰서 제안용 표 작성

5. 보고서 작성 문장 예시

“교통약자 종합 위험점수를 기준으로 상위 구간을 도출한 결과, 병원·복지시설 접근로와 대형 교차로 인접 구간의 위험도가 높게 나타났다. 이는 교통약자의 이동수요와 보행환경 위험요인이 결합될 때 잠재적 사고위험이 커질 수 있음을 의미한다.”

6. SPSS Syntax 예시

별도 파일 03_spss_syntax.sps를 SPSS Syntax 창에서 열어 실행할 수 있다. 단, 파일 경로는 각자의 컴퓨터 환경에 맞게 수정해야 한다.

Syntax

03_spss_syntax.sps

원본: 03_spss_syntax.sps

* Project 03: 학교 주변 교통 약자 안전지도 만들기.
* 파일 경로는 사용자 PC에 맞게 수정하세요.

GET DATA
  /TYPE=XLSX
  /FILE='03_dataset.xlsx'
  /SHEET=name 'Data'
  /READNAMES=ON.
EXECUTE.

VARIABLE LEVEL segment_id route_name route_type road_type risk_grade risk_type recommended_action recommended_agency (NOMINAL).
VARIABLE LEVEL priority_rank slope_risk_1_5 visibility_risk_1_5 lighting_risk_1_5 (ORDINAL).
VARIABLE LEVEL vulnerable_risk_score exposure_score facility_score environment_score infrastructure_lack_score accident_score (SCALE).

FREQUENCIES VARIABLES=risk_grade risk_type route_type road_type
  /ORDER=ANALYSIS.

DESCRIPTIVES VARIABLES=vulnerable_risk_score exposure_score facility_score environment_score infrastructure_lack_score accident_score sidewalk_width_m slope_grade_pct accident_count_3yr ped_accident_count_3yr
  /STATISTICS=MEAN STDDEV MIN MAX.

CORRELATIONS
  /VARIABLES=vulnerable_risk_score exposure_score facility_score environment_score infrastructure_lack_score accident_score
  /PRINT=TWOTAIL NOSIG
  /MISSING=PAIRWISE.

REGRESSION
  /DEPENDENT vulnerable_risk_score
  /METHOD=ENTER exposure_score facility_score environment_score infrastructure_lack_score accident_score.

QUICK CLUSTER exposure_score facility_score environment_score infrastructure_lack_score accident_score
  /CRITERIA=CLUSTER(3) MXITER(10)
  /METHOD=KMEANS
  /SAVE CLUSTER.

GRAPH
  /SCATTERPLOT(BIVAR)=x_m_from_school WITH y_m_from_school
  /TITLE='학교 주변 교통약자 안전지도: 학교 기준 상대좌표'.
