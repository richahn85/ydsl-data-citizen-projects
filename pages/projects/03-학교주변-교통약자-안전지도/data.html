child_facility_count_300m 300m 내 어린이시설/보호구역 수 개 척도 0~5 어린이 보행자 수요 대리변수
disabled_facility_count_300m 300m 내 장애인복지시설 수 개 척도 0~3 장애인 보행자 수요 대리변수

disabled_flow_2h 등하교 2시간 장애인 보행량 추정 명 척도 2~90 장애인 노출
vehicle_speed_limit_kmh 제한속도 km/h 척도 20/30/40/50 차량 위험요인
illegal_parking_30min 30분 관찰 불법주정차 수 대 척도 0~15 시야방해·보행공간 침해
visibility_risk_1_5 시야 방해 위험도 1~5점 서열 1 낮음~5 높음 불법주정차·굴곡도로 등 반영
lighting_risk_1_5 야간 조명 부족 위험도 1~5점 서열 1 낮음~5 높음 야간·우천 안전
accident_count_3yr 최근 3년 전체 사고 건수 건 척도 0~15 TAAS/사고자료 대체 변수
ped_accident_count_3yr 최근 3년 보행자 사고 건수 건 척도 0~10 보행 안전 핵심 변수
vulnerable_ped_accident_3yr 최근 3년 교통약자 보행 사고 건수 건 척도 0~8 노인·어린이·장애인 관련 사고
exposure_score 교통약자 노출 점수 0~100 척도 높을수록 위험 보행량·시설 이용수요 기반
facility_score 교통약자 시설밀집 점수 0~100 척도 높을수록 시설밀집 병원·복지시설·어린이시설 기반
environment_score 보행환경 위험 점수 0~100 척도 높을수록 위험 보도 폭·경사·시야·조명·속도
infrastructure_lack_score 보행안전시설 부족 점수 0~100 척도 높을수록 부족 횡단보도·신호등 부족
accident_score 사고 이력 점수 0~100 척도 높을수록 사고 많음 전체·보행자·교통약자 사고 반영
vulnerable_risk_score 교통약자 종합 위험점수 0~100 척도 높을수록 우선 개선 가중합 결과
priority_rank 개선 우선순위 순위 서열 1이 최우선 위험점수 기준 순위
risk_grade 위험등급 범주 서열 낮음/보통/높음/매우 높음 개선 우선 판단
risk_type 주요 위험유형 범주 명목 노출형 등 개선 전략 분류

SPSS_Steps

SPSS_Steps: 9행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 XLSX/CSV를 SPSS로 불러와 변수 측도 확인 전체 변수 파일 → 열기 → 데이터 첫 행을 변수명으로 읽고, 범주형은 명목/서열, 점수형은 척도로 설정
2. 기술통계 위험점수와 보행환경 변수의 평균·최대값 확인 vulnerable_risk_score, environment_score, exposure_score 분석 → 기술통계량 → 기술통계 어떤 변수의 편차가 큰지 확인
3. 빈도분석 위험등급과 위험유형 분포 확인 risk_grade, risk_type, route_type 분석 → 기술통계량 → 빈도분석 높음/매우 높음 구간 비율 확인
4. 상관분석 교통약자 위험점수와 주요 위험요인 관계 확인 vulnerable_risk_score, exposure_score, environment_score, accident_score, infrastructure_lack_score 분석 → 상관분석 → 이변량 상관계수 위험점수와 가장 관련이 큰 요인 파악
5. 회귀분석 위험점수를 설명하는 주요 요인 검증 종속: vulnerable_risk_score / 독립: exposure_score, environment_score, accident_score, infrastructure_lack_score, facility_score 분석 → 회귀분석 → 선형 표준화계수 Beta로 영향력 비교
6. 군집분석 구간을 위험유형별로 분류 exposure_score, environment_score, accident_score, infrastructure_lack_score, facility_score 분석 → 분류분석 → K-평균 군집분석 노출형·환경취약형·사고이력형 등 정책유형 분류
7. 지도 시각화 학교 주변 위험구간을 좌표 기반으로 표시 x_m_from_school, y_m_from_school, vulnerable_risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 점 크기/색으로 위험 수준 표시
8. 우선순위표 작성 상위 10개 개선대상 구간 도출 priority_rank, segment_id, risk_grade, recommended_action 데이터 → 케이스 정렬 성북구청·경찰서 제안용 표 작성
