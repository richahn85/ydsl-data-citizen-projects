1. 프로젝트 개요

본 프로젝트는 학생, 노인, 장애인, 어린이가 함께 이용하는 학교 주변 도로에서 누구에게 가장 위험한 구간이 형성되는지를 데이터로 분석하는 교통약자 안전지도 프로젝트이다. 핵심 질문은 “학교 주변 도로는 누구에게 가장 위험한가?”이다. 이 질문은 단순한 교통사고 분석이 아니라, 학생이 매일 걷는 길을 교통약자의 관점에서 다시 바라보는 공공문제 해결형 탐구이다.

사용 데이터는 보행자 사고, 어린이보호구역, 노인복지시설, 병원, 장애인복지시설, 경사도, 보도 폭, 버스정류장, 횡단보도, 신호등, 현장 관찰 기록으로 구성한다. 산출물은 교통 약자 안전지도, 개선 우선순위표, 성북구청·경찰서·학교 제출용 개선 제안서이다.

2. 작업 지식 프레임 적용

단계 적용 내용
내 주변에서 발견한 불편함 학교 주변 보도에서 학생과 고령 보행자, 어린이가 함께 이동하지만 보도 폭·경사·차량 시야 문제가 겹치는 구간이 존재한다.
데이터로 측정 가능한 문제 보행자 사고, 교통약자 시설, 병원·복지시설, 경사도, 보도 폭, 보행량, 불법주정차 수를 변수화한다.
분석·예측·시각화 가능 위험요인 가중치 모델로 교통약자 위험점수를 산출하고, 좌표 기반 산점도와 GIS 지도로 시각화한다.
학교·지역사회·공공기관 제안 가능 상위 위험구간을 성북구청, 성북경찰서, 학교, 학부모회에 개선 제안한다.
전공 관심과 연결 가능 공공정책, 도시계획, 사회복지, 교통공학, 데이터사이언스와 연결된다.

3. 문제 정의 카드

질문 답변
내가 본 문제는 무엇인가? 학생, 노인, 어린이, 장애인이 같은 길을 이용하지만 보행환경은 평균적인 성인 보행자를 기준으로 설계된 듯 보인다.
왜 중요한가? 교통약자는 사고가 나면 피해가 크고, 작은 보도 단차·경사·시야 방해도 이동권을 제한할 수 있다.
어떤 데이터를 사용할 것인가? 보행자 사고, 어린이보호구역, 노인·장애인복지시설, 병의원 위치, 보도 폭, 경사도, 현장 관찰 데이터.
어떤 분석을 할 것인가? 위험요인 가중치 모델, 상관분석, 회귀분석, K-평균 군집분석, 지도 시각화.
누구에게 제안할 것인가? 학교, 학부모회, 성북구청 교통행정과, 성북경찰서, 동문회.
전공 관심과 연결은? 사람 중심 도시계획과 데이터 기반 공공정책 설계에 대한 관심으로 발전시킬 수 있다.

4. 데이터 설계

분석 단위는 “학교 주변 보행 구간”이다. 이번 시범 데이터는 공식 공공데이터 구조를 반영하되 실제 현장자료를 대체한 50행의 시범용 가상 데이터다. 실제 연구에서는 서울 열린데이터광장, 공공데이터포털, TAAS 교통사고 GIS, 현장 관찰자료로 교체한다.

6. 위험점수 산식

교통약자 종합 위험점수는 다음과 같이 구성한다.

구성요소 설명

8. SPSS 분석 절차 요약

분석 단계 분석 목적 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 XLSX/CSV를 SPSS로 불러와 변수 측도 확인 파일 → 열기 → 데이터 첫 행을 변수명으로 읽고, 범주형은 명목/서열, 점수형은 척도로 설정
2. 기술통계 위험점수와 보행환경 변수의 평균·최대값 확인 분석 → 기술통계량 → 기술통계 어떤 변수의 편차가 큰지 확인
3. 빈도분석 위험등급과 위험유형 분포 확인 분석 → 기술통계량 → 빈도분석 높음/매우 높음 구간 비율 확인
4. 상관분석 교통약자 위험점수와 주요 위험요인 관계 확인 분석 → 상관분석 → 이변량 상관계수 위험점수와 가장 관련이 큰 요인 파악
5. 회귀분석 위험점수를 설명하는 주요 요인 검증 분석 → 회귀분석 → 선형 표준화계수 Beta로 영향력 비교
6. 군집분석 구간을 위험유형별로 분류 분석 → 분류분석 → K-평균 군집분석 노출형·환경취약형·사고이력형 등 정책유형 분류
7. 지도 시각화 학교 주변 위험구간을 좌표 기반으로 표시 그래프 → 차트 작성기 → 산점도 점 크기/색으로 위험 수준 표시
8. 우선순위표 작성 상위 10개 개선대상 구간 도출 데이터 → 케이스 정렬 성북구청·경찰서 제안용 표 작성

9. 개선 제안서

제안 대상 제안 내용 근거 데이터 기대 효과 실행 난이도
성북구청 교통행정과 상위 10개 위험구간 보도 폭·횡단시설 정비 위험점수, 보도 폭, 경사도, 사고이력 교통약자 이동권과 안전성 향상 중
성북경찰서 대형 교차로·정류장 주변 우회전 관리와 등하교 시간 단속 사고점수, 불법주정차, 시야위험 사고 예방 및 운전자 주의 강화 중
학교·학부모회 학생 교통안전 모니터링단 운영 현장 관찰 데이터, 위험지도 학생 참여형 안전문화 형성 낮음
동문회 교통약자 안전지도 개선 캠페인 후원 상위 위험구간 사진·지도 지역사회 기여와 학교 이미지 제고 낮음
