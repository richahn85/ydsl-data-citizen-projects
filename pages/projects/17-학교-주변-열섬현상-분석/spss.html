  1. 학교 주변 열섬현상 분석 SPSS 분석 가이드

이 가이드는 17_dataset.xlsx 또는 17_dataset.csv를 SPSS에서 열어 분석하는 절차를 설명한다. 데이터는 시범용 가상 데이터이며 실제 분석 시 공공데이터와 현장조사 값으로 교체한다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 → 17_dataset.xlsx 선택 → Data 시트 선택 → 첫 행에서 변수 이름 읽기 체크

2. 변수 측도 설정

record_id, location_id, zone_type, surface_type은 명목으로 설정한다. shade_score, sky_view_score, heat_risk_grade, improvement_priority는 서열로 설정한다. 기온, 지표면 온도, 녹지율, 학생 보행량, 위험점수는 척도로 설정한다.

3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
데이터 불러오기 Excel/CSV 데이터를 SPSS로 가져오기 전체 변수 파일 → 열기 → 데이터 첫 행을 변수명으로 읽고 측도 설정
기술통계 기온, 지표온도, 녹지율, 위험점수의 평균과 범위 확인 air_temp_c, surface_temp_c, green_ratio_pct, heat_risk_score 분석 → 기술통계량 → 기술통계 어느 변수가 평균적으로 높은지 확인
빈도분석 위험등급 분포와 공간 유형 분포 확인 heat_risk_grade, zone_type, surface_type 분석 → 기술통계량 → 빈도분석 매우 높음/높음 구간 비율 확인
상관분석 녹지율·불투수면·그늘과 위험점수 관계 확인 green_ratio_pct, impervious_ratio_pct, shade_score, surface_temp_c, heat_risk_score 분석 → 상관분석 → 이변량 상관계수 녹지율은 음의 상관, 불투수면은 양의 상관 예상
회귀분석 열섬 위험점수를 설명하는 주요 요인 파악 heat_risk_score, green_ratio_pct, impervious_ratio_pct, building_density_pct, shade_score, student_flow_30min 분석 → 회귀분석 → 선형 어떤 요인이 통계적으로 크게 작용하는지 파악
그룹 비교 공간 유형별 평균 위험점수 차이 확인 zone_type, heat_risk_score 분석 → 평균비교 → 일원배치 분산분석 상가밀집/버스정류장/정문 앞이 높은지 비교
군집분석 열섬 유형을 구분 surface_temp_c, green_ratio_pct, impervious_ratio_pct, shade_score, student_flow_30min 분석 → 분류분석 → K-평균 군집분석 고온·저녹지형, 보행노출형, 저위험 녹지형 등 명명
지도 시각화 학교 주변 열섬 위험지도 작성 x_coord_m, y_coord_m, heat_risk_score, heat_risk_grade 그래프 → 차트 작성기 → 산점도 점 크기/색상으로 위험구간 표시

4. SPSS Syntax 예시

GET DATA /TYPE=XLSX /FILE='17_dataset.xlsx' /SHEET=name 'Data' /READNAMES=on.

EXECUTE.

FREQUENCIES VARIABLES=heat_risk_grade zone_type surface_type.

DESCRIPTIVES VARIABLES=air_temp_c surface_temp_c green_ratio_pct impervious_ratio_pct shade_score student_flow_30min heat_risk_score.

CORRELATIONS /VARIABLES=surface_temp_c green_ratio_pct impervious_ratio_pct building_density_pct shade_score student_flow_30min heat_risk_score /PRINT=TWOTAIL NOSIG.

REGRESSION /DEPENDENT heat_risk_score /METHOD=ENTER surface_temp_c green_ratio_pct impervious_ratio_pct building_density_pct shade_score student_flow_30min.

ONEWAY heat_risk_score BY zone_type /STATISTICS DESCRIPTIVES.

QUICK CLUSTER surface_temp_c green_ratio_pct impervious_ratio_pct shade_score student_flow_30min /CRITERIA=CLUSTER(3) MXITER(10).

GRAPH /SCATTERPLOT(BIVAR)=x_coord_m WITH y_coord_m BY heat_risk_grade.

5. 해석 체크포인트

녹지율이 낮을수록 위험점수가 높아지는지 확인한다.

불투수면 비율과 건물 밀도가 지표면 온도 및 위험점수와 어떤 관계를 갖는지 본다.

정문 앞, 버스정류장, 횡단보도 대기 공간처럼 학생 보행량이 많은 지점의 개선 우선순위를 확인한다.

분석 결과는 반드시 그늘막, 가로수, 학교숲, 쿨페이브먼트, 우회통학로 안내 같은 실행 제안으로 연결한다.
