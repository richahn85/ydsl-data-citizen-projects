변수명 변수 설명 측정 단위 데이터 유형 SPSS 측도 실제 데이터 출처 비고
record_id 관측 ID 텍스트 문자 명목 시범용 생성 열섬 관측 1건
date 관측일 YYYY-MM-DD 날짜 명목 기상청/현장조사 SPSS 날짜형 변환 가능
time_slot 관측 시간대 시각 문자 명목 현장조사 등교/점심/하교 시간 비교
location_id 학교 주변 관측 지점 ID 텍스트 문자 명목 현장조사/GIS 지도 시각화 단위
location_name 관측 지점명 텍스트 문자 명목 현장조사 학생이 이해할 수 있는 장소명
zone_type 공간 유형 범주 문자 명목 현장조사 school_gate, bus_stop 등
surface_type 주요 지표면 유형 범주 문자 명목 토지피복/현장조사 asphalt, concrete, mixed 등
latitude 위도 도 숫자 척도 학교 위치/GIS Google My Maps, QGIS 활용
longitude 경도 도 숫자 척도 학교 위치/GIS Google My Maps, QGIS 활용
x_coord_m 학교 기준 상대 X좌표 m 숫자 척도 시범용 생성/GIS 변환 SPSS 산점도 X축
y_coord_m 학교 기준 상대 Y좌표 m 숫자 척도 시범용 생성/GIS 변환 SPSS 산점도 Y축
max_air_temp_c 해당일 최고기온 ℃ 숫자 척도 기상청 ASOS/단기예보 폭염일 판단
air_temp_c 관측 지점 기온 ℃ 숫자 척도 S-DoT/현장 온도계 지역별 미세 기온 차이
surface_temp_c 지표면 온도 ℃ 숫자 척도 열화상/위성/현장조사 열섬 핵심 종속변수
humidity_pct 습도 % 숫자 척도 S-DoT/기상청 체감온도 보정
wind_speed_ms 풍속 m/s 숫자 척도 기상청 바람이 적을수록 체감열 증가
green_ratio_pct 녹지율 % 숫자 척도 서울시 녹지현황/토지피복 낮을수록 위험 증가
tree_canopy_pct 수관 피복률 % 숫자 척도 현장조사/위성지도 그늘 형성 정도
impervious_ratio_pct 불투수면 비율 % 숫자 척도 토지피복지도 아스팔트/콘크리트 비율
building_density_pct 건물 밀도 % 숫자 척도 건축물 공간정보/현장추정 복사열 및 바람길 영향
shade_score 그늘 수준 1~5점 숫자 서열 현장조사 1 낮음, 5 높음
sky_view_score 하늘 노출도 1~5점 숫자 서열 현장조사 5는 직사광 노출 큼
traffic_volume_30min 30분 차량 통행량 대 숫자 척도 현장조사/교통량 복사열·배기가스 보조요인
student_flow_30min 30분 학생 보행량 명 숫자 척도 현장조사 학생 노출 정도
distance_to_green_m 가장 가까운 녹지까지 거리 m 숫자 척도 GIS 거리분석 멀수록 완충 효과 낮음
sidewalk_width_m 보도 폭 m 숫자 척도 현장조사/보도통계 좁은 보도는 대기 열노출 증가
heat_alert_dummy 폭염주의보 조건 여부 0/1 숫자 명목 기상청 1=폭염 조건일
perceived_temp_c 학생 체감온도 추정 ℃ 숫자 척도 계산변수 기온+습도+지표열+그늘 보정
heat_risk_score 열섬 위험점수 0~100 숫자 척도 계산변수 높을수록 개선 우선
heat_risk_grade 열섬 위험등급 낮음/보통/높음/매우 높음 문자 서열 계산변수 보고서용 등급
improvement_priority 개선 우선순위 순위 숫자 서열 계산변수 1위가 최우선
recommended_action 권장 개선안 텍스트 문자 명목 계산/해석 그늘막·녹지·쿨페이브먼트 등

SPSS_Steps

SPSS_Steps: 9행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
데이터 불러오기 Excel/CSV 데이터를 SPSS로 가져오기 전체 변수 파일 → 열기 → 데이터 첫 행을 변수명으로 읽고 측도 설정
기술통계 기온, 지표온도, 녹지율, 위험점수의 평균과 범위 확인 air_temp_c, surface_temp_c, green_ratio_pct, heat_risk_score 분석 → 기술통계량 → 기술통계 어느 변수가 평균적으로 높은지 확인
빈도분석 위험등급 분포와 공간 유형 분포 확인 heat_risk_grade, zone_type, surface_type 분석 → 기술통계량 → 빈도분석 매우 높음/높음 구간 비율 확인
상관분석 녹지율·불투수면·그늘과 위험점수 관계 확인 green_ratio_pct, impervious_ratio_pct, shade_score, surface_temp_c, heat_risk_score 분석 → 상관분석 → 이변량 상관계수 녹지율은 음의 상관, 불투수면은 양의 상관 예상
회귀분석 열섬 위험점수를 설명하는 주요 요인 파악 heat_risk_score, green_ratio_pct, impervious_ratio_pct, building_density_pct, shade_score, student_flow_30min 분석 → 회귀분석 → 선형 어떤 요인이 통계적으로 크게 작용하는지 파악
그룹 비교 공간 유형별 평균 위험점수 차이 확인 zone_type, heat_risk_score 분석 → 평균비교 → 일원배치 분산분석 상가밀집/버스정류장/정문 앞이 높은지 비교
군집분석 열섬 유형을 구분 surface_temp_c, green_ratio_pct, impervious_ratio_pct, shade_score, student_flow_30min 분석 → 분류분석 → K-평균 군집분석 고온·저녹지형, 보행노출형, 저위험 녹지형 등 명명
지도 시각화 학교 주변 열섬 위험지도 작성 x_coord_m, y_coord_m, heat_risk_score, heat_risk_grade 그래프 → 차트 작성기 → 산점도 점 크기/색상으로 위험구간 표시
