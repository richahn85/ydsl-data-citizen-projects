1. 데이터 불러오기

SPSS에서 28_dataset.xlsx 또는 28_dataset.csv를 연다. Excel 파일을 사용할 때는 첫 행을 변수명으로 읽어오도록 설정한다. 변수 보기에 들어가 Codebook의 SPSS 측도에 맞게 명목, 서열, 척도를 지정한다.

2. 주요 분석 절차

분석 SPSS 메뉴 핵심 해석
빈도분석 분석 → 기술통계량 → 빈도분석 priority_level과 recommendation의 분포를 확인한다.
기술통계 분석 → 기술통계량 → 기술통계 accessibility_score, cost_score, safety_score, priority_score의 평균과 표준편차를 확인한다.
상관분석 분석 → 상관분석 → 이변량 상관계수 공공좌석 수, 이용료, 거리와 우선순위 점수의 관계를 확인한다.
분산분석 분석 → 평균비교 → 일원배치 분산분석 우선순위 등급별 접근성 차이가 통계적으로 의미 있는지 확인한다.
회귀분석 분석 → 회귀분석 → 선형 priority_score를 종속변수로 두고 영향 요인을 분석한다.
군집분석 분석 → 분류분석 → K-평균 군집분석 생활권을 유형별로 묶고 정책명을 붙인다.
산점도 그래프 → 차트 작성기 → 산점도 x_m, y_m을 이용해 지도형 그래프를 만든다.

3. SPSS Syntax 예시

GET DATA /TYPE=XLSX /FILE='28_dataset.xlsx' /SHEET=name 'Data' /READNAMES=on.

EXECUTE.

FREQUENCIES VARIABLES=priority_level recommendation.

DESCRIPTIVES VARIABLES=accessibility_score cost_score safety_score demand_score priority_score

/STATISTICS=MEAN STDDEV MIN MAX.

CORRELATIONS

/VARIABLES=priority_score accessibility_score cost_score safety_score demand_score avg_hourly_fee_won public_seats_1km nearest_public_library_m

/PRINT=TWOTAIL NOSIG.

ONEWAY accessibility_score cost_score safety_score BY priority_level

/STATISTICS DESCRIPTIVES.

REGRESSION

/DEPENDENT priority_score

/METHOD=ENTER accessibility_score cost_score safety_score demand_score.

QUICK CLUSTER accessibility_score cost_score safety_score demand_score

/CRITERIA=CLUSTER(3) MXITER(10)

/SAVE CLUSTER.

4. 결과 해석 문장 예시

분석 결과, 접근성 점수가 낮고 비용 부담이 높은 생활권일수록 개선 우선순위 점수가 높게 나타났다. 이는 청소년 학습공간의 문제를 단순한 시설 수가 아니라 공공성, 비용, 운영시간, 안전성의 결합 문제로 보아야 함을 의미한다.
