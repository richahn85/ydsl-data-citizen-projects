crime_safety_score 체감 안전 점수 1~5 서열 학생 설문/현장 학습공간 이동 안전
walking_slope_score 보행 경사 편의 점수 1~5 서열 현장조사/경사도 1=가파름, 5=평탄
afterschool_student_count 방과후 학생 수요 추정 명 척도 학생 설문/관찰 공간 수요 지표
survey_need_score 학습공간 필요도 1~5 서열 학생 설문 1=낮음, 5=매우 높음
home_study_difficulty 가정학습 어려움 1~5 서열 학생 설문 가정 내 학습환경 격차
accessibility_score 접근성 종합점수 0~100 척도 계산 시설·시간·비용·안전 반영
cost_score 비용 접근성 점수 0~100 척도 계산 무료/저비용일수록 높음
safety_score 안전 접근성 점수 0~100 척도 계산 조명·보행·체감안전 반영
demand_score 학습공간 수요점수 0~100 척도 계산 학생 수요와 가정학습 어려움 반영
deprivation_score 접근성 취약점수 0~100 척도 계산 100-접근성 점수
priority_score 개선 우선순위 점수 0~100 척도 계산 높을수록 개선 우선
priority_level 개선 우선순위 등급 문자 서열 계산 Very High/High/Moderate/Low

SPSS_Steps

SPSS_Steps: 9행 x 5열

Step Purpose Variables SPSS Menu Interpretation
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러오기 전체 변수 파일 → 열기 → 데이터 첫 행을 변수명으로 읽고, Codebook대로 측도 지정
2. 빈도분석 우선순위 등급 분포 확인 priority_level, recommendation 분석 → 기술통계량 → 빈도분석 Very High/High 비율 확인
3. 기술통계 접근성·비용·안전·수요 평균 확인 accessibility_score, cost_score, safety_score, demand_score, priority_score 분석 → 기술통계량 → 기술통계 어떤 요인이 평균적으로 취약한지 판단
4. 상관분석 비용·거리·공공좌석과 우선순위 관계 확인 priority_score, accessibility_score, avg_hourly_fee_won, public_seats_1km, nearest_public_library_m 분석 → 상관분석 → 이변량 상관계수 높은 우선순위와 관련 높은 변수 확인
5. 그룹 비교 우선순위 등급별 접근성 차이 검토 priority_level, accessibility_score, cost_score, safety_score 분석 → 평균비교 → 일원배치 분산분석 등급별 평균 차이를 정책 근거로 활용
6. 회귀분석 개선 우선순위 예측 priority_score ← accessibility_score, cost_score, safety_score, demand_score 분석 → 회귀분석 → 선형 어떤 요인이 우선순위를 가장 많이 설명하는지 해석
7. 군집분석 생활권 유형 분류 accessibility_score, cost_score, safety_score, demand_score 분석 → 분류분석 → K-평균 군집분석 공공공간 부족형, 비용부담형, 안전취약형 등으로 명명
8. 지도 시각화 취약지역 표시 x_m, y_m, priority_score, priority_level 그래프 → 차트 작성기 → 산점도 점 크기/색상으로 개선 우선순위 표현
