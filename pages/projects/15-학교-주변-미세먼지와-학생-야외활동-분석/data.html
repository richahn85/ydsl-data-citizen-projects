변수명 변수 설명 측정 단위 SPSS 측도 실제 데이터 출처 비고
case_id 관찰 사례 ID 문자 명목 시범용 생성 AQ001 형식
date 관찰 날짜 YYYY-MM-DD 명목 기상청/서울시 대기질/학생 관찰 SPSS 날짜형으로 변환 가능
weekday 요일 문자 명목 관찰 기록 월~일
location_id 학교 주변 공간 ID 문자 명목 학교 현장조사 L01~L05
location_name 공간명 문자 명목 학교 현장조사 운동장, 정문, 후문 등
location_type 공간 유형 문자 명목 학교 현장조사 playground/gate/route/roadside/greenway
x_m 학교 기준 상대 X좌표 m 척도 현장조사/GIS 학교 중심 좌표화
y_m 학교 기준 상대 Y좌표 m 척도 현장조사/GIS 학교 중심 좌표화
activity_type 야외활동 유형 범주 명목 체육수업 일정/학생 관찰 체육수업, 등교보행 등
scheduled_outdoor_min 계획된 야외활동 시간 분 척도 체육수업 일정
actual_outdoor_min 실제 야외활동 시간 분 척도 관찰 기록 미세먼지에 따라 축소 가능
pm10 미세먼지 PM10 농도 ㎍/㎥ 척도 서울시 대기환경정보
pm25 초미세먼지 PM2.5 농도 ㎍/㎥ 척도 서울시 대기환경정보
ozone 오존 농도 ppb 또는 ㎍/㎥ 환산 척도 서울시 대기환경정보
temperature_c 기온 ℃ 척도 기상청 단기예보/ASOS
humidity_pct 습도 % 척도 기상청 단기예보/ASOS
wind_mps 풍속 m/s 척도 기상청 단기예보/ASOS
rain_yn 비 여부 0/1 명목 기상청 단기예보 0=없음, 1=있음
air_quality_level 대기질 단계 1~4 서열 환경부/학교 대응 기준 참고 1 좋음~4 매우나쁨
air_quality_label 대기질 단계명 문자 서열 환경부/학교 대응 기준 참고 좋음/보통/나쁨/매우나쁨
student_count 활동 참여 학생 수 명 척도 학교 관찰
sensitive_students 호흡기 민감군 학생 수 명 척도 익명 설문/보건실 자료 민감정보는 익명화
symptom_score 기침·눈 따가움 등 증상 점수 1~5 서열 학생 건강 설문 높을수록 증상 강함
perceived_air_quality_bad 체감 공기 나쁨 정도 1~5 서열 학생 설문 높을수록 나쁘게 느낌
concentration_after 야외활동 후 수업 집중도 1~5 서열 학생 설문 높을수록 집중도 높음
absence_count 해당 활동 결석/보건실 방문 수 명 척도 학교 관찰/보건실 기록 개인정보 제거
risk_score 야외활동 위험점수 0~100 척도 계산 변수 PM2.5, PM10, 오존, 증상, 민감군 비율 반영
decision_code 야외활동 의사결정 코드 1~4 서열 계산 변수 1 정상~4 실내대체
decision_label 야외활동 의사결정 문자 서열 계산 변수 정상진행/민감군보호/시간축소/실내대체
outdoor_allowed_yn 야외활동 허용 여부 0/1 명목 계산 변수 1=가능, 0=비권장
improvement_priority 개선 우선순위 점수 0~100 척도 계산 변수 위험점수, 축소시간, 민감군 비율 반영

SPSS_Steps

SPSS_Steps: 200행 x 26열 / 미리보기 10행 x 10열

분석 단계 SPSS 메뉴 경로 사용 변수 해석 포인트
1. 데이터 불러오기 파일 → 열기 → 데이터 15_dataset.xlsx 또는 CSV 첫 행 변수명 읽기, 결측치 확인
2. 빈도분석 분석 → 기술통계량 → 빈도분석 air_quality_label, decision_label, activity_type 대기질 단계와 활동 조정 결정 분포 확인
3. 기술통계 분석 → 기술통계량 → 기술통계 pm10, pm25, ozone, risk_score, actual_outdoor_min 평균·최소·최대·표준편차 확인
4. 평균 비교 분석 → 평균비교 → 일원배치 분산분석 air_quality_level → actual_outdoor_min, risk_score 미세먼지 단계별 야외활동 시간 차이 검증
5. 상관분석 분석 → 상관분석 → 이변량 상관계수 pm25, pm10, risk_score, actual_outdoor_min, symptom_score 오염도와 위험점수·활동시간 관계 확인
6. 회귀분석 분석 → 회귀분석 → 선형 종속: risk_score / 독립: pm25, pm10, ozone, symptom_score, sensitive_students 위험점수 예측 요인 확인
7. 로지스틱 회귀 분석 → 회귀분석 → 이분형 로지스틱 종속: outdoor_allowed_yn / 독립: pm25, pm10, ozone, rain_yn 야외활동 가능 여부 예측
8. 지도 시각화 그래프 → 차트 작성기 → 산점도 x_m, y_m, risk_score, location_name 학교 주변 공간별 위험지도를 작성

Summary

Summary: 200행 x 26열 / 미리보기 10행 x 10열

지표 값 해석
평균 야외활동 위험점수 44.2 전체 관찰 사례의 평균 위험도
미세먼지 나쁨 이상 사례 수 22 PM 기준 나쁨 이상으로 분류된 관찰 수
나쁨 이상 평균 실제 야외활동 시간 25.9 대기질 악화 시 실제 야외활동이 축소됨
좋음/보통 평균 실제 야외활동 시간 43.2 정상 대기질일 때 활동 시간이 상대적으로 길음
PM2.5-위험점수 상관 0.97 초미세먼지가 위험점수와 강하게 연결됨
PM2.5-실제 야외활동시간 상관 -0.781 초미세먼지가 높을수록 활동시간이 감소하는 관계
증상점수-위험점수 상관 0.807 학생 체감 건강 반응이 위험판단에 중요
