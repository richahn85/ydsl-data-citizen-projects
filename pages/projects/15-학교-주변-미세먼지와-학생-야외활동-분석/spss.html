15번 SPSS 분석 가이드: 학교 주변 미세먼지와 학생 야외활동 분석

이 문서는 15_dataset.xlsx 또는 15_dataset.csv를 SPSS에서 불러와 분석하는 절차를 안내한다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 → Excel 또는 CSV 선택

Excel을 사용할 경우 “데이터 첫 행에서 변수 이름 읽어오기”를 체크한다.

변수 보기에서 case_id, weekday, location_id, location_name, activity_type 등은 명목, air_quality_level은 서열, pm10, pm25, risk_score 등은 척도로 설정한다.

2. 분석 순서

순서 메뉴 변수 결과 해석
1. 데이터 불러오기 파일 → 열기 → 데이터 15_dataset.xlsx 또는 CSV 첫 행 변수명 읽기, 결측치 확인
2. 빈도분석 분석 → 기술통계량 → 빈도분석 air_quality_label, decision_label, activity_type 대기질 단계와 활동 조정 결정 분포 확인
3. 기술통계 분석 → 기술통계량 → 기술통계 pm10, pm25, ozone, risk_score, actual_outdoor_min 평균·최소·최대·표준편차 확인
4. 평균 비교 분석 → 평균비교 → 일원배치 분산분석 air_quality_level → actual_outdoor_min, risk_score 미세먼지 단계별 야외활동 시간 차이 검증
5. 상관분석 분석 → 상관분석 → 이변량 상관계수 pm25, pm10, risk_score, actual_outdoor_min, symptom_score 오염도와 위험점수·활동시간 관계 확인
6. 회귀분석 분석 → 회귀분석 → 선형 종속: risk_score / 독립: pm25, pm10, ozone, symptom_score, sensitive_students 위험점수 예측 요인 확인
7. 로지스틱 회귀 분석 → 회귀분석 → 이분형 로지스틱 종속: outdoor_allowed_yn / 독립: pm25, pm10, ozone, rain_yn 야외활동 가능 여부 예측
8. 지도 시각화 그래프 → 차트 작성기 → 산점도 x_m, y_m, risk_score, location_name 학교 주변 공간별 위험지도를 작성

3. 권장 결과표

표 1: 대기질 단계별 사례 수와 의사결정 분포

표 2: PM10, PM2.5, 오존, 위험점수의 기술통계

표 3: 대기질 단계별 실제 야외활동 시간 평균 비교

표 4: PM2.5, 증상점수, 위험점수 간 상관분석

표 5: 야외활동 위험점수 회귀분석 결과

4. 보고서 해석 문장 예시

“분석 결과 PM2.5가 높을수록 야외활동 위험점수가 높아졌고, 실제 야외활동 시간은 줄어드는 경향을 보였다. 따라서 학교는 미세먼지 농도만 확인하는 데서 그치지 말고, 학생 증상 설문과 민감군 학생 수를 함께 반영한 야외활동 의사결정 기준을 마련할 필요가 있다.”

* Project 15: 학교 주변 미세먼지와 학생 야외활동 분석.
GET DATA /TYPE=XLSX
  /FILE='15_dataset.xlsx'
  /SHEET=name 'Data'
  /READNAMES=ON.
EXECUTE.

FREQUENCIES VARIABLES=air_quality_label decision_label activity_type.
DESCRIPTIVES VARIABLES=pm10 pm25 ozone temperature_c humidity_pct actual_outdoor_min symptom_score risk_score improvement_priority.

ONEWAY actual_outdoor_min risk_score BY air_quality_level
  /STATISTICS DESCRIPTIVES
  /POSTHOC=TUKEY.

CORRELATIONS
  /VARIABLES=pm10 pm25 ozone risk_score actual_outdoor_min symptom_score concentration_after
  /PRINT=TWOTAIL NOSIG.

REGRESSION
  /DEPENDENT risk_score
  /METHOD=ENTER pm25 pm10 ozone symptom_score sensitive_students rain_yn.

LOGISTIC REGRESSION VARIABLES outdoor_allowed_yn
  /METHOD=ENTER pm25 pm10 ozone rain_yn sensitive_students
  /PRINT=CI(95).

QUICK CLUSTER pm25 pm10 ozone symptom_score risk_score actual_outdoor_min
  /CRITERIA=CLUSTER(3) MXITER(10)
  /SAVE CLUSTER.
