변수명 변수 설명 측정 단위 데이터 유형 SPSS 측도 실제 데이터 출처 비고
segment_id 도로·보행 구간 ID 텍스트 범주형 명목 현장 구간 분할 각 행의 고유 식별자
segment_name 구간명 텍스트 범주형 명목 현장 구간 분할 보고서에서 해석할 구간명
district 자치구명 텍스트 범주형 명목 서울시 행정구역 성북구 등
road_type 도로·공간 유형 텍스트 범주형 명목 현장 관찰 교차로, 역세권, 병원앞 등
latitude 위도 도 연속형 척도 공간 데이터/GIS 시범 데이터는 가상 좌표
longitude 경도 도 연속형 척도 공간 데이터/GIS 시범 데이터는 가상 좌표
x_m 학교/지역 중심 기준 동서 상대좌표 m 연속형 척도 GIS 계산 SPSS 산점도 X축
y_m 학교/지역 중심 기준 남북 상대좌표 m 연속형 척도 GIS 계산 SPSS 산점도 Y축
distance_from_school_m 학교 또는 분석 중심점과의 거리 m 연속형 척도 GIS 거리 계산 거리 기반 접근성 보조
elderly_pop_300m 반경 300m 내 65세 이상 인구 추정 명 연속형 척도 서울시 고령자현황 통계 실제 분석 시 행정동/격자 결합
elderly_density_km2 반경 300m 내 고령인구 밀도 명/km² 연속형 척도 서울시 고령자현황 통계 노출위험 변수
senior_welfare_count_300m 반경 300m 내 노인복지시설 수 개 연속형 척도 서울시 사회복지시설 목록 경로당/노인복지관 등
hospital_count_300m 반경 300m 내 병의원 수 개 연속형 척도 서울시 병의원 위치 정보 고령자 이동 유발지
subway_distance_m 가장 가까운 지하철역까지 거리 m 연속형 척도 서울교통공사 역 정보 역세권 보행노출 변수
bus_stop_count_200m 반경 200m 내 버스정류소 수 개 연속형 척도 서울시 버스정류소 위치정보 정류장 주변 혼잡
crosswalk_count_200m 반경 200m 내 횡단보도 수 개 연속형 척도 서울시 횡단보도 위치정보 보행 동선 변수
signal_count_200m 반경 200m 내 신호등/보행신호 수 개 연속형 척도 자치구별 신호등 및 횡단보도 현황 신호 부족 위험 판단
slope_degree 도로 경사도 도 연속형 척도 서울시 경사도/등고선 고령자 보행 부담
sidewalk_width_m 보도 폭 m 연속형 척도 서울시 보도통계/현장조사 좁을수록 위험
traffic_speed_kmh 관찰 또는 구간 평균 차량속도 km/h 연속형 척도 교통속도/현장관찰 높을수록 사고 심각도 증가
illegal_parking_obs_30m 30분 관찰 불법주정차 수 대 연속형 척도 현장조사/단속자료 시야 방해 변수
lighting_score 야간 조명 상태 1~5점 서열형 서열 현장조사 1=매우 어두움, 5=매우 밝음
ped_acc_3yr 최근 3년 전체 보행자 사고건수 건 연속형 척도 TAAS/교통사고 다발지역 API 시범 데이터
elderly_ped_acc_3yr 최근 3년 고령 보행자 사고건수 건 연속형 척도 보행노인 사고다발지역 API 핵심 종속/위험 변수
severe_ped_acc_3yr 최근 3년 중상 이상 보행자 사고건수 건 연속형 척도 TAAS/KOROAD API 사고 심각도
exposure_score 고령자 노출위험 점수 0~100점 연속형 척도 계산 변수 고령인구·시설·병원·정류장 결합
accident_score 사고위험 점수 0~100점 연속형 척도 계산 변수 보행자 사고·고령 사고·중상 사고 결합
environment_score 보행환경 위험 점수 0~100점 연속형 척도 계산 변수 경사·보도폭·속도·시야요인 결합
accessibility_score 접근혼잡 점수 0~100점 연속형 척도 계산 변수 지하철·정류장·횡단보도 접근성 결합
elderly_risk_score 고령층 보행안전 종합 위험점수 0~100점 연속형 척도 계산 변수 가중합 위험지수
risk_grade 위험등급 낮음/보통/높음/매우 높음 범주형 서열 계산 변수 우선순위 해석
risk_type 위험유형 텍스트 범주형 명목 계산 변수 사고집중형/환경취약형 등
priority_rank 개선 우선순위 순위 서열형 서열 계산 변수 1위가 최우선

SPSS_Steps

SPSS_Steps: 9행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 가져오기 전체 변수 파일 → 열기 → 데이터 문자형 변수와 숫자형 변수의 측도 확인
2. 기술통계 위험요인의 전체 분포 확인 elderly_pop_300m, ped_acc_3yr, sidewalk_width_m, slope_degree, elderly_risk_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최댓값으로 고위험 구간의 기준 파악
3. 빈도분석 위험등급과 위험유형 분포 확인 risk_grade, risk_type 분석 → 기술통계량 → 빈도분석 고위험 구간이 몇 개인지, 어떤 유형이 많은지 확인
4. 상관분석 위험점수와 요인 간 관계 확인 elderly_risk_score, elderly_pop_300m, elderly_ped_acc_3yr, slope_degree, sidewalk_width_m 분석 → 상관분석 → 이변량 상관계수 위험점수와 강하게 연결된 요인을 찾음
5. 회귀분석 위험점수 예측 요인 파악 종속: elderly_risk_score / 독립: elderly_pop_300m, elderly_ped_acc_3yr, slope_degree, sidewalk_width_m, hospital_count_300m 분석 → 회귀분석 → 선형 고령 보행위험에 가장 큰 영향을 주는 변수 해석
6. 군집분석 위험구간 유형화 exposure_score, accident_score, environment_score, accessibility_score 분석 → 분류분석 → K-평균 군집분석 사고집중형, 환경취약형, 노출집중형 등으로 명명
7. 지도 시각화 위험구간을 지도형으로 표현 x_m, y_m, elderly_risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 점의 위치와 크기를 통해 고위험 공간을 직관적으로 확인
8. 정책 제안 분석 결과를 구청 제안으로 연결 priority_rank, risk_type, elderly_risk_score 보고서 작성 상위 10개 구간에 맞춤형 개선안 제시
