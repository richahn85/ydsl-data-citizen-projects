SPSS 분석 가이드: 고령층 보행 안전 위험 예측 모델

용문고 YDSL 데이터 기반 문제해결 프로젝트

파일명: 11_dataset.xlsx 또는 11_dataset.csv

분석 목표: 고령 보행자가 많은 지역에서 사고위험, 보행환경, 접근혼잡 요인이 결합해 어느 구간이 가장 위험한지 파악한다.

1. 데이터 불러오기

SPSS에서 파일 → 열기 → 데이터 메뉴를 선택하고 11_dataset.xlsx의 Data 시트를 불러온다. CSV 파일을 사용할 경우 인코딩은 UTF-8 또는 UTF-8 BOM을 선택한다.

2. 변수 보기 설정

SPSS 측도 변수
명목 segment_id, segment_name, district, road_type, risk_type
서열 risk_grade, priority_rank, lighting_score
척도 latitude, longitude, x_m, y_m, elderly_pop_300m, 사고건수, 점수 변수 전체

3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 가져오기 전체 변수 파일 → 열기 → 데이터 문자형 변수와 숫자형 변수의 측도 확인
2. 기술통계 위험요인의 전체 분포 확인 elderly_pop_300m, ped_acc_3yr, sidewalk_width_m, slope_degree, elderly_risk_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최댓값으로 고위험 구간의 기준 파악
3. 빈도분석 위험등급과 위험유형 분포 확인 risk_grade, risk_type 분석 → 기술통계량 → 빈도분석 고위험 구간이 몇 개인지, 어떤 유형이 많은지 확인
4. 상관분석 위험점수와 요인 간 관계 확인 elderly_risk_score, elderly_pop_300m, elderly_ped_acc_3yr, slope_degree, sidewalk_width_m 분석 → 상관분석 → 이변량 상관계수 위험점수와 강하게 연결된 요인을 찾음
5. 회귀분석 위험점수 예측 요인 파악 종속: elderly_risk_score / 독립: elderly_pop_300m, elderly_ped_acc_3yr, slope_degree, sidewalk_width_m, hospital_count_300m 분석 → 회귀분석 → 선형 고령 보행위험에 가장 큰 영향을 주는 변수 해석
6. 군집분석 위험구간 유형화 exposure_score, accident_score, environment_score, accessibility_score 분석 → 분류분석 → K-평균 군집분석 사고집중형, 환경취약형, 노출집중형 등으로 명명
7. 지도 시각화 위험구간을 지도형으로 표현 x_m, y_m, elderly_risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 점의 위치와 크기를 통해 고위험 공간을 직관적으로 확인
8. 정책 제안 분석 결과를 구청 제안으로 연결 priority_rank, risk_type, elderly_risk_score 보고서 작성 상위 10개 구간에 맞춤형 개선안 제시

4. 해석 방법

기술통계에서는 위험점수의 평균과 최댓값을 확인한다. 위험점수 상위 10개 구간은 개선 우선순위가 된다.

상관분석에서는 elderly_risk_score와 elderly_ped_acc_3yr, slope_degree, sidewalk_width_m, elderly_pop_300m의 관계를 본다. 보도 폭은 낮을수록 위험해질 수 있으므로 음의 상관이 나타날 수 있다.

회귀분석에서는 어떤 변수가 위험점수를 가장 잘 설명하는지 본다. 표준화계수 Beta가 큰 변수를 정책 우선순위로 해석한다.

군집분석에서는 점수 유형에 따라 도로구간을 나눈다. 예를 들어 사고집중형은 단속과 신호개선, 환경취약형은 보도·경사 개선, 고령자노출형은 복지시설 주변 안전동선 설계가 필요하다.

5. 그래프 작성

그래프 → 차트 작성기 → 산점도를 선택한다. X축에는 x_m, Y축에는 y_m을 넣고, 점 크기나 라벨은 elderly_risk_score 또는 segment_id로 설정한다. SPSS만으로 실제 지도를 만들기 어렵다면 CSV를 Google My Maps나 QGIS로 불러온다.

6. 보고서 문장 예시

시범 분석 결과, 고령층 보행위험 점수는 고령자 인구, 고령 보행자 사고건수, 경사도, 보도 폭과 밀접하게 관련되어 있었다. 특히 위험점수 상위 구간은 병원·복지관·정류장 접근성이 높고 보행환경이 취약한 구간으로 나타났다. 따라서 고령층 보행안전 정책은 사고가 이미 발생한 곳만이 아니라, 고령자의 생활동선과 보행환경이 겹치는 구간을 중심으로 설계되어야 한다.

* 11번 프로젝트: 고령층 보행 안전 위험 예측 모델.
* 데이터 파일을 SPSS에서 먼저 불러온 뒤 아래 명령문을 실행한다.

DATASET NAME ElderlyPedestrianRisk.

FREQUENCIES VARIABLES=risk_grade risk_type road_type
  /ORDER=ANALYSIS.

DESCRIPTIVES VARIABLES=elderly_pop_300m elderly_ped_acc_3yr ped_acc_3yr severe_ped_acc_3yr
  slope_degree sidewalk_width_m exposure_score accident_score environment_score
  accessibility_score elderly_risk_score
  /STATISTICS=MEAN STDDEV MIN MAX.

CORRELATIONS
  /VARIABLES=elderly_risk_score elderly_pop_300m elderly_ped_acc_3yr ped_acc_3yr
  severe_ped_acc_3yr slope_degree sidewalk_width_m hospital_count_300m
  senior_welfare_count_300m bus_stop_count_200m signal_count_200m
  /PRINT=TWOTAIL NOSIG
  /MISSING=PAIRWISE.

REGRESSION
  /DEPENDENT elderly_risk_score
  /METHOD=ENTER elderly_pop_300m elderly_ped_acc_3yr slope_degree sidewalk_width_m
  hospital_count_300m senior_welfare_count_300m bus_stop_count_200m signal_count_200m.

QUICK CLUSTER exposure_score accident_score environment_score accessibility_score
  /CRITERIA=CLUSTER(4) MXITER(10)
  /PRINT INITIAL FINAL CLUSTER.

GRAPH
  /SCATTERPLOT(BIVAR)=x_m WITH y_m
  /TITLE='고령층 보행 안전 위험지도: 상대좌표 산점도'.

GRAPH
  /SCATTERPLOT(BIVAR)=exposure_score WITH environment_score
  /TITLE='고령자 노출위험과 보행환경 위험의 관계'.
