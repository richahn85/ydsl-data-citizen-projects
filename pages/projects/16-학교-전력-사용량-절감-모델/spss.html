3. 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 Excel/CSV를 SPSS로 불러와 변수 측도 확인 전체 변수 파일 → 열기 → 데이터 문자/숫자 변수 구분, 결측값 확인
2. 기술통계 전력사용량·외기온도·낭비점수의 기본 수준 파악 electricity_kwh, outdoor_temp_c, waste_score 분석 → 기술통계량 → 기술통계 평균, 표준편차, 최대값으로 피크 확인
3. 공간유형별 비교 일반교실·특별실·공용공간의 사용량 차이 확인 room_type, electricity_kwh 분석 → 평균비교 → 일원배치 분산분석 어떤 공간유형이 가장 많이 쓰는가
4. 상관분석 온도, 재실인원, 조명, 기기 수와 전력 사용량 관계 확인 electricity_kwh, outdoor_temp_c, occupants, lights_on_rate, device_count 분석 → 상관분석 → 이변량 상관계수 전력사용량과 관련이 큰 요인 확인
5. 회귀분석 전력사용량을 예측하는 핵심 요인 확인 electricity_kwh ← outdoor_temp_c, class_blocks, occupants, ac_on, lights_on_rate, device_count 분석 → 회귀분석 → 선형 온도와 냉방 가동의 영향 해석
6. 피크 여부 예측 피크 사용 여부를 예측하는 조건 확인 peak_flag ← outdoor_temp_c, ac_on, lights_on_rate, device_count, window_open_while_ac 분석 → 회귀분석 → 이분형 로지스틱 피크가 발생할 가능성이 높은 조건 도출
7. 우선순위 시각화 절감 우선순위가 높은 공간을 찾음 priority_score, room_id, weekday 그래프 → 차트 작성기 행정실·학생회 제안 대상 선정
8. 에너지 절감 대시보드 총 사용량, 예상 절감량, 비용·탄소 절감효과 제시 electricity_kwh, expected_saving_kwh, electricity_cost_krw, co2_kg Excel/대시보드 또는 SPSS 출력표 절감안을 숫자로 설명

4. SPSS Syntax 예시

GET DATA /TYPE=XLSX /FILE='16_dataset.xlsx' /SHEET=name 'Data' /READNAMES=on. EXECUTE.

DESCRIPTIVES VARIABLES=electricity_kwh outdoor_temp_c lights_on_rate device_count standby_device_count waste_score expected_saving_kwh priority_score /STATISTICS=MEAN STDDEV MIN MAX.

ONEWAY electricity_kwh BY room_type /STATISTICS DESCRIPTIVES /POSTHOC=TUKEY ALPHA(0.05).

CORRELATIONS /VARIABLES=electricity_kwh outdoor_temp_c occupants class_blocks lights_on_rate device_count standby_device_count waste_score /PRINT=TWOTAIL NOSIG.

REGRESSION /DEPENDENT electricity_kwh /METHOD=ENTER outdoor_temp_c class_blocks occupants ac_on lights_on_rate device_count window_open_while_ac.

LOGISTIC REGRESSION VARIABLES peak_flag /METHOD=ENTER outdoor_temp_c ac_on lights_on_rate device_count standby_device_count window_open_while_ac.

QUICK CLUSTER electricity_kwh waste_score expected_saving_rate_pct priority_score /CRITERIA=CLUSTER(3) MXITER(10) /SAVE CLUSTER.

5. 해석 문장 예시

전력사용량에 대한 회귀분석 결과, 외기온도와 냉방 가동 여부가 유의하면 ‘더운 날씨 자체보다 냉방 운영 방식이 전력 피크를 만드는 핵심 요인’으로 해석할 수 있다.

공간유형별 차이가 유의하면 일반교실 전체를 동일하게 관리하기보다 특별실, 공용공간, 교무공간 등 공간 특성별 절감전략이 필요하다고 제안한다.
