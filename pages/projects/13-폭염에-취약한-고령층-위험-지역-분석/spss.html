SPSS 분석 가이드

  1. 폭염에 취약한 고령층 위험 지역 분석

이 가이드는 13_dataset.xlsx 또는 13_dataset.csv를 SPSS에서 불러와 폭염 취약지수, 위험등급, 상관·회귀·군집분석, 지도형 산점도를 작성하는 절차를 안내한다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 → 13_dataset.xlsx 선택

Excel 파일 읽기 창에서 '데이터 첫 행에서 변수 이름 읽어오기' 체크

변수 보기에서 area_id, area_name, dong, area_type, risk_grade, recommended_action은 명목 또는 서열로 지정하고, 나머지 숫자 변수는 척도로 지정

2. 기술통계

메뉴: 분석 → 기술통계량 → 기술통계

변수: avg_max_temp_c, heatwave_days, green_ratio_pct, elderly_pop65, living_alone_elderly, nearest_shelter_m, heat_vulnerability_index

해석: 폭염 노출, 녹지, 고령인구, 쉼터 접근성의 기본 분포를 파악한다.

3. 위험등급 빈도분석

메뉴: 분석 → 기술통계량 → 빈도분석

변수: risk_grade, area_type, recommended_action

해석: 매우 높음·높음 등급이 몇 개 생활권인지 확인하고, 필요한 개선유형을 파악한다.

4. 상관분석

메뉴: 분석 → 상관분석 → 이변량 상관계수

변수: heat_vulnerability_index, heat_exposure_score, demographic_vulnerability_score, shelter_deficit_score, cooling_environment_risk, green_ratio_pct, nearest_shelter_m

해석: 취약지수와 관련성이 큰 요인을 확인한다. 녹지율은 음의 상관, 쉼터 거리와 취약성 점수는 양의 상관을 보일 수 있다.

5. 회귀분석

메뉴: 분석 → 회귀분석 → 선형

종속변수: heat_vulnerability_index

독립변수: heat_exposure_score, demographic_vulnerability_score, shelter_deficit_score, cooling_environment_risk

해석: 어떤 요인이 취약지수를 가장 크게 설명하는지 확인한다.

6. K-평균 군집분석

메뉴: 분석 → 분류분석 → K-평균 군집분석

변수: heat_exposure_score, demographic_vulnerability_score, shelter_deficit_score, cooling_environment_risk

군집 수: 4개 권장. 폭염노출형, 고령취약형, 쉼터부족형, 저위험형으로 해석할 수 있다.

7. 지도형 산점도

메뉴: 그래프 → 차트 작성기 → 산점도

X축: longitude, Y축: latitude

점 크기: heat_vulnerability_index, 색상 또는 패널: risk_grade

해석: 고위험 생활권이 공간적으로 어디에 모여 있는지 확인한다.

8. 보고서 해석 문장 예시

폭염 취약지수를 종속변수로 하는 회귀분석 결과, 쉼터 부족 점수와 고령층 취약성 점수가 취약지수를 크게 설명하는 것으로 나타났다. 이는 폭염 대응 정책이 단순한 기온 관리가 아니라, 고령층 생활권의 쉼터 접근성과 돌봄 인프라 개선으로 이어져야 함을 보여준다.

* Project 13: Heat-vulnerable elderly risk area analysis.
* Import CSV.
GET DATA
  /TYPE=TXT
  /FILE='13_dataset.csv'
  /ENCODING='UTF8'
  /DELCASE=LINE
  /DELIMITERS=","
  /ARRANGEMENT=DELIMITED
  /FIRSTCASE=2
  /VARIABLES=
    area_id A8
    area_name A40
    dong A20
    area_type A20
    latitude F10.6
    longitude F10.6
    avg_max_temp_c F4.1
    heatwave_days F3.0
    tropical_nights F3.0
    green_ratio_pct F5.1
    shade_score F1.0
    pavement_heat_score F1.0
    elderly_pop65 F5.0
    elderly75_ratio_pct F5.1
    living_alone_elderly F5.0
    field_observed_elderly_2h F4.0
    heat_shelters_500m F2.0
    nearest_shelter_m F5.0
    shelter_capacity F5.0
    bus_stops_500m F2.0
    clinics_500m F2.0
    slope_score F1.0
    old_building_ratio_pct F5.1
    heat_exposure_score F5.1
    demographic_vulnerability_score F5.1
    shelter_deficit_score F5.1
    cooling_environment_risk F5.1
    heat_vulnerability_index F5.1
    risk_grade A10
    priority_rank F2.0
    recommended_action A60.
EXECUTE.

VARIABLE LABELS
 avg_max_temp_c 'Average summer maximum temperature'
 heatwave_days 'Number of heatwave days'
 tropical_nights 'Number of tropical nights'
 green_ratio_pct 'Green ratio percent'
 elderly_pop65 'Population aged 65 and older'
 living_alone_elderly 'Elderly living alone'
 nearest_shelter_m 'Distance to nearest heat shelter in meters'
 heat_vulnerability_index 'Heat vulnerability index'.

DESCRIPTIVES VARIABLES=avg_max_temp_c heatwave_days tropical_nights green_ratio_pct elderly_pop65 living_alone_elderly nearest_shelter_m heat_vulnerability_index
  /STATISTICS=MEAN STDDEV MIN MAX.

FREQUENCIES VARIABLES=risk_grade area_type recommended_action.

CORRELATIONS
  /VARIABLES=heat_vulnerability_index heat_exposure_score demographic_vulnerability_score shelter_deficit_score cooling_environment_risk green_ratio_pct nearest_shelter_m
  /PRINT=TWOTAIL NOSIG
  /MISSING=PAIRWISE.

REGRESSION
  /DEPENDENT heat_vulnerability_index
  /METHOD=ENTER heat_exposure_score demographic_vulnerability_score shelter_deficit_score cooling_environment_risk.

QUICK CLUSTER heat_exposure_score demographic_vulnerability_score shelter_deficit_score cooling_environment_risk
  /CRITERIA=CLUSTER(4) MXITER(10)
  /SAVE CLUSTER.

GRAPH
  /SCATTERPLOT(BIVAR)=longitude WITH latitude
  /TITLE='Heat Vulnerability Map (Demo)'.
