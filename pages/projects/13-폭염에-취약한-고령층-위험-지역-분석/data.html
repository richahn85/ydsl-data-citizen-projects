elderly_pop65 65세 이상 인구 명 척도 서울시 고령자현황 통계와 연결

living_alone_elderly 독거노인 수 명 척도 독거노인 통계와 연결 가능
field_observed_elderly_2h 2시간 현장 관찰 고령 보행자 수 명 척도 학생 현장조사
heat_shelters_500m 500m 내 무더위쉼터 수 개 척도 무더위쉼터 위치정보
nearest_shelter_m 가장 가까운 무더위쉼터 거리 m 척도 GIS 거리 분석
shelter_capacity 인근 쉼터 수용가능 인원 명 척도 무더위쉼터 인원 정보
bus_stops_500m 500m 내 버스정류장 수 개 척도 대중교통 접근성 보조
clinics_500m 500m 내 병의원 수 개 척도 의료 접근성 보조
slope_score 경사 위험도 1~5점 서열 고령자 이동 부담
old_building_ratio_pct 노후건물 비율 % 척도 냉방 취약 주거환경 대리 변수
heat_exposure_score 폭염 노출 점수 0~100 척도 기온·폭염일수·열대야·복사열 기반 지수
demographic_vulnerability_score 고령층 취약성 점수 0~100 척도 고령인구·독거노인·75세 이상 비율 기반
shelter_deficit_score 쉼터 부족 점수 0~100 척도 쉼터 거리·개수·수용능력 기반
cooling_environment_risk 냉각환경 부족 점수 0~100 척도 낮은 녹지·그늘 부족·노후건물·복사열
heat_vulnerability_index 폭염 취약지수 0~100 척도 4개 하위지수 가중합
risk_grade 위험등급 문자 서열 낮음/보통/높음/매우 높음
priority_rank 개선 우선순위 순위 서열 점수가 높을수록 우선순위 높음
recommended_action 추천 개선안 문자 명목 쉼터 설치, 그늘 보강, 방문돌봄 등

Field_Checklist: 11행 x 4열 / 미리보기 10행 x 4열

문항ID 현장 관찰 항목 척도/기록 방식 연결 변수
F1 생활권의 도로·보행로에 그늘이 충분한가? 1=거의 없음 ~ 5=매우 충분 shade_score
F2 아스팔트·콘크리트 포장이 많아 열기가 강하게 느껴지는가? 1=낮음 ~ 5=매우 높음 pavement_heat_score
F3 고령자가 쉬어갈 벤치나 쉼터 안내 표지가 있는가? 0=없음, 1=있음 보완 관찰 변수
F4 무더위쉼터까지 이동하는 길에 급경사나 계단이 많은가? 1=낮음 ~ 5=높음 slope_score
F5 정오~오후 3시 사이 고령 보행자가 얼마나 관찰되는가? 명수 기록 field_observed_elderly_2h
F6 무더위쉼터 입구가 고령자가 찾기 쉽게 표시되어 있는가? 1=매우 부족 ~ 5=매우 좋음 현장 개선안
F7 폭염 안내문, 냉방시설 안내문이 보이는가? 0=없음, 1=있음 현장 개선안
F8 생활권 내 병의원·약국 접근이 쉬운가? 1=어려움 ~ 5=쉬움 clinics_500m 보조

SPSS_Steps

SPSS_Steps: 8행 x 4열

분석 단계 SPSS 메뉴 경로 사용 변수 해석 포인트
1. 데이터 불러오기 파일 → 열기 → 데이터 13_dataset.xlsx 또는 13_dataset.csv 변수 보기에서 명목/서열/척도 확인
2. 기술통계 분석 → 기술통계량 → 기술통계 avg_max_temp_c, heatwave_days, green_ratio_pct, elderly_pop65, nearest_shelter_m, heat_vulnerability_index 폭염·녹지·고령인구·쉼터 접근성의 기본 분포 확인
3. 위험등급 빈도 분석 → 기술통계량 → 빈도분석 risk_grade, area_type, recommended_action 위험등급별 생활권 수와 우선 개선 유형 확인
4. 상관분석 분석 → 상관분석 → 이변량 상관계수 heat_vulnerability_index와 주요 지표 취약지수와 가장 관련 큰 요인 확인
5. 회귀분석 분석 → 회귀분석 → 선형 종속: heat_vulnerability_index / 독립: heat_exposure_score, demographic_vulnerability_score, shelter_deficit_score, cooling_environment_risk 어떤 하위지수가 위험을 가장 잘 설명하는지 확인
6. 군집분석 분석 → 분류분석 → K-평균 군집분석 4개 하위지수 폭염노출형, 쉼터부족형, 고령취약형, 저위험형 등으로 유형화
7. 지도 시각화 그래프 → 차트 작성기 → 산점도 X=longitude, Y=latitude, 점 크기=heat_vulnerability_index, 색상=risk_grade 고위험 생활권의 공간적 집중 확인
