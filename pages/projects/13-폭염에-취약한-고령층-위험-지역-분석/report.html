  1. 폭염에 취약한 고령층 위험 지역 분석

무더위쉼터 추가 설치 우선지역과 폭염 취약지수 모델

본 보고서는 여름철 폭염이 어느 지역의 고령층에게 더 큰 위험으로 작용하는지를 데이터로 분석하고, 무더위쉼터 추가 설치와 그늘·녹지 보강, 방문돌봄 강화가 필요한 지역을 제안하기 위한 시범 프로젝트이다. 데이터는 공식 공공데이터 구조를 반영한 시범용 가상 데이터이며, 실제 수업에서는 서울시 무더위쉼터, 고령자현황, 기상청 폭염일수, S-DoT 환경정보, 녹지현황 데이터로 교체할 수 있다.

1. 프로젝트 개요

문제의식 여름철 폭염은 어느 지역의 노인에게 가장 위험한가?
왜 중요한가 폭염은 개인의 체력 문제가 아니라 기온, 그늘, 녹지, 쉼터 접근성, 고령층 분포가 결합된 생활환경 문제이다.
학생의 실제 경험과 연결 동네에서 더운 날 그늘이 부족한 골목과 쉼터가 멀리 떨어진 고령층 생활권을 관찰할 수 있다.
제안 가능성 구청, 주민센터, 보건소, 복지관에 무더위쉼터 추가 설치와 폭염 취약지역 관리를 제안할 수 있다.
연결 전공 환경공학, 공공보건, 도시정책, 데이터사이언스, 사회복지
산출물 폭염 취약지역 지도, 무더위쉼터 추가 설치 제안서, 고령층 폭염 대응 가이드

2. 문제 정의 카드

내가 본 문제는 무엇인가? 여름철 오후 시간대에 고령 보행자가 많은 생활권 중 일부는 녹지와 그늘이 부족하고, 무더위쉼터까지의 거리가 멀어 폭염 위험이 높다.

왜 이 문제가 나에게 중요한가? 폭염은 단순히 날씨가 더운 문제가 아니라, 누가 더 취약한 환경에 놓여 있는지를 보여주는 사회적 문제이다.

어떤 데이터를 사용할 것인가? 기온, 폭염일수, 열대야일수, 녹지율, 무더위쉼터 위치, 고령인구, 독거노인, 병의원·버스정류장 접근성, 현장 관찰 자료를 사용한다.

어떤 분석을 할 것인가? 폭염 노출, 고령층 취약성, 쉼터 부족, 냉각환경 부족을 각각 점수화하고 가중합으로 폭염 취약지수를 만든다.

누구에게 제안할 것인가? 성북구청, 주민센터, 보건소, 노인복지관, 학교 사회공헌 동아리, 동문회에 제안한다.

전공 관심과 어떻게 연결되는가? 환경공학과 데이터사이언스를 활용해 사람 중심의 기후 적응 정책을 설계하는 경험으로 연결된다.

5. 분석 방법

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로
1. 데이터 불러오기 변수 보기에서 명목/서열/척도 확인 13_dataset.xlsx 또는 13_dataset.csv 파일 → 열기 → 데이터
2. 기술통계 폭염·녹지·고령인구·쉼터 접근성의 기본 분포 확인 avg_max_temp_c, heatwave_days, green_ratio_pct, elderly_pop65, nearest_shelter_m, heat_vulnerability_index 분석 → 기술통계량 → 기술통계
3. 위험등급 빈도 위험등급별 생활권 수와 우선 개선 유형 확인 risk_grade, area_type, recommended_action 분석 → 기술통계량 → 빈도분석
4. 상관분석 취약지수와 가장 관련 큰 요인 확인 heat_vulnerability_index와 주요 지표 분석 → 상관분석 → 이변량 상관계수
5. 회귀분석 어떤 하위지수가 위험을 가장 잘 설명하는지 확인 종속: heat_vulnerability_index / 독립: heat_exposure_score, demographic_vulnerability_score, shelter_deficit_score, cooling_environment_risk 분석 → 회귀분석 → 선형
6. 군집분석 폭염노출형, 쉼터부족형, 고령취약형, 저위험형 등으로 유형화 4개 하위지수 분석 → 분류분석 → K-평균 군집분석
7. 지도 시각화 고위험 생활권의 공간적 집중 확인 X=longitude, Y=latitude, 점 크기=heat_vulnerability_index, 색상=risk_grade 그래프 → 차트 작성기 → 산점도

시범 데이터에서 가장 높은 폭염 취약지수를 보인 생활권은 보문동 생활권 30이며, 주요 원인은 높은 평균 최고기온, 낮은 녹지율, 긴 쉼터 접근거리, 고령·독거노인 집중으로 해석된다. 상위 10개 지역은 단순히 더운 곳이 아니라, 더위에 취약한 사람이 많이 살고 냉각·돌봄 인프라가 부족한 곳으로 나타났다.

그림 1. 폭염 취약지역 지도 예시: 점의 크기는 취약지수, 색상은 위험등급을 의미한다.

그림 2. 폭염 취약지수 상위 10개 생활권. 구청이 우선 점검할 생활권을 선정하는 데 활용할 수 있다.

그림 3. 녹지율과 폭염 취약지수의 관계. 녹지율이 낮은 생활권일수록 취약지수가 높아지는 경향을 확인하도록 설계했다.

8. 최종 보고서 본문

연구 배경

기후위기로 폭염의 빈도와 강도가 커지면, 같은 도시 안에서도 누구에게 위험이 집중되는지가 중요해진다. 고령층은 체온 조절 능력과 이동 능력이 상대적으로 약하고, 독거노인은 위험 상황에서 도움을 요청하기 어렵다. 따라서 폭염 대응은 평균 기온만이 아니라 고령층의 생활권과 접근 가능한 인프라를 함께 보아야 한다.

문제의식

본 연구는 '여름철 폭염은 어느 지역의 노인에게 가장 위험한가'라는 질문에서 출발하였다. 폭염 위험은 높은 기온, 녹지 부족, 쉼터 접근성 부족, 고령인구 집중이 결합될 때 커진다.

데이터 설명

분석 단위는 생활권 또는 250~500m 격자 1개이다. 본 시범 데이터는 50개 생활권에 대해 평균 최고기온, 폭염일수, 열대야일수, 녹지율, 그늘 점수, 고령인구, 독거노인 수, 쉼터 접근성, 병의원 접근성 등을 포함한다.

분석 방법

폭염 노출 점수, 고령층 취약성 점수, 쉼터 부족 점수, 냉각환경 부족 점수를 산출한 뒤 가중합으로 폭염 취약지수를 만들었다. 이후 위험등급을 나누고 상관분석, 회귀분석, 군집분석, 지도 시각화를 수행한다.

분석 결과

시범 결과에서 위험 상위지역은 대체로 고령·독거노인 수가 많고, 녹지율이 낮으며, 무더위쉼터까지의 거리가 먼 생활권이었다. 특히 위험지수가 높은 곳은 '더운 곳'이 아니라 '더운 환경 속에 취약한 사람이 많이 사는 곳'이었다.

한계점

본 자료는 시범용 가상 데이터이므로 실제 정책 제안 전에는 행정동별 고령자현황, 무더위쉼터 좌표, S-DoT 환경정보, 기상청 폭염일수, 현장 관찰 자료로 교체해야 한다.
