SPSS 분석 가이드 비 오는 날 통학 위험도 분석

이 가이드는 05_dataset.xlsx 또는 05_dataset.csv를 SPSS에 불러와 우천 시 통학 위험을 분석하는 절차를 설명한다.

1. 데이터 불러오기

SPSS 실행 → 파일 → 열기 → 데이터 선택

파일 유형을 Excel 또는 CSV로 선택하고 05_dataset 파일을 연다.

변수 보기에서 obs_id, date, day, route_id, route_name, risk_grade는 명목 또는 서열로 지정한다.

rainfall_mm, bus_delay_min, avg_late_min, total_rain_risk_score 등은 척도로 지정한다.

2. 분석 절차 요약

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 시범 데이터 또는 실제 결합 데이터를 SPSS로 불러오기 05_dataset.csv 또는 05_dataset.xlsx 파일 → 열기 → 데이터 변수 보기에서 명목·서열·척도 측도 확인
2. 빈도분석 위험등급과 비 여부 분포 확인 rain_dummy, risk_grade 분석 → 기술통계량 → 빈도분석 비 오는 날과 아닌 날의 관찰값 수 확인
3. 기술통계 강수량, 버스지연, 지각시간, 위험점수의 평균·최대값 확인 rainfall_mm, bus_delay_min, avg_late_min, total_rain_risk_score 분석 → 기술통계량 → 기술통계 우천일 평균 위험 수준과 극단값 확인
4. 독립표본 t검정 비 오는 날과 비 안 오는 날의 평균 차이 검정 집단: rain_dummy / 검정변수: bus_delay_min, avg_late_min, total_rain_risk_score 분석 → 평균비교 → 독립표본 t 검정 p<.05이면 날씨에 따른 차이가 통계적으로 유의
5. 상관분석 강수량과 통학 위험 요인의 관계 확인 rainfall_mm, bus_delay_min, road_speed_kmh, avg_late_min, total_rain_risk_score 분석 → 상관분석 → 이변량 상관계수 강수량이 높을수록 버스지연과 위험점수가 증가하는지 확인
6. 회귀분석 우천 통학 위험점수를 예측하는 핵심 요인 파악 종속: total_rain_risk_score / 독립: rainfall_mm, bus_delay_min, sidewalk_width_m, drainage_risk_1_5, visibility_risk_1_5, traffic_congestion_1_5 분석 → 회귀분석 → 선형 표준화계수 β로 영향력이 큰 요인 확인
7. 군집분석 통학구간을 위험 유형별로 분류 weather_risk_score, environment_risk_score, commute_risk_score, exposure_score, accident_score 분석 → 분류분석 → K-평균 군집분석 우천교통지연형, 배수·시야위험형, 저위험형 등으로 명명
8. 지도 시각화 학교 주변 위험구간을 산점도로 표시 x_m, y_m, total_rain_risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 점 크기와 색으로 위험도를 표현하고 구간 ID 표시

3. 주요 해석 문장 예시

비 오는 날의 평균 버스 지연시간이 비가 오지 않는 날보다 높게 나타나면, 우천이 교통운영 위험을 높인다고 해석할 수 있다.

강수량과 총 위험점수 간 상관이 양(+)의 방향이면, 강수량 증가가 통학 위험 증가와 관련 있음을 의미한다.

회귀분석에서 drainage_risk_1_5 또는 visibility_risk_1_5의 영향이 크면 배수 정비와 시야 확보가 주요 개선 과제가 된다.

위험지도에서 학교 정문, 버스정류장, 무신호 횡단지점 주변이 크게 표시되면 우천 시 교통지도 우선 배치 대상이다.

4. SPSS Syntax 예시

별도 제공된 05_spss_syntax.sps 파일을 열어 실행할 수 있다. 파일 경로는 SPSS 환경에 맞게 수정해야 한다.

Syntax

05_spss_syntax.sps

원본: 05_spss_syntax.sps

* Project 05: Rainy day commuting risk analysis.
* Change FILE path to your local file path before running.

GET DATA
  /TYPE=TXT
  /FILE='05_dataset.csv'
  /ENCODING='UTF8'
  /DELCASE=LINE
  /DELIMITERS=','
  /ARRANGEMENT=DELIMITED
  /FIRSTCASE=2
  /VARIABLES=
    obs_id A8 date A10 day A2 route_id A4 route_name A40 latitude F10.6 longitude F10.6 x_m F8.0 y_m F8.0
    distance_school_m F8.0 rainfall_mm F8.1 rain_dummy F1.0 temp_c F8.1 wind_speed_ms F8.1 student_flow_2h F8.0
    crosswalk_count_100m F2.0 bus_stop_count_100m F2.0 sidewalk_width_m F8.1 slope_risk_1_5 F1.0 drainage_risk_1_5 F1.0
    visibility_risk_1_5 F1.0 puddle_count F2.0 traffic_congestion_1_5 F1.0 bus_delay_min F8.1 road_speed_kmh F8.1
    accident_count_3yr F3.0 ped_accident_3yr F3.0 rain_accident_3yr F3.0 avg_late_min F8.1 perceived_risk_1_5 F1.0
    weather_risk_score F8.1 environment_risk_score F8.1 commute_risk_score F8.1 exposure_score F8.1 accident_score F8.1
    total_rain_risk_score F8.1 risk_grade A12 recommended_action A80 priority_rank F3.0.
EXECUTE.

VALUE LABELS rain_dummy 0 '비 안 옴' 1 '비 옴'.

FREQUENCIES VARIABLES=rain_dummy risk_grade.
DESCRIPTIVES VARIABLES=rainfall_mm bus_delay_min avg_late_min total_rain_risk_score.

T-TEST GROUPS=rain_dummy(0 1)
  /VARIABLES=bus_delay_min avg_late_min total_rain_risk_score
  /MISSING=ANALYSIS.

CORRELATIONS
  /VARIABLES=rainfall_mm bus_delay_min road_speed_kmh avg_late_min total_rain_risk_score
  /PRINT=TWOTAIL NOSIG
  /MISSING=PAIRWISE.

REGRESSION
  /DEPENDENT total_rain_risk_score
  /METHOD=ENTER rainfall_mm bus_delay_min sidewalk_width_m drainage_risk_1_5 visibility_risk_1_5 traffic_congestion_1_5.

QUICK CLUSTER weather_risk_score environment_risk_score commute_risk_score exposure_score accident_score
  /CRITERIA=CLUSTER(3) MXITER(10)
  /SAVE CLUSTER.
