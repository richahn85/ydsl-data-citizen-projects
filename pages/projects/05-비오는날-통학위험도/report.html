상위 노트: 비오는날 통학위험도

원본 문서: 05_report.docx

비 오는 날 통학 위험도 분석 용문고 주변 우천 시 통학 안전·지각 위험 예측과 개선 제안

프로젝트 05 | 학생 제출용 완성 보고서 | 데이터는 공식 공공데이터 구조를 반영한 시범용 가상 데이터

1. 프로젝트 개요

본 프로젝트는 비 오는 날 학생들의 통학 안전과 지각 가능성이 어떻게 달라지는지를 데이터로 분석한다. 문제의 핵심은 “날씨가 통학 안전과 지각률에 어떤 영향을 미치는가”이다. 비 오는 날은 보행 시야가 좁아지고, 우산으로 인해 학생 간 보행 충돌이 증가하며, 버스 지연과 차량 혼잡이 함께 발생한다. 따라서 이 주제는 개인의 조심성 문제가 아니라 보행환경, 교통운영, 날씨가 결합된 시스템 문제로 보아야 한다.

항목 내용
문제의식 날씨가 통학 안전과 지각률에 어떤 영향을 미치는가
핵심 데이터 강수량, 기온, 풍속, 버스 지연, 도로 속도, 보행자 사고, 학생 체감 위험 설문
분석 방법 날씨별 사고·지연 패턴 분석, 위험점수화, 회귀분석, 지도 시각화
산출물 우천 시 통학 안전 가이드, 위험지도, 우선 개선구간 제안서
연결 전공 환경공학, 데이터사이언스, 기후 리스크 분석, 교통공학, 공공정책

2. 문제 정의 카드

질문 답변
내가 본 문제는 무엇인가? 비 오는 날 등교 시간에 학생들이 좁은 보도에서 우산을 쓰고 이동하면서 시야가 가려지고, 버스가 늦어져 지각 가능성도 높아지는 문제를 보았다.
왜 이 문제가 중요한가? 비 오는 날의 위험은 학생 개인의 부주의만으로 설명되지 않는다. 보도 폭, 배수 상태, 차량 혼잡, 버스 지연이 결합되어 발생하는 구조적 위험이다.
어떤 데이터를 사용할 것인가? 기상청 강수량·기온·풍속, 서울시 버스도착·버스위치 정보, 도로 통행속도, 횡단보도·정류장 위치, TAAS 교통사고 GIS, 학생 체감 위험 설문을 사용한다.
어떤 분석을 할 것인가? 비 오는 날과 맑은 날의 버스 지연·지각시간·위험점수 차이를 비교하고, 강수량이 위험점수에 미치는 영향을 회귀분석으로 확인한다.
누구에게 제안할 것인가? 학교, 학생회, 학부모회, 성북구청 교통행정 부서, 경찰서 교통안전 담당 부서에 제안한다.
전공 관심과 어떻게 연결되는가? 기후 리스크와 도시교통 데이터를 결합하여 학생 안전을 개선하는 사람 중심 데이터사이언스 프로젝트로 확장할 수 있다.

3. 데이터 설명

분석 단위는 학생×날짜×통학구간이다. 10개 통학구간을 5일 동안 관찰한 형태로 총 50행의 시범 데이터를 구성하였다. 실제 프로젝트에서는 공공데이터와 학생 현장조사를 결합하여 동일한 구조로 대체할 수 있다.

데이터명 제공기관 활용 변수 확보 방식
기상청 단기예보 조회서비스 기상청 강수량, 기온, 풍속 API, XML/JSON
서울시 버스도착 정보 조회 서울특별시 도착예정시간, 현재위치, 지연시간 API, 인증키 필요
서울시 도로별 일자별 통행속도 서울특별시 도로별 평균 속도, 혼잡도 OpenAPI/File
서울시 횡단보도·정류장 위치정보 서울특별시 횡단보도 수, 정류장 수, 좌표 파일/API
TAAS 교통사고 GIS 한국도로교통공단 사고건수, 보행자 사고, 사고다발지역 지도 조회 후 수작업 코딩
학생 현장조사 용문고 학생 물웅덩이, 체감위험, 지각시간 Google Form/관찰지

4. 분석 방법

분석 단계 분석 목적 사용 변수 해석 포인트

기술통계 강수량·버스지연·지각시간의 평균 확인 rainfall_mm, bus_delay_min, avg_late_min 비 오는 날 평균 위험 수준 파악
t검정 비 오는 날과 아닌 날의 평균 차이 검정 rain_dummy, bus_delay_min, avg_late_min, risk_score p값으로 우천 효과 확인
상관분석 강수량과 위험요인 관계 확인 rainfall_mm, bus_delay_min, road_speed_kmh 강수량이 증가할수록 버스지연과 위험점수가 증가하는지 확인

7. 한계점과 후속 연구

본 데이터는 시범용 가상 데이터이므로 실제 정책 제안 전에는 공식 공공데이터와 현장 관찰 데이터로 교체해야 한다.

우천 사고는 단순 사고건수만으로 파악하기 어렵기 때문에 사고 당시 기상 상태를 TAAS 또는 뉴스·현장 기록과 함께 수작업 코딩할 필요가 있다.

후속 연구에서는 비 오는 날, 눈 오는 날, 폭염일 등 기후 조건별 통학 리스크를 비교할 수 있다.

학생별 실제 GPS 경로를 수집할 경우 개인정보 보호와 익명화 원칙을 반드시 지켜야 한다.

8. 나의 진로·전공 관심과의 연결

이 프로젝트는 환경공학, 데이터사이언스, 교통공학, 공공정책이 만나는 주제이다. 학생은 날씨 데이터를 단순히 “비가 온다”는 정보로 보는 것이 아니라, 강수량이 보행환경과 교통흐름, 지각 가능성, 사고위험에 어떤 영향을 미치는지 분석하게 된다. 이는 기후변화 시대에 필요한 도시 안전 데이터 분석 역량과 연결된다.

9. 에세이형 성찰 문장

처음에는 비 오는 날 지각이 학생 개인의 준비 부족 때문이라고 생각했다. 그러나 데이터를 보며 강수량이 많을수록 버스 지연, 보행 시야 방해, 물웅덩이, 체감위험이 함께 증가한다는 점을 확인했다. 이제 나는 지각과 안전을 개인 책임으로만 보지 않고, 학생이 매일 통과하는 교통·기후·보행 시스템의 결과로 바라보게 되었다.
