date 조사일자 YYYY-MM-DD 명목 강수·등교 조사 일자 기상청 단기예보/학생 현장조사 대체

latitude 위도 도 척도 구간 대표 위치 위도 지도 좌표 기반 대체

y_m 학교 기준 Y좌표 m 척도 학교 정문 기준 상대 Y좌표 지도 시각화용
distance_school_m 학교와의 거리 m 척도 학교 정문과 구간 대표지점의 거리 계산값
rainfall_mm 강수량 mm 척도 해당 등교 시간대 강수량 기상청 단기예보 조회서비스로 대체 가능
rain_dummy 비 여부 0/1 명목 1=비 오는 날, 0=비 안 오는 날 rainfall_mm 기준
temp_c 기온 ℃ 척도 등교 시간대 기온 기상청 단기예보
wind_speed_ms 풍속 m/s 척도 등교 시간대 바람 세기 기상청 단기예보
student_flow_2h 등교 2시간 유동 학생수 명 척도 등교시간 해당 구간 추정 보행량 현장 관찰 또는 설문
crosswalk_count_100m 반경 100m 횡단보도 수 개 척도 구간 주변 횡단보도 수 서울시 횡단보도 위치정보
bus_stop_count_100m 반경 100m 버스정류장 수 개 척도 구간 주변 버스정류소 수 서울시 버스정류소 위치정보
sidewalk_width_m 보도 폭 m 척도 보행로 평균 폭 현장 측정/보도통계
slope_risk_1_5 경사 위험 1~5 서열 비 오는 날 미끄럼 위험에 영향을 주는 경사 수준 현장 관찰
drainage_risk_1_5 배수 불량 위험 1~5 서열 물 고임과 배수 상태 위험 현장 관찰
visibility_risk_1_5 시야 위험 1~5 서열 우산·차량·상가 구조물로 인한 시야 방해 현장 관찰
puddle_count 물웅덩이 수 개 척도 30분 관찰 중 보행로 물웅덩이 수 현장 관찰
traffic_congestion_1_5 차량 혼잡도 1~5 서열 등교 시간대 도로 혼잡도 서울시 통행속도/관찰
bus_delay_min 버스 지연시간 분 척도 예상 도착 대비 실제 지연 서울시 버스도착 API/학생 기록
road_speed_kmh 도로 평균 속도 km/h 척도 해당 구간 또는 인접 도로 평균 속도 서울시 도로별 일자별 통행속도
accident_count_3yr 최근 3년 사고건수 건 척도 해당 구간 주변 전체 교통사고 TAAS GIS
ped_accident_3yr 최근 3년 보행자 사고건수 건 척도 해당 구간 주변 보행자 사고 TAAS GIS
rain_accident_3yr 최근 3년 우천 관련 사고건수 건 척도 비 오는 날 또는 젖은 노면 관련 사고 추정 TAAS/수작업 코딩
avg_late_min 평균 지각시간 분 척도 해당 관찰조건에서 평균 지각시간 학생 설문/도착시간 기록
perceived_risk_1_5 학생 체감 위험 1~5 서열 학생이 느끼는 비 오는 날 통학 위험도 학생 설문
weather_risk_score 날씨 위험점수 0~100 척도 강수량·풍속·기온 기반 점수 계산값
environment_risk_score 보행환경 위험점수 0~100 척도 보도 폭·배수·시야·경사 기반 점수 계산값
commute_risk_score 교통운영 위험점수 0~100 척도 버스지연·혼잡·도로속도 기반 점수 계산값
exposure_score 노출 위험점수 0~100 척도 학생 유동인구·횡단보도·정류장 노출도 계산값
accident_score 사고 이력 점수 0~100 척도 사고건수와 보행자·우천 사고 이력 계산값
total_rain_risk_score 종합 우천 통학 위험점수 0~100 척도 종합 위험 우선순위 점수 계산값
risk_grade 위험등급 낮음/보통/높음/매우 높음 서열 개선 우선순위 등급 계산값
recommended_action 권장 조치 문자 명목 구간별 개선 제안 분석 결과 기반
priority_rank 개선우선순위 순위 서열 총점 기준 우선순위 계산값

SPSS_Steps

SPSS_Steps: 9행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 시범 데이터 또는 실제 결합 데이터를 SPSS로 불러오기 05_dataset.csv 또는 05_dataset.xlsx 파일 → 열기 → 데이터 변수 보기에서 명목·서열·척도 측도 확인
2. 빈도분석 위험등급과 비 여부 분포 확인 rain_dummy, risk_grade 분석 → 기술통계량 → 빈도분석 비 오는 날과 아닌 날의 관찰값 수 확인
3. 기술통계 강수량, 버스지연, 지각시간, 위험점수의 평균·최대값 확인 rainfall_mm, bus_delay_min, avg_late_min, total_rain_risk_score 분석 → 기술통계량 → 기술통계 우천일 평균 위험 수준과 극단값 확인
4. 독립표본 t검정 비 오는 날과 비 안 오는 날의 평균 차이 검정 집단: rain_dummy / 검정변수: bus_delay_min, avg_late_min, total_rain_risk_score 분석 → 평균비교 → 독립표본 t 검정 p<.05이면 날씨에 따른 차이가 통계적으로 유의
5. 상관분석 강수량과 통학 위험 요인의 관계 확인 rainfall_mm, bus_delay_min, road_speed_kmh, avg_late_min, total_rain_risk_score 분석 → 상관분석 → 이변량 상관계수 강수량이 높을수록 버스지연과 위험점수가 증가하는지 확인
6. 회귀분석 우천 통학 위험점수를 예측하는 핵심 요인 파악 종속: total_rain_risk_score / 독립: rainfall_mm, bus_delay_min, sidewalk_width_m, drainage_risk_1_5, visibility_risk_1_5, traffic_congestion_1_5 분석 → 회귀분석 → 선형 표준화계수 β로 영향력이 큰 요인 확인
7. 군집분석 통학구간을 위험 유형별로 분류 weather_risk_score, environment_risk_score, commute_risk_score, exposure_score, accident_score 분석 → 분류분석 → K-평균 군집분석 우천교통지연형, 배수·시야위험형, 저위험형 등으로 명명
8. 지도 시각화 학교 주변 위험구간을 산점도로 표시 x_m, y_m, total_rain_risk_score, risk_grade 그래프 → 차트 작성기 → 산점도 점 크기와 색으로 위험도를 표현하고 구간 ID 표시
