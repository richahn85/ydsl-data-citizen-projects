변수명 변수 설명 측정 단위 SPSS 측도 코딩/값 비고
student_id 익명 학생 ID 문자 명목 S001~S050 개인 식별 불가한 익명 ID
grade 학년 1~3 서열 1=1학년, 2=2학년, 3=3학년 학년별 차이 분석
gender 성별 남/여 명목 남, 여 선택형 설문
main_platform 가장 많이 이용하는 플랫폼 범주 명목 YouTube, TikTok, Instagram, Naver/News, Mixed 주 이용 플랫폼
interest_topic 주 관심 콘텐츠 주제 범주 명목 진로/교육, 정치사회, 게임/엔터 등 추천 콘텐츠 분석 기준
daily_youtube_min 하루 유튜브 사용시간 분 척도 0 이상 정수 자기보고식
daily_sns_min 하루 SNS 사용시간 분 척도 0 이상 정수 자기보고식
shortform_min 하루 숏폼 시청시간 분 척도 0 이상 정수 릴스/쇼츠/틱톡 포함
news_min 하루 뉴스·시사 콘텐츠 이용시간 분 척도 0 이상 정수 시사정보 노출 정도
news_source_diversity 정보 출처 다양성 개수 척도 1~6 뉴스/친구/SNS/검색/공식기관 등
content_category_diversity 콘텐츠 카테고리 다양성 개수 척도 1~6 일주일 내 다른 카테고리 수
recommended_same_topic_ratio 추천 콘텐츠 중 동일 주제 비율 % 척도 0~100 필터버블 가능성 지표
recommended_opposing_view_ratio 반대 관점 콘텐츠 노출 비율 % 척도 0~100 다양한 관점 노출 정도
recommendation_dependence 추천 피드 의존도 1~5 서열 1=낮음, 5=높음 추천 알고리즘 의존 정도
algorithm_awareness 추천 알고리즘 이해도 1~5 서열 1=전혀 모름, 5=잘 이해 알고리즘 리터러시
echo_chamber_awareness 확증편향/에코챔버 인식 1~5 서열 1=낮음, 5=높음 정보 편향 인식
fact_check_frequency 팩트체크 빈도 1~5 서열 1=거의 안 함, 5=자주 함 비판적 정보 검증
content_variety_behavior 의도적 다양한 콘텐츠 탐색 행동 1~5 서열 1=낮음, 5=높음 관심사 확장 행동
algorithm_trust 추천 알고리즘 신뢰도 1~5 서열 1=낮음, 5=높음 추천 결과 신뢰 수준
exposure_to_extreme_content 극단·자극 콘텐츠 노출 경험 1~5 서열 1=낮음, 5=높음 위험 노출 변수
misinformation_experience 허위정보 경험 빈도 1~5 서열 1=낮음, 5=높음 정보 오염 경험
discussion_with_others 타인과 관점 토론 빈도 1~5 서열 1=낮음, 5=높음 사회적 검증 행동
information_bias_perception 정보 편향 체감도 1~5 서열 1=낮음, 5=높음 본인이 느끼는 편향
digital_literacy_score 디지털 리터러시 점수 0~100 척도 높을수록 우수 알고리즘 이해·검증·다양성 행동 종합
content_bias_index 콘텐츠 편향 지수 0~100 척도 높을수록 편향 위험 큼 동일주제 비율·추천 의존도·자극노출 기반
algorithmic_bias_risk_score 알고리즘 정보 편향 위험점수 0~100 척도 70 이상 고위험 최종 위험지표
user_type 정보 이용 유형 범주 명목 균형탐색형, 숏폼몰입형 등 군집/분류 결과
recommended_education 추천 교육 프로그램 범주 명목 팩트체크 집중 교육 등 개선 제안용

Survey_Items

Survey_Items: 13행 x 4열 / 미리보기 10행 x 4열

문항번호 설문 문항 척도/코딩 연결 변수
Q1 나는 유튜브·SNS 추천 피드가 내가 볼 콘텐츠를 많이 결정한다고 느낀다. 1=전혀 그렇지 않다 ~ 5=매우 그렇다 recommendation_dependence
Q2 나는 추천 알고리즘이 어떤 기준으로 콘텐츠를 보여주는지 어느 정도 이해한다. 1~5 algorithm_awareness
Q3 나는 내가 좋아하는 관점과 다른 관점의 콘텐츠도 일부러 찾아본다. 1~5 content_variety_behavior
Q4 나는 온라인 콘텐츠를 보기 전에 출처와 근거를 확인한다. 1~5 fact_check_frequency
Q5 나는 같은 주제의 영상이 계속 추천되는 경험을 자주 한다. 1~5 information_bias_perception
Q6 나는 자극적이거나 극단적인 콘텐츠를 추천받은 경험이 있다. 1~5 exposure_to_extreme_content
Q7 나는 온라인에서 본 정보를 친구·가족·교사와 토론하거나 확인한다. 1~5 discussion_with_others
Q8 나는 온라인 정보가 내 생각을 한쪽으로 기울게 만들 수 있다고 생각한다. 1~5 echo_chamber_awareness
Q9 나는 추천 피드보다 검색·공식기관·다양한 언론 자료를 함께 활용한다. 1~5 news_source_diversity

SPSS_Steps

SPSS_Steps: 9행 x 5열

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러와 변수 측도 확인 전체 변수 파일 → 열기 → 데이터 문자형 ID와 명목형 변수는 명목, 점수형 변수는 척도로 설정
2. 빈도분석 주 이용 플랫폼과 정보 이용 유형 분포 확인 main_platform, user_type 분석 → 기술통계량 → 빈도분석 어떤 플랫폼과 유형이 많은지 파악
3. 기술통계 이용시간·편향지수·리터러시 평균 확인 shortform_min, digital_literacy_score, content_bias_index, algorithmic_bias_risk_score 분석 → 기술통계량 → 기술통계 위험 평균과 분포 확인
4. 상관분석 추천 의존도·숏폼 시간·디지털 리터러시와 편향 위험 관계 확인 recommendation_dependence, shortform_min, digital_literacy_score, algorithmic_bias_risk_score 분석 → 상관분석 → 이변량 상관계수 위험을 높이거나 낮추는 요인 해석
5. 일원배치 분산분석 정보 이용 유형별 위험점수 차이 검증 user_type → algorithmic_bias_risk_score 분석 → 평균비교 → 일원배치 분산분석 어떤 유형이 가장 위험한지 비교
6. 회귀분석 알고리즘 편향 위험을 예측하는 주요 변수 확인 종속: algorithmic_bias_risk_score / 독립: shortform_min, recommendation_dependence, fact_check_frequency, content_category_diversity, digital_literacy_score 분석 → 회귀분석 → 선형 위험 예측 요인을 찾고 교육 우선순위 결정
7. 군집분석 학생의 정보 이용 유형 자동 분류 shortform_min, source_diversity, fact_check_frequency, recommendation_dependence, digital_literacy_score 분석 → 분류분석 → K-평균 군집분석 균형탐색형·알고리즘의존형 등 유형화
8. 시각화 위험점수와 리터러시의 관계 표시 digital_literacy_score, algorithmic_bias_risk_score 그래프 → 차트 작성기 고위험 학생군과 교육 필요 지점 시각화
