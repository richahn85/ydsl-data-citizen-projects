SPSS 분석 가이드: AI 추천 알고리즘이 청소년 정보 편향에 미치는 영향

이 가이드는 30_dataset.xlsx 또는 30_dataset.csv를 SPSS에서 불러와 분석하는 절차를 안내한다. 데이터는 설문 기반 시범용 가상 데이터이며, 실제 조사에서는 익명성과 개인정보 보호를 반드시 지켜야 한다.

1. 데이터 불러오기

SPSS → 파일 → 열기 → 데이터 → 30_dataset.xlsx 선택 → 첫 행을 변수명으로 읽기. 이후 변수 보기에서 student_id, gender, main_platform, interest_topic, user_type은 명목으로, 점수형 변수는 척도로 지정한다.

2. 기본 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러와 변수 측도 확인 전체 변수 파일 → 열기 → 데이터 문자형 ID와 명목형 변수는 명목, 점수형 변수는 척도로 설정
2. 빈도분석 주 이용 플랫폼과 정보 이용 유형 분포 확인 main_platform, user_type 분석 → 기술통계량 → 빈도분석 어떤 플랫폼과 유형이 많은지 파악
3. 기술통계 이용시간·편향지수·리터러시 평균 확인 shortform_min, digital_literacy_score, content_bias_index, algorithmic_bias_risk_score 분석 → 기술통계량 → 기술통계 위험 평균과 분포 확인
4. 상관분석 추천 의존도·숏폼 시간·디지털 리터러시와 편향 위험 관계 확인 recommendation_dependence, shortform_min, digital_literacy_score, algorithmic_bias_risk_score 분석 → 상관분석 → 이변량 상관계수 위험을 높이거나 낮추는 요인 해석
5. 일원배치 분산분석 정보 이용 유형별 위험점수 차이 검증 user_type → algorithmic_bias_risk_score 분석 → 평균비교 → 일원배치 분산분석 어떤 유형이 가장 위험한지 비교
6. 회귀분석 알고리즘 편향 위험을 예측하는 주요 변수 확인 종속: algorithmic_bias_risk_score / 독립: shortform_min, recommendation_dependence, fact_check_frequency, content_category_diversity, digital_literacy_score 분석 → 회귀분석 → 선형 위험 예측 요인을 찾고 교육 우선순위 결정
7. 군집분석 학생의 정보 이용 유형 자동 분류 shortform_min, source_diversity, fact_check_frequency, recommendation_dependence, digital_literacy_score 분석 → 분류분석 → K-평균 군집분석 균형탐색형·알고리즘의존형 등 유형화
8. 시각화 위험점수와 리터러시의 관계 표시 digital_literacy_score, algorithmic_bias_risk_score 그래프 → 차트 작성기 고위험 학생군과 교육 필요 지점 시각화

3. 핵심 해석 문장 예시

상관분석 결과 추천 피드 의존도와 알고리즘 편향 위험점수 간 정적 상관이 나타난다면, 추천 피드에 많이 의존할수록 정보 편향 위험이 높다고 해석할 수 있다.

디지털 리터러시 점수와 편향 위험점수 간 부적 상관이 나타난다면, 알고리즘 이해·팩트체크·출처 다양화 행동이 보호요인임을 의미한다.

군집분석 결과 알고리즘의존형 집단의 위험점수가 높다면, 전체 학생에게 동일한 교육보다 유형별 맞춤 교육이 필요하다는 제안으로 연결할 수 있다.

4. 보고서 작성 요령

결과표를 그대로 나열하지 말고, 반드시 “그래서 학교가 무엇을 해야 하는가”로 연결한다. 예를 들어 고위험 집단에는 알고리즘 이해와 팩트체크 교육을, 다양성 부족 집단에는 관심사 확장 미션을 제안한다.

* Project 30: AI recommendation algorithms and youth information bias.
GET DATA
  /TYPE=XLSX
  /FILE='30_dataset.xlsx'
  /SHEET=name 'Data'
  /READNAMES=ON.
EXECUTE.

VARIABLE LEVEL grade (ORDINAL) gender main_platform interest_topic user_type recommended_education (NOMINAL)
 /daily_youtube_min daily_sns_min shortform_min news_min news_source_diversity content_category_diversity
 recommended_same_topic_ratio recommended_opposing_view_ratio recommendation_dependence algorithm_awareness
 echo_chamber_awareness fact_check_frequency content_variety_behavior algorithm_trust exposure_to_extreme_content
 misinformation_experience discussion_with_others information_bias_perception digital_literacy_score content_bias_index
 algorithmic_bias_risk_score (SCALE).

FREQUENCIES VARIABLES=main_platform user_type recommended_education.
DESCRIPTIVES VARIABLES=shortform_min recommendation_dependence digital_literacy_score content_bias_index algorithmic_bias_risk_score.
CORRELATIONS
 /VARIABLES=shortform_min recommendation_dependence fact_check_frequency content_category_diversity digital_literacy_score algorithmic_bias_risk_score
 /PRINT=TWOTAIL NOSIG.
ONEWAY algorithmic_bias_risk_score BY user_type
 /STATISTICS DESCRIPTIVES.
REGRESSION
 /DEPENDENT algorithmic_bias_risk_score
 /METHOD=ENTER shortform_min recommendation_dependence fact_check_frequency content_category_diversity digital_literacy_score.
QUICK CLUSTER shortform_min news_source_diversity fact_check_frequency recommendation_dependence digital_literacy_score
 /CRITERIA=CLUSTER(4) MXITER(10)
 /SAVE CLUSTER.
