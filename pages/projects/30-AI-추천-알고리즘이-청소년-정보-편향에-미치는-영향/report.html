AI 추천 알고리즘이 청소년 정보 편향에 미치는 영향

청소년 디지털 리터러시 교육안 개발을 위한 데이터 기반 분석 보고서

1. 프로젝트 개요

항목 내용
문제의식 유튜브·SNS 추천 알고리즘은 학생들의 세계관을 좁히는가
왜 중요한가 청소년은 추천 피드를 통해 정보를 접하는 비중이 크다. 정보가 반복적으로 한 방향으로 추천되면 세계관이 좁아지고, 사회적 쟁점에 대한 균형 잡힌 판단이 어려워질 수 있다.
학생 경험과 연결 학생들이 “알고리즘이 계속 비슷한 영상만 보여준다”는 경험을 실제 데이터로 측정하고, 정보 편향을 줄이는 행동 전략을 설계한다.
분석 방법 콘텐츠 카테고리 분석, 편향 지수 개발, 상관분석, 회귀분석, 군집분석
산출물 청소년 디지털 리터러시 교육안
연결 전공 컴퓨터과학, 미디어학, 사회학, AI윤리

2. 문제 정의 카드

항목 내용
내가 본 문제는 무엇인가? 학생들이 숏폼·SNS·유튜브를 통해 정보를 얻지만, 추천 피드가 비슷한 주제와 관점의 콘텐츠를 반복적으로 보여주는 현상이 있다.
왜 이 문제가 나에게 중요한가? 온라인 정보 소비는 단순한 취미가 아니라 진로, 사회 인식, 가치관 형성에 영향을 준다. 따라서 학생의 세계관을 넓히는 교육적 개입이 필요하다.
어떤 데이터를 사용할 것인가? 익명 콘텐츠 이용 패턴, 주 이용 플랫폼, 관심 주제, 추천 콘텐츠 유형, 정보 출처 다양성, 팩트체크 습관, 알고리즘 이해도 설문을 사용한다.
어떤 분석을 할 것인가? 동일 주제 추천 비율, 반대 관점 노출 비율, 콘텐츠 다양성, 추천 피드 의존도 등을 결합해 콘텐츠 편향 지수와 알고리즘 편향 위험점수를 산출한다.
누구에게 제안할 것인가? 학교, 정보·컴퓨터 교사, 진로부, 학생자치회, 학부모회에 디지털 리터러시 교육안을 제안한다.
전공 관심과 어떻게 연결되는가? 컴퓨터과학의 추천 시스템 원리, 미디어학의 정보 소비, 사회학의 확증편향, AI윤리의 책임 있는 알고리즘 문제를 함께 탐구한다.

5. 분석 방법

분석 단계 분석 목적 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러와 변수 측도 확인 파일 → 열기 → 데이터 문자형 ID와 명목형 변수는 명목, 점수형 변수는 척도로 설정
2. 빈도분석 주 이용 플랫폼과 정보 이용 유형 분포 확인 분석 → 기술통계량 → 빈도분석 어떤 플랫폼과 유형이 많은지 파악
3. 기술통계 이용시간·편향지수·리터러시 평균 확인 분석 → 기술통계량 → 기술통계 위험 평균과 분포 확인
4. 상관분석 추천 의존도·숏폼 시간·디지털 리터러시와 편향 위험 관계 확인 분석 → 상관분석 → 이변량 상관계수 위험을 높이거나 낮추는 요인 해석
5. 일원배치 분산분석 정보 이용 유형별 위험점수 차이 검증 분석 → 평균비교 → 일원배치 분산분석 어떤 유형이 가장 위험한지 비교
6. 회귀분석 알고리즘 편향 위험을 예측하는 주요 변수 확인 분석 → 회귀분석 → 선형 위험 예측 요인을 찾고 교육 우선순위 결정
7. 군집분석 학생의 정보 이용 유형 자동 분류 분석 → 분류분석 → K-평균 군집분석 균형탐색형·알고리즘의존형 등 유형화
8. 시각화 위험점수와 리터러시의 관계 표시 그래프 → 차트 작성기 고위험 학생군과 교육 필요 지점 시각화

시범 데이터에서는 숏폼 시청시간과 추천 피드 의존도가 높을수록 알고리즘 정보 편향 위험점수가 높아지는 경향이 나타났다. 반대로 정보 출처 다양성, 팩트체크 빈도, 의도적 다양한 콘텐츠 탐색 행동이 높은 학생은 위험점수가 낮았다. 이는 청소년의 정보 편향이 개인의 성향만이 아니라 플랫폼 구조, 추천 시스템, 미디어 리터러시 수준이 함께 만든 결과임을 보여준다.

7. 개선 제안서

제안 대상 제안 내용 근거 데이터 기대 효과 실행 난이도
학교 1학년 공통 디지털 리터러시 수업에 “추천 알고리즘과 필터버블” 단원을 신설한다. 알고리즘 이해도가 낮고 추천 피드 의존도가 높은 학생군의 위험점수 상승 추천 시스템을 비판적으로 이해하고 정보 편향을 줄임 중
정보·컴퓨터 교사 유튜브·SNS 추천 피드 관찰일지를 작성하게 하고, 동일 주제 반복 추천 비율을 직접 계산하게 한다. recommended_same_topic_ratio와 content_bias_index 활용 학생이 자신의 정보 소비를 데이터로 성찰 하
진로부 진로·사회 이슈 탐색 과제에서 최소 3개 출처와 반대 관점 1개를 포함하도록 안내한다. 정보 출처 다양성이 높을수록 위험점수 감소 진로 탐색의 폭과 사회적 판단력 확장 하
학생자치회 “관심사 확장 주간”을 운영해 서로 다른 주제의 콘텐츠를 추천하고 토론한다. content_category_diversity가 낮은 학생군 존재 관심사의 다양성과 토론 문화 강화 중
학부모회 가정에서 알고리즘 추천·팩트체크·허위정보 대화를 돕는 안내문을 배포한다. 토론 빈도와 팩트체크 빈도가 보호요인으로 작용 가정 내 미디어 대화 문화 확산 중

8. 한계점과 후속 연구

이 보고서는 시범용 가상 데이터에 기반한 분석 예시이므로 실제 학생의 미디어 이용 실태를 대표하지 않는다. 또한 추천 알고리즘의 내부 작동 원리를 직접 측정한 것이 아니라, 학생이 경험한 추천 콘텐츠와 인식 데이터를 기반으로 위험을 추정했다. 후속 연구에서는 학생들이 일정 기간 실제 추천 피드를 기록하고, 콘텐츠 카테고리를 직접 코딩하여 더 정밀한 알고리즘 편향 분석을 수행할 수 있다.

9. 진로·전공 관심과의 연결

이 프로젝트는 컴퓨터과학의 추천 시스템, 미디어학의 콘텐츠 소비, 사회학의 여론 형성과 확증편향, AI윤리의 책임 있는 알고리즘 설계를 통합한다. 학생은 단순히 AI를 사용하는 소비자가 아니라, AI가 인간의 사고와 사회적 판단에 어떤 영향을 주는지 질문하는 시민이 될 수 있다.

10. 에세이형 성찰 문장

나는 알고리즘이 내가 좋아할 만한 콘텐츠를 편리하게 보여주는 도구라고만 생각했다. 그러나 데이터를 분석하며 추천 알고리즘이 나의 관심사를 넓히기도 하지만, 동시에 나의 세계를 좁힐 수도 있다는 점을 보았다. 앞으로 나는 온라인에서 마주치는 정보가 우연히 주어진 것이 아니라 플랫폼의 선택과 나의 클릭이 함께 만든 결과임을 인식하고, 다양한 출처와 관점을 의도적으로 탐색하는 태도를 갖고 싶다.

11. 에세이 질문

내가 자주 보는 추천 콘텐츠는 내 생각을 넓히는가, 좁히는가?

내가 접하는 정보는 내가 선택한 것인가, 알고리즘이 선택한 것인가?

정보 편향을 줄이기 위해 학생 개인, 학교, 플랫폼은 각각 무엇을 해야 하는가?

AI 시대의 디지털 리터러시는 단순한 기기 활용 능력인가, 아니면 세계를 비판적으로 읽는 능력인가?
