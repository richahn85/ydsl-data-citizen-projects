SPSS 분석 가이드: 학교 급식 만족도와 영양 균형 분석

1. 파일 불러오기

SPSS에서 09_dataset.csv 또는 09_dataset.xlsx를 불러온다. CSV를 사용할 경우 인코딩은 UTF-8 또는 UTF-8-SIG로 설정한다.

메뉴: 파일 → 열기 → 데이터 → 파일 유형에서 CSV 또는 Excel 선택

2. 변수 보기 설정

Codebook 시트를 참고해 ID·성별·메뉴유형은 명목, 설문 문항은 서열, 만족도·영양점수·잔반율은 척도로 지정한다.

변수명 SPSS 측도 비고
student_id 명목 개인정보 미포함
grade 서열 1=1학년, 2=2학년, 3=3학년
gender 명목 선택 문항
meal_date 명목 날짜형으로 변환 가능
weekday 명목 월~금
menu_name 명목 요리명 조합
menu_category 명목 한식균형형 등
energy_kcal 척도 CAL_INFO 참조
carbohydrate_g 척도 NTR_INFO 참조
protein_g 척도 NTR_INFO 참조
fat_g 척도 NTR_INFO 참조
sodium_mg 척도 NTR_INFO 참조
fiber_g 척도 시범 변수
calcium_mg 척도 NTR_INFO 참조
iron_mg 척도 NTR_INFO 참조
fruit_veg_servings 척도 학생 분석용 보완변수
allergy_item_count 척도 DDISH_NM 알레르기 번호
spicy_level 서열 1 낮음~5 높음

3. 기본 분석 절차

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러와 변수 보기에서 측도를 지정한다. 전체 변수 파일 → 열기 → 데이터 날짜·범주·척도 변수 구분
2. 기술통계 만족도, 영양균형, 잔반율의 평균과 표준편차를 확인한다. satisfaction_score, nutrition_balance_score, leftover_pct 분석 → 기술통계량 → 기술통계 전체 급식 품질의 기본 수준 파악
3. 메뉴유형별 평균 비교 메뉴 유형별 만족도·영양균형·잔반율 차이를 비교한다. menu_category, satisfaction_score, nutrition_balance_score, leftover_pct 분석 → 평균비교 → 일원배치 분산분석 인기 메뉴와 균형 메뉴의 차이 확인
4. 상관분석 영양균형과 만족도, 잔반율 간 관계를 확인한다. satisfaction_score, nutrition_balance_score, leftover_pct, sodium_mg, fruit_veg_servings 분석 → 상관분석 → 이변량 상관계수 학생 선호와 건강성의 일치/불일치 판단
5. 회귀분석 만족도를 설명하는 요인을 파악한다. 종속: satisfaction_score / 독립: taste_q, freshness_q, portion_q, nutrition_balance_score, processed_food_level 분석 → 회귀분석 → 선형 무엇이 만족도를 가장 크게 높이는지 확인
6. 군집분석 메뉴를 선호도와 영양균형 기준으로 유형화한다. preference_score, nutrition_balance_score, leftover_pct 분석 → 분류분석 → K-평균 군집분석 건강·선호 균형형, 선호 높음·영양 보완형 등 분류
7. 시각화 메뉴 유형별 결과와 선호-영양 차이를 그래프로 표현한다. menu_category, satisfaction_score, nutrition_balance_score, preference_nutrition_gap 그래프 → 차트 작성기 보고서용 핵심 시각화 생성

4. Syntax 예시

GET DATA /TYPE=TXT /FILE='09_dataset.csv' /ENCODING='UTF8' /DELCASE=LINE /DELIMITERS=',' /ARRANGEMENT=DELIMITED /FIRSTCASE=2 /VARIABLES= student_id A8 grade F1.0 gender A2 meal_date A10 weekday A2 menu_name A200 menu_category A30 energy_kcal F8.0 carbohydrate_g F8.1 protein_g F8.1 fat_g F8.1 sodium_mg F8.0 fiber_g F8.1 calcium_mg F8.0 iron_mg F8.1 fruit_veg_servings F8.1 allergy_item_count F8.0 spicy_level F8.0 processed_food_level F8.0 taste_q F8.0 appearance_q F8.0 freshness_q F8.0 portion_q F8.0 variety_q F8.0 nutrition_info_q F8.0 vegetable_accept_q F8.0 satiety_q F8.0 leftover_low_q F8.0 recommend_q F8.0 satisfaction_score F8.2 health_perception_score F8.2 leftover_pct F8.1 nutrition_balance_score F8.1 preference_score F8.1 preference_nutrition_gap F8.1 improvement_priority F8.0 meal_profile A30. EXECUTE.

  • 기술통계.

DESCRIPTIVES VARIABLES=satisfaction_score nutrition_balance_score leftover_pct preference_nutrition_gap improvement_priority /STATISTICS=MEAN STDDEV MIN MAX.

  • 메뉴 유형별 평균 비교.

ONEWAY satisfaction_score nutrition_balance_score leftover_pct BY menu_category /STATISTICS DESCRIPTIVES /MISSING ANALYSIS.

  • 상관분석.

CORRELATIONS /VARIABLES=satisfaction_score nutrition_balance_score leftover_pct sodium_mg fruit_veg_servings processed_food_level /PRINT=TWOTAIL NOSIG /MISSING=PAIRWISE.

  • 회귀분석: 만족도 예측.

REGRESSION /DEPENDENT satisfaction_score /METHOD=ENTER taste_q appearance_q freshness_q portion_q variety_q nutrition_balance_score processed_food_level.

  • K-평균 군집분석.

QUICK CLUSTER preference_score nutrition_balance_score leftover_pct /CRITERIA=CLUSTER(3) MXITER(10) /SAVE CLUSTER.

  • 산점도.

GRAPH /SCATTERPLOT(BIVAR)=nutrition_balance_score WITH satisfaction_score. GRAPH /SCATTERPLOT(BIVAR)=satisfaction_score WITH leftover_pct.

5. 해석 포인트

· 만족도와 영양균형 점수가 동시에 높은 메뉴는 ‘학생 친화형 건강 급식’의 모범 사례로 본다.

· 만족도는 높지만 영양균형이 낮은 메뉴는 조리 방식과 구성 반찬을 보완한다.

· 영양균형은 높지만 만족도가 낮고 잔반율이 높은 메뉴는 학생 수용성을 높이는 방식으로 개선한다.

· 잔반율은 음식물 쓰레기 문제이면서 학생의 실제 섭취 여부를 보여주는 중요한 지표이다.
