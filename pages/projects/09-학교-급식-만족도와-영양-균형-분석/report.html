학교 급식 만족도와 영양 균형 분석 보고서

학생들이 선호하는 급식과 영양적으로 좋은 급식은 일치하는가

1. 프로젝트 개요

본 프로젝트는 학교 급식이 학생들에게 맛있고 만족스러운 동시에 영양적으로 균형 잡힌 식사인지 확인하기 위한 데이터 기반 문제해결 프로젝트이다. 학생이 선호하는 메뉴와 영양적으로 좋은 메뉴가 항상 일치하지 않을 수 있다는 문제의식에서 출발한다.

분석의 핵심 질문은 ‘학생들이 좋아하는 급식은 영양적으로도 균형 잡혀 있는가, 그리고 영양적으로 좋은 급식은 학생들이 실제로 잘 먹고 만족하는가?’이다. 이 질문은 단순한 급식 취향 조사가 아니라, 학생 건강과 학교 급식 운영의 균형을 찾기 위한 질문이다.

연결 전공: 식품영양학, 보건학, 데이터 분석, 경영학

2. 문제 정의 카드

질문 답변
내가 본 문제는 무엇인가? 급식 메뉴 중 학생들이 매우 좋아하지만 영양균형 점수가 낮거나, 반대로 영양적으로 우수하지만 잔반이 많이 남는 메뉴가 존재한다.
왜 중요한가? 급식은 학생의 하루 에너지와 건강을 좌우하지만, 학생이 먹지 않으면 영양적으로 좋은 메뉴도 실제 효과를 얻기 어렵다.
어떤 데이터를 사용할 것인가? 급식 메뉴, 칼로리·영양성분, 알레르기 정보, 잔반율, 학생 만족도 설문, 메뉴 유형 분류 데이터를 사용한다.
어떤 분석을 할 것인가? 메뉴별 만족도 분석, 영양균형 점수화, 잔반율 비교, 상관분석, 회귀분석, 군집분석을 수행한다.
누구에게 제안할 것인가? 학교 급식실, 영양교사, 학생회, 학교운영위원회, 학부모회에 학생 친화형 건강 급식 개선안을 제안한다.
전공 관심과의 연결은? 식품영양학·보건학 관점에서는 건강 식단 설계, 경영학 관점에서는 학생 만족도와 서비스 개선, 데이터 분석 관점에서는 의사결정 지원으로 연결된다.

3. 데이터 설명

이번 파일의 데이터는 NEIS 급식식단정보와 학교급식 영양관리기준의 구조를 반영해 만든 시범용 가상 데이터이다. 실제 프로젝트에서는 학교 급식표와 NEIS API의 급식식단정보를 사용하고, 학생 설문과 잔반 관찰 자료를 결합한다.

분석 단위는 ‘익명 학생 1명이 특정 급식일의 메뉴를 평가한 응답’이다. 10개의 급식 메뉴를 대상으로 각 메뉴당 5명의 학생 평가를 가정하여 총 50행으로 구성했다.

변수명 변수 설명 측정 단위 SPSS 측도
student_id 익명 학생 ID 문자 명목
grade 학년 1/2/3 서열

4. 영양균형 점수화 방식

영양균형 점수는 100점 만점으로 구성하였다. 열량 적정성, 탄수화물·단백질·지방 비율, 나트륨 수준, 과일·채소 제공량을 반영하였다. 이 점수는 학생들의 선호도와 비교하기 위한 교육용 지표이다.

구성요소 가중치 설명
열량 적정성 35% 1식 기준 목표 열량에서 지나치게 벗어나지 않는지 평가
3대 영양소 균형 30% 탄수화물·단백질·지방 에너지 비율이 기준 범위에 가까운지 평가
나트륨 관리 20% 나트륨이 높을수록 감점
과일·채소 제공량 15% 과일·채소가 충분히 포함될수록 가점

5. 분석 방법

분석 단계 분석 목적 사용 변수 SPSS 메뉴 경로 해석 포인트
1. 데이터 불러오기 CSV/XLSX를 SPSS로 불러와 변수 보기에서 측도를 지정한다. 전체 변수 파일 → 열기 → 데이터 날짜·범주·척도 변수 구분
2. 기술통계 만족도, 영양균형, 잔반율의 평균과 표준편차를 확인한다. satisfaction_score, nutrition_balance_score, leftover_pct 분석 → 기술통계량 → 기술통계 전체 급식 품질의 기본 수준 파악
3. 메뉴유형별 평균 비교 메뉴 유형별 만족도·영양균형·잔반율 차이를 비교한다. menu_category, satisfaction_score, nutrition_balance_score, leftover_pct 분석 → 평균비교 → 일원배치 분산분석 인기 메뉴와 균형 메뉴의 차이 확인
4. 상관분석 영양균형과 만족도, 잔반율 간 관계를 확인한다. satisfaction_score, nutrition_balance_score, leftover_pct, sodium_mg, fruit_veg_servings 분석 → 상관분석 → 이변량 상관계수 학생 선호와 건강성의 일치/불일치 판단
5. 회귀분석 만족도를 설명하는 요인을 파악한다. 종속: satisfaction_score / 독립: taste_q, freshness_q, portion_q, nutrition_balance_score, processed_food_level 분석 → 회귀분석 → 선형 무엇이 만족도를 가장 크게 높이는지 확인
6. 군집분석 메뉴를 선호도와 영양균형 기준으로 유형화한다. preference_score, nutrition_balance_score, leftover_pct 분석 → 분류분석 → K-평균 군집분석 건강·선호 균형형, 선호 높음·영양 보완형 등 분류
7. 시각화 메뉴 유형별 결과와 선호-영양 차이를 그래프로 표현한다. menu_category, satisfaction_score, nutrition_balance_score, preference_nutrition_gap 그래프 → 차트 작성기 보고서용 핵심 시각화 생성

6. 분석 결과 예시

전체 평균 급식 만족도는 3.96점, 평균 영양균형 점수는 83.0점, 평균 잔반율은 10.3%로 나타났다.

상관분석 결과, 급식 만족도와 영양균형 점수의 상관은 -0.19, 급식 만족도와 잔반율의 상관은 -0.53로 나타났다. 이 시범 결과는 학생들이 좋아하는 메뉴가 반드시 영양적으로 우수한 메뉴와 일치하지 않을 수 있음을 보여준다.

메뉴 유형별 평균을 보면 패스트푸드형과 인기고열량형은 만족도가 높지만 영양균형 점수가 낮은 경향이 있고, 한식균형형과 채소강화형은 영양균형 점수가 높지만 일부 메뉴에서 잔반율과 수용성 문제가 나타났다.

7. 개선 제안서

제안 대상 제안 내용 근거 데이터 기대 효과 실행 난이도
학교 급식실·영양교사 인기 메뉴는 유지하되 튀김·가공식품 비율을 낮추고 채소·과일을 자연스럽게 결합한다. 인기고열량형의 만족도는 높지만 영양균형 점수가 낮음 선호도와 영양균형의 동시 개선 중
학생회 월 1회 학생 선호 메뉴 투표와 영양 균형 피드백을 함께 운영한다. 만족도 문항과 영양정보 인지 문항 학생 참여와 급식 수용성 향상 하
학교운영위원회 잔반율 높은 메뉴를 대상으로 조리법·양·제공 방식 개선 회의를 정례화한다. 잔반율, 만족도, 메뉴 유형별 분석 음식물 쓰레기 감소와 예산 효율화 중
학부모회 급식표에 ‘오늘의 영양 포인트’를 쉬운 문장으로 표시한다. 영양정보 인지 점수의 편차 학생·학부모의 건강 정보 접근성 개선 하
교육청 급식 만족도와 영양균형을 함께 보는 학교 단위 대시보드 시범사업을 제안한다. 영양균형 점수와 설문 데이터 결합 데이터 기반 학교급식 정책 개선 상

8. 한계점과 후속 연구

이번 자료는 실제 급식 데이터가 아니라 공공데이터 구조를 반영한 시범용 가상 데이터이므로, 실제 분석에서는 NEIS 급식식단정보와 학교 급식실의 잔반량 자료, 학생 설문 결과를 직접 수집해야 한다.

후속 연구에서는 메뉴별 사진, 조리 방식, 배식 시간, 줄 서는 시간, 급식실 혼잡도, 식사 시간 부족 여부 등을 추가하면 학생 만족도와 잔반율을 더 정확하게 설명할 수 있다.

9. 나의 진로·전공 관심과의 연결

이 프로젝트는 식품영양학과 보건학을 배우고 싶은 학생에게는 건강한 식단 설계의 실제 문제를 보여준다. 경영학과 데이터 분석에 관심 있는 학생에게는 고객 만족도와 서비스 품질 개선의 사례가 된다. 중요한 것은 학생을 ‘편식하는 사람’으로 판단하는 것이 아니라, 건강한 식단이 실제로 먹히도록 만드는 환경을 설계하는 것이다.

10. 에세이형 성찰 질문

· 데이터는 나의 직관과 같았는가, 달랐는가?

· 학교 급식은 학생의 건강과 행복을 어떻게 동시에 고려해야 하는가?
